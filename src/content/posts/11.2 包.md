## Python 包（Package）与标准库学习笔记
### 一、`dir()` 函数
`dir()` 是 Python 内置函数，用于列出对象的属性和方法，或当前作用域中定义的名称。

#### 1.1 查看模块内容
```python
import math

# 查看 math 模块中的所有名称
print(dir(math))
# ['__doc__', '__loader__', '__name__', '__package__', 'acos', 'asin', ...]
```

#### 1.2 查看对象属性和方法
```python
class MyClass:
    def __init__(self):
        self.x = 1
        self.y = 2
    
    def method1(self):
        pass

obj = MyClass()
print(dir(obj))
# ['__class__', '__init__', 'method1', 'x', 'y', ...]
```

#### 1.3 查看当前作用域
```python
def my_function():
    pass

variable = 10

print(dir())
# ['MyClass', '__builtins__', '__doc__', '__name__', 'my_function', 'variable', ...]
```

### 二、包（Package）概述
**包**是一种管理 Python 模块命名空间的方式，通过 `包名.模块名` 来组织模块。

| 概念 | 说明 |
| --- | --- |
| **包** | 包含 `__init__.py` 文件的目录 |
| **子包** | 包内的子目录，也包含 `__init__.py` |
| **模块** | 包内的 `.py` 文件 |


> 💡 **包 vs 文件夹**：包是一个特殊的文件夹，必须包含 `__init__.py` 文件（可以是空文件）。
>

#### 包的目录结构示例
```plain
sound/                          # 最高层级的包
      __init__.py               # 初始化 sound 包
      formats/                  # 用于文件格式转换的子包
              __init__.py
              wavread.py
              wavwrite.py
              aiffread.py
              aiffwrite.py
              ...
      effects/                  # 用于音效的子包
              __init__.py
              echo.py
              surround.py
              reverse.py
              ...
      filters/                  # 用于过滤器的子包
              __init__.py
              equalizer.py
              vocoder.py
              karaoke.py
              ...
```

### 三、创建包
**创建包 **`graphic`：

1. 创建 `graphic` 文件夹
2. 在文件夹中创建 `__init__.py`（可为空）
3. 创建模块文件

`graphic/circle.py`：

```python
"""circle.py - 圆形计算模块"""

radius = 10
PI = 3.1415926

def area(radius):
    """计算圆面积"""
    return PI * radius * radius

def perimeter(radius):
    """计算圆周长"""
    return 2 * PI * radius
```

`graphic/rectangle.py`：

```python
"""rectangle.py - 矩形计算模块"""

rectangle_width = 10
rectangle_height = 10

def area(width, height):
    """计算矩形面积"""
    return width * height

def perimeter(width, height):
    """计算矩形周长"""
    return 2 * (width + height)
```

**目录结构**：

```plain
graphic/                    # 包
    __init__.py              # 包初始化文件
    circle.py                # 模块
    rectangle.py             # 模块
main.py                      # 使用包的脚本
```

### 四、导入包
#### 4.1 全局导入 `import 包名.模块名`
导入包中模块的所有成员，通过 `包名.模块名.成员名` 访问。

```python
import graphic.circle

print(graphic.circle.area(10))   # 314.15926
print(graphic.circle.PI)         # 3.1415926
```

**使用别名**：

```python
import graphic.circle as c

print(c.area(10))                # 314.15926
```

> ⚠️ **注意**：使用 `import` 时，除最后一项外都必须是**包**，最后一项可以是模块或包。
>

#### 4.2 局部导入 `from 包名 import 模块名`
从包中导入模块，通过 `模块名.成员名` 访问。

```python
from graphic import circle

print(circle.area(10))   # 314.15926
print(circle.PI)         # 3.1415926
```

#### 4.3 局部导入 `from 包名.模块名 import 成员名`
从包中的模块导入指定成员，直接通过**成员名**访问。

```python
from graphic.circle import area, PI

print(area(10))   # 314.15926
print(PI)         # 3.1415926
```

#### 4.4 从包中导入模块 `from 包名 import *`
使用 `from 包名 import *` 时，Python **不会**自动导入所有子模块。需要在 `__init__.py` 中定义 `__all__` 指定要导入的模块。

**在 **`graphic/__init__.py`** 中**：

```python
# graphic/__init__.py
__all__ = ["circle"]   # 只允许导入 circle 模块
```

**在 **`main.py`** 中使用**：

```python
from graphic import *

print(circle.area(10))   # 314.15926（可以访问）
# print(rectangle.area(10))  # ❌ NameError: name 'rectangle' is not defined
```

> ⚠️ **注意**：如果 `__all__` 中不指定，`from ... import *` 无法导入任何模块。
>

#### 4.5 包内模块间导入（`__init__.py`）
在包内的 `__init__.py` 中导入子模块时，建议使用**相对路径**：

```python
# graphic/__init__.py
from . import circle   # ✅ 推荐：相对导入
# import circle        # ❌ 不推荐：绝对导入（会在 sys.path 中查找）
```

### 五、`__init__.py` 的作用
| 作用 | 说明 |
| --- | --- |
| **标识包** | 让 Python 将该目录视为包 |
| **初始化代码** | 包导入时执行的代码 |
| **控制 **`*`** 导入** | 通过 `__all__` 控制 `from ... import *` |
| **便捷导入** | 在 `__init__.py` 中导入常用模块，简化使用 |


**示例：便捷导入**

```python
# graphic/__init__.py
from .circle import area as circle_area
from .rectangle import area as rect_area

__all__ = ["circle_area", "rect_area"]
```

```python
# main.py
from graphic import circle_area, rect_area

print(circle_area(10))   # 314.15926
print(rect_area(5, 3))   # 15
```

### 六、Python 标准库
Python 自带的标准库提供了丰富的功能。以下是常用模块：

| 分类 | 模块 | 说明 |
| --- | --- | --- |
| **系统** | `os` | 操作系统接口 |
|  | `sys` | 系统相关参数和函数 |
| **时间** | `time` | 时间访问和转换 |
|  | `datetime` | 日期和时间操作 |
| **数学** | `math` | 数学函数 |
|  | `random` | 随机数生成 |
| **文本** | `re` | 正则表达式 |
|  | `json` | JSON 编码/解码 |
| **数据结构** | `collections` | 专用容器数据类型 |
|  | `copy` | 浅拷贝和深拷贝 |
| **网络** | `socket` | 低层级网络接口 |
|  | `urllib` | URL 处理 |
| **并行** | `threading` | 基于线程的并行 |
|  | `multiprocessing` | 基于进程的并行 |
| **文件** | `shutil` | 高级文件操作 |
|  | `glob` | Unix 风格路径模式匹配 |
| **加密** | `hashlib` | 安全哈希 |


**使用示例**：

```python
import os
import sys
import math
import random
import datetime
import json

# os：获取当前目录
print(os.getcwd())

# sys：查看 Python 版本
print(sys.version)

# math：计算平方根
print(math.sqrt(16))

# random：生成随机数
print(random.randint(1, 100))

# datetime：获取当前时间
print(datetime.datetime.now())

# json：处理 JSON 数据
data = {"name": "张三", "age": 18}
json_str = json.dumps(data)
print(json_str)
```

### 七、安装第三方库
#### 7.1 使用 `pip` 命令
**pip** 是 Python 的包管理工具，默认从 [PyPI](https://pypi.org) 下载安装。

**常用命令**：

```bash
# 查看已安装的包
pip list

# 安装包
pip install 包名

# 安装指定版本
pip install 包名==1.2.3

# 卸载包
pip uninstall 包名

# 升级包
pip install --upgrade 包名
```

**使用国内镜像源加速**：

```bash

pip install -i http://mirrors.aliyun.com/pypi/simple/ 包名
```

### 八、打包自己的库并安装
#### 8.1 安装 `setuptools`
```bash
pip install setuptools
```

#### 8.2 创建 `setup.py`
在包的同级目录下创建 `setup.py` 文件：

```plain
project/
    graphic/           # 包
        __init__.py
        circle.py
        rectangle.py
    setup.py           # 打包配置文件
```

`setup.py`** 内容**：

```python
from distutils.core import setup

setup(
    name="graphic",                    # 包名
    version="1.0",                     # 版本号
    py_modules=["graphic.circle", "graphic.rectangle"],  # 要打包的模块
    author="Your Name",                # 作者（可选）
    description="图形计算包",           # 描述（可选）
)
```

#### 8.3 构建包
```bash
# 在 setup.py 同级目录下执行

# 构建（生成 .egg 文件）
python setup.py bdist_egg

# 生成压缩包（.tar.gz）
python setup.py sdist
```

#### 8.4 安装自己的包
```bash
# 安装 .tar.gz 包
pip install dist/graphic-1.0.tar.gz

# 或在 PyCharm 中通过界面安装
```

#### 8.5 验证安装
```python
# 安装后即可导入使用
import graphic.circle

print(graphic.circle.area(10))   # 314.15926
```

### 九、包的导入方式对比
| 导入方式 | 语法 | 访问方式 | 适用场景 |
| --- | --- | --- | --- |
| 全局导入 | `import 包.模块` | `包.模块.成员` | 避免命名冲突 |
| 模块导入 | `from 包 import 模块` | `模块.成员` | 简化路径 |
| 成员导入 | `from 包.模块 import 成员` | `成员` | 只使用少量成员 |
| `*` 导入 | `from 包 import *` | `模块.成员` | 需要 `__all__` 控制 |


### 十、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 包只是普通文件夹 | ❌ 包必须包含 `__init__.py` 文件 |
| `from 包 import *` 自动导入所有子模块 | ❌ 需在 `__init__.py` 中定义 `__all__` |
| `__init__.py` 只能为空 | ✅ 可以执行初始化代码、定义 `__all__` |
| 包内绝对导入比相对导入好 | ✅ 包内推荐使用相对导入 `from . import 模块` |
| `import 包.模块.函数` 可以直接使用 | ❌ `import` 最后一项不能是函数/类/变量 |


### 十一、快速参考卡片
```python
# === 创建包 ===
# 目录结构：
# my_package/
#     __init__.py
#     module1.py
#     module2.py

# === 导入方式 ===

# 1. 全局导入
import my_package.module1
my_package.module1.function()

# 2. 模块导入
from my_package import module1
module1.function()

# 3. 成员导入
from my_package.module1 import function
function()

# 4. 带别名
import my_package.module1 as m1
m1.function()

# === __init__.py 控制 ===
# 控制 from ... import *
__all__ = ["module1", "module2"]

# 便捷导入
from .module1 import function as f1
from .module2 import function as f2

# === 打包自己的库 ===
# setup.py
from distutils.core import setup
setup(
    name="package_name",
    version="1.0",
    py_modules=["package.module1", "package.module2"],
)

# 构建
# python setup.py bdist_egg
# python setup.py sdist

# 安装
# pip install dist/package_name-1.0.tar.gz
```

> 📌 **核心记忆**：
>
> + **包** = 包含 `__init__.py` 的文件夹，用于组织模块
> + `__init__.py` 标识包，可控制 `*` 导入，执行初始化代码
> + 包内导入推荐使用**相对导入**（`from . import 模块`）
> + 使用 `from 包 import *` 时需在 `__init__.py` 中定义 `__all__`
> + **pip** 是 Python 的包管理工具，用于安装/卸载第三方库
> + 打包自己的库需要 `setup.py` 和 `setuptools`
>
