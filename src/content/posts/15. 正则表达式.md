### 一、什么是正则表达式
**正则表达式（Regular Expression）** 是一种用于匹配和操作文本的强大工具，由一系列字符和特殊字符组成的模式，用于描述要匹配的文本模式。

| 用途 | 说明 |
| --- | --- |
| **查找** | 在文本中搜索匹配的模式 |
| **替换** | 将匹配的内容替换为指定文本 |
| **提取** | 从文本中提取符合模式的内容 |
| **验证** | 检查文本是否符合指定格式 |


### 二、re 模块常用方法
#### 2.1 方法对比
| 方法 | 说明 | 返回类型 |
| --- | --- | --- |
| `search(pattern, string)` | 扫描整个字符串，返回第一个匹配位置 | `Match` 或 `None` |
| `match(pattern, string)` | 从字符串开头匹配 | `Match` 或 `None` |
| `findall(pattern, string)` | 返回所有非重叠匹配 | 字符串列表 |
| `sub(pattern, repl, string)` | 替换匹配的内容 | 新字符串 |
| `split(pattern, string)` | 按模式分割字符串 | 字符串列表 |


#### 2.2 示例代码
```python
import re

text = "Hello, my email is test@example.com and phone is 13812345678"

# search：查找第一个匹配
email_match = re.search(r"\w+@\w+\.\w+", text)
if email_match:
    print(f"找到邮箱: {email_match.group()}")  # test@example.com

# match：从开头匹配
result = re.match(r"Hello", text)
print(f"match 结果: {result.group() if result else None}")  # Hello

# findall：查找所有匹配
numbers = re.findall(r"\d+", text)
print(f"所有数字: {numbers}")  # ['13812345678']

# sub：替换匹配内容
new_text = re.sub(r"\d", "X", text)
print(f"替换后: {new_text}")

# split：分割字符串
parts = re.split(r"[,\s]+", text)
print(f"分割结果: {parts}")
```

### 三、正则表达式语法
#### 3.1 表示字符
| 字符 | 说明 | 示例 |
| --- | --- | --- |
| `.` | 匹配除换行符外的任意单个字符 | `a.b` 匹配 `acb`、`a b` |
| `\d` | 匹配数字 `[0-9]` | `\d` 匹配 `5` |
| `\D` | 匹配非数字 `[^0-9]` | `\D` 匹配 `a` |
| `\w` | 匹配单词字符 `[A-Za-z0-9_]` | `\w` 匹配 `a`、`1`、`_` |
| `\W` | 匹配非单词字符 | `\W` 匹配 `@`、`#` |
| `\s` | 匹配空白字符（空格、制表符等） | `\s` 匹配 ` `、`\t` |
| `\S` | 匹配非空白字符 | `\S` 匹配 `a`、`1` |
| `[xyz]` | 匹配 x、y 或 z 中的任意一个 | `[abc]` 匹配 `a`、`b`、`c` |
| `[^xyz]` | 匹配除了 x、y、z 之外的字符 | `[^abc]` 匹配 `d`、`e` |
| `[a-z]` | 匹配 a 到 z 范围内的任意字符 | `[a-z]` 匹配小写字母 |


#### 3.2 表示数量
| 字符 | 说明 | 示例 |
| --- | --- | --- |
| `*` | 零次或多次（贪婪） | `a*` 匹配 `""`、`"a"`、`"aa"` |
| `+` | 一次或多次（贪婪） | `a+` 匹配 `"a"`、`"aa"` |
| `?` | 零次或一次（贪婪） | `a?` 匹配 `""`、`"a"` |
| `{n}` | 恰好 n 次 | `a{3}` 匹配 `"aaa"` |
| `{n,}` | 至少 n 次 | `a{3,}` 匹配 `"aaa"`、`"aaaa"` |
| `{n,m}` | n 到 m 次 | `a{2,4}` 匹配 `"aa"`、`"aaa"`、`"aaaa"` |
| `*?` `+?` `??` `{n,m}?` | 非贪婪匹配（尽可能少） | `o+?` 匹配 `"o"`（而非 `"oo"`） |


#### 3.3 表示边界
| 字符 | 说明 |
| --- | --- |
| `^` | 匹配字符串的开始位置 |
| `$` | 匹配字符串的结束位置 |
| `\b` | 匹配单词边界 |
| `\B` | 匹配非单词边界 |


#### 3.4 匹配分组
| 字符 | 说明 |
| --- | --- |
| `x|y` | 匹配 x 或 y |
| `(pattern)` | 分组捕获 |
| `(?P<name>pattern)` | 命名分组 |
| `(?P=name)` | 引用命名分组 |
| `\num` | 引用第 num 个分组 |


### 四、原始字符串
在正则表达式中，建议使用**原始字符串**（`r""`）避免转义问题。

```python
import re

text = "abcdef123456"

# 使用原始字符串（推荐）
print(re.search(r"\w+", text))

# 不使用原始字符串（有语法警告）
# print(re.search("\w+", text))  # SyntaxWarning
```

### 五、实战案例
#### 5.1 匹配电话号码
```python
import re

test = [
    "13812345678",   # 合法
    "11456817239",   # 非法（第二位不是 3/4/5/7/8/9）
    "19912345678",   # 合法
    "17138412356",   # 合法
    "1234567890",    # 非法（长度不足）
    "14752345673",   # 合法
    "1800123456",    # 非法（长度不足）
]

# 规则：以1开头，第二位为3/4/5/7/8/9，后面9位数字
pattern = r"^1[345789]\d{9}$"

for phone in test:
    is_valid = "合法" if re.match(pattern, phone) else "非法"
    print(f"{phone:15} {is_valid}")
```

**输出**：

```plain
13812345678     合法
11456817239     非法
19912345678     合法
17138412356     合法
1234567890      非法
14752345673     合法
1800123456      非法
```

#### 5.2 匹配邮箱地址
```python
import re

test = [
    "example@example.com",
    "user.name@subdomain.example.co",
    "username@.com",
    "@missingusername.com",
    "-dasd@qq.com",
]

# 邮箱格式：用户名@域名.后缀
pattern = r"[\w!#$%&'*+/=?^`{|}~.]+@[\w!#$%&'*+/=?^`{|}~.]+\.[a-zA-Z]{2,}$"

for email in test:
    is_valid = "合法" if re.match(pattern, email) else "非法"
    print(f"{email:40} {is_valid}")
```

#### 5.3 匹配 0-255 之间的数字
```python
import re

test = ["0", "9", "50", "100", "199", "200", "255", "256", "-1", "01", "001"]

# 分析 0-255：
# 1位数：0-9 → [1-9]?\d
# 2位数：10-99 → [1-9]\d
# 3位数：100-199 → 1\d{2}
# 3位数：200-249 → 2[0-4]\d
# 3位数：250-255 → 25[0-5]

pattern = r"^([1-9]?\d|1\d{2}|2[0-4]\d|25[0-5])$"

for num in test:
    is_valid = "合法" if re.match(pattern, num) else "非法"
    print(f"{num:5} {is_valid}")
```

**输出**：

```plain
0     合法
9     合法
50    合法
100   合法
199   合法
200   合法
255   合法
256   非法
-1    非法
01    合法
001   非法
```

#### 5.4 从 HTML 标签中提取 URL
```python
import re

html = """
<link rel="alternate" hreflang="zh" href="https://zh.wikipedia.org/wiki/%E6%AD%A3%E5%88%99%E8%A1%A8%E8%BE%BE%E5%BC%8F">
<link rel="alternate" hreflang="zh-Hans" href="https://zh.wikipedia.org/zh-hans/%E6%AD%A3%E5%88%99%E8%A1%A8%E8%BE%BE%E5%BC%8F">
<link rel="alternate" hreflang="zh-Hant" href="https://zh.wikipedia.org/zh-hant/%E6%AD%A3%E5%88%99%E8%A1%A8%E8%BE%BE%E5%BC%8F">
"""

# 提取 href 属性中的 URL（非贪婪匹配）
pattern = r'href="(.+?)"'

urls = re.findall(pattern, html)
for url in urls:
    print(url)
```

**输出**：

```plain
https://zh.wikipedia.org/wiki/%E6%AD%A3%E5%88%99%E8%A1%A8%E8%BE%BE%E5%BC%8F
https://zh.wikipedia.org/zh-hans/%E6%AD%A3%E5%88%99%E8%A1%A8%E8%BE%BE%E5%BC%8F
https://zh.wikipedia.org/zh-hant/%E6%AD%A3%E5%88%99%E8%A1%A8%E8%BE%BE%E5%BC%8F
```

#### 5.5 替换数字为对应单词
```python
import re

text = "I have 2 apples and 3 oranges."

# 数字到单词的映射
num_map = {
    "1": "one", "2": "two", "3": "three",
    "4": "four", "5": "five"
}

# 使用函数进行替换
def replace_number(match):
    return num_map.get(match.group(0), match.group(0))

result = re.sub(r"\d", replace_number, text)
print(result)  # I have two apples and three oranges.
```

#### 5.6 命名分组示例
```python
import re

text = "姓名: 张三, 年龄: 25, 邮箱: zhangsan@example.com"

# 使用命名分组提取信息
pattern = r"姓名:\s*(?P<name>\w+),\s*年龄:\s*(?P<age>\d+),\s*邮箱:\s*(?P<email>\w+@\w+\.\w+)"

match = re.search(pattern, text)
if match:
    # 通过名称访问分组
    print(f"姓名: {match.group('name')}")
    print(f"年龄: {match.group('age')}")
    print(f"邮箱: {match.group('email')}")
    
    # 获取所有分组（字典形式）
    print(match.groupdict())
```

### 六、正则表达式速查表
| 类别 | 语法 | 说明 |
| --- | --- | --- |
| **字符** | `.` `\d` `\D` `\w` `\W` `\s` `\S` `[abc]` `[^abc]` `[a-z]` | 匹配特定字符 |
| **数量** | `*` `+` `?` `{n}` `{n,}` `{n,m}` | 重复次数 |
| **边界** | `^` `$` `\b` `\B` | 位置匹配 |
| **分组** | `(...)` `(?P<name>...)` `\1` `(?:...)` | 捕获和非捕获分组 |
| **选择** | `|` | 或 |
| **标志** | `re.I` `re.M` `re.S` `re.U` `re.X` | 匹配模式 |


#### 常用标志
| 标志 | 说明 |
| --- | --- |
| `re.I` | 忽略大小写 |
| `re.M` | 多行模式（`^` `$` 匹配行首行尾） |
| `re.S` | 让 `.` 匹配换行符 |
| `re.U` | Unicode 匹配 |
| `re.X` | 忽略空白和注释 |


```python
# 忽略大小写匹配
result = re.search(r"hello", "HELLO World", re.I)
print(result.group())  # HELLO

# 多行模式
text = "line1\nline2\nline3"
matches = re.findall(r"^line\d", text, re.M)
print(matches)  # ['line1', 'line2', 'line3']
```

### 七、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| `match()` 和 `search()` 的区别 | `match` 从开头匹配，`search` 扫描整个字符串 |
| 贪婪匹配导致结果过长 | 使用 `*?` `+?` 等进行非贪婪匹配 |
| 忘记使用原始字符串 | 使用 `r""` 避免反斜杠转义 |
| `\w` 在 Unicode 模式下匹配中文 | 默认匹配中文，如需排除使用 `re.ASCII` |
| `.` 不匹配换行符 | 使用 `re.S` 标志让 `.` 匹配换行符 |
| 分组编号从 1 开始 | `\0` 表示整个匹配，`\1` 表示第一个分组 |


### 八、快速参考卡片
```python
import re

# === 常用方法 ===
re.search(pattern, string)       # 查找第一个匹配
re.match(pattern, string)        # 从开头匹配
re.findall(pattern, string)      # 查找所有匹配
re.finditer(pattern, string)     # 查找所有匹配（迭代器）
re.sub(pattern, repl, string)    # 替换
re.split(pattern, string)        # 分割

# === Match 对象 ===
match.group()        # 整个匹配
match.group(n)       # 第 n 个分组
match.group(name)    # 命名分组
match.groupdict()    # 所有命名分组（字典）
match.start()        # 匹配起始位置
match.end()          # 匹配结束位置
match.span()         # (起始, 结束)

# === 编译正则 ===
pattern = re.compile(r"\d+")
pattern.findall(text)
```

> 📌 **核心记忆**：
>
> + `match` 从开头匹配，`search` 扫描全文
> + **原始字符串**（`r""`）避免转义问题
> + **贪婪匹配**（`*`、`+`）尽可能多，**非贪婪**（`*?`、`+?`）尽可能少
> + `.` 不匹配换行，用 `re.S` 改变
> + `^` 匹配开头，`$` 匹配结尾
> + **分组捕获**用 `()`，引用用 `\1`、`\2`
>
