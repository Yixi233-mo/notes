## Python 文件操作学习笔记
### 一、文件的基本概念
**文件**是存储在磁盘上的数据集合，可以是文本、图像、音频、视频或程序代码等。

| 概念 | 说明 |
| --- | --- |
| **文件名** | 包含名称和扩展名（如 `test.txt`） |
| **扩展名** | 表示文件类型（`.txt`文本、`.jpg`图片、`.py`Python代码） |
| **文件路径** | 文件在文件系统中的位置 |
| **持久化** | 将内存中的数据写入磁盘文件的过程 |


#### 1.1 文件的分类
| 类型 | 特点 | 编码 | 示例 |
| --- | --- | --- | --- |
| **纯文本文件** | 有统一编码，可看作长字符串 | ASCII、UTF-8、GBK 等 | `.txt`、`.py`、`.csv` |
| **二进制文件** | 无统一编码，由0和1组成 | 无 | `.jpg`、`.png`、`.avi`、`.exe` |


#### 1.2 文件路径
| 路径类型 | 说明 | 示例 |
| --- | --- | --- |
| **相对路径** | 从当前位置开始 | `./hello.py`（当前目录）、`../hello.py`（上级目录） |
| **绝对路径** | 从根目录开始 | `E:/Hello/hello.py`（Windows）、`/home/user/hello.py`（Linux/Mac） |


```python
# 路径表示方式
# Windows 推荐使用正斜杠或原始字符串
path1 = "E:/Hello/hello.py"      # 正斜杠
path2 = r"E:\Hello\hello.py"     # 原始字符串
path3 = "E:\\Hello\\hello.py"    # 转义反斜杠
```

### 二、文件的打开与关闭
#### 2.1 `open()` 函数
```python
f = open(file, mode="r", encoding=None, ...)
```

**常用模式**

| 模式 | 说明 | 文件不存在 | 原有内容 |
| --- | --- | --- | --- |
| `'r'` | 只读（默认） | ❌ 报错 | 保留 |
| `'w'` | 写入 | ✅ 创建 | ❌ 清空 |
| `'a'` | 追加 | ✅ 创建 | 保留，追加到末尾 |
| `'x'` | 独占创建 | ✅ 创建 | ❌ 报错（文件已存在） |
| `'b'` | 二进制模式 | — | — |
| `'t'` | 文本模式（默认） | — | — |
| `'+'` | 读写模式 | — | — |


**模式组合示例**

| 模式 | 含义 |
| --- | --- |
| `'r+'` | 读写（文件必须存在） |
| `'w+'` | 读写（清空或创建） |
| `'a+'` | 读写（追加） |
| `'rb'` | 二进制只读 |
| `'wb'` | 二进制写入 |


```python
# 基本用法
f = open("test.txt", "w")      # 写入模式（文本）
f = open("test.txt", "rb")     # 二进制只读
f = open("test.txt", "w+", encoding="utf-8")  # 指定编码
```

#### 2.2 关闭文件
```python
f.close()  # 关闭文件，释放系统资源
```

> ⚠️ **重要**：使用 `close()` 确保：
>
> + 写入的数据从缓冲区刷新到磁盘
> + 释放文件占用的系统资源
>

#### 2.3 使用 `with` 语句（推荐）
`with` 语句会自动管理文件上下文，即使发生异常也会自动关闭文件。

```python
# 推荐写法
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
    # 文件在 with 块结束后自动关闭

# 不需要显式调用 close()
```

### 三、文件读写操作
#### 3.1 写数据
| 方法 | 说明 |
| --- | --- |
| `write(str)` | 写入字符串 |
| `writelines(seq)` | 写入字符串序列（需自己添加换行符） |


```python
# 基本写入
with open("test.txt", "w", encoding="utf-8") as f:
    f.write("hello world\n")
    f.write("nihao python\n")

# 写入序列
lines = ["第一行\n", "第二行\n", "第三行\n"]
with open("test.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)

# 注意：writelines 不会自动添加换行符
lines_no_newline = ["line1", "line2", "line3"]
with open("test.txt", "w", encoding="utf-8") as f:
    f.writelines(line + "\n" for line in lines_no_newline)
```

#### 3.2 读数据
| 方法 | 说明 |
| --- | --- |
| `read([size])` | 读取 size 个字节，不指定则读取全部 |
| `readline([size])` | 读取一行，可指定读取字节数 |
| `readlines([size])` | 读取所有行，返回列表 |


```python
# 准备测试文件
with open("test.txt", "w", encoding="utf-8") as f:
    f.write("第一行：Hello\n")
    f.write("第二行：World\n")
    f.write("第三行：Python\n")

# read() - 读取全部
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print("全部内容:")
    print(content)

# read(size) - 读取指定字节数
with open("test.txt", "rb") as f:  # 二进制模式读取字节
    data = f.read(5)               # 读取5个字节
    print(f"读取5个字节: {data}")

# readline() - 逐行读取
with open("test.txt", "r", encoding="utf-8") as f:
    line1 = f.readline()
    line2 = f.readline()
    print(f"第一行: {line1.rstrip()}")
    print(f"第二行: {line2.rstrip()}")

# 遍历文件对象（推荐）
with open("test.txt", "r", encoding="utf-8") as f:
    for line in f:                 # 逐行遍历，内存友好
        print(line.rstrip())

# readlines() - 读取所有行到列表
with open("test.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    print(lines)  # ['第一行：Hello\n', '第二行：World\n', '第三行：Python\n']
```

### 四、文件指针操作
#### 4.1 `tell()` - 获取当前位置
```python
with open("test.txt", "r", encoding="utf-8") as f:
    print(f"初始位置: {f.tell()}")  # 0
    f.read(3)
    print(f"读取3个字符后: {f.tell()}")
```

#### 4.2 `seek()` - 移动文件指针
```python
seek(offset, whence=0)
```

| `whence` | 值 | 说明 |
| --- | --- | --- |
| `os.SEEK_SET` | 0 | 从文件开头开始（默认） |
| `os.SEEK_CUR` | 1 | 从当前位置开始 |
| `os.SEEK_END` | 2 | 从文件末尾开始 |


```python
with open("test.txt", "rb") as f:  # 非二进制模式只能使用 whence=0
    # 移动到第10个字节
    f.seek(10, 0)      # 从开头偏移10字节
    print(f"当前位置: {f.tell()}")
    
    # 相对于当前位置移动
    f.seek(5, 1)       # 从当前位置向前偏移5字节
    print(f"当前位置: {f.tell()}")
    
    # 移动到文件末尾向前5字节
    f.seek(-5, 2)      # 从末尾向前偏移5字节
    print(f"当前位置: {f.tell()}")

# 文本模式示例（只能从开头移动）
with open("test.txt", "r", encoding="utf-8") as f:
    f.seek(5)          # 移动到第5个字符位置
    print(f.read(10))
```

**完整示例：递归遍历并统计**

```python
import os

def walk_directory(path):
    """递归遍历目录，统计文件和目录数量"""
    file_count = 0
    dir_count = 0
    
    for root, dirs, files in os.walk(path):
        dir_count += len(dirs)
        file_count += len(files)
        print(f"\n目录: {root}")
        print(f"  子目录: {dirs}")
        print(f"  文件: {files}")
    
    print(f"\n总计: {dir_count} 个子目录, {file_count} 个文件")
    return dir_count, file_count

# 遍历当前目录
walk_directory(os.getcwd())
```

### 七、文件拷贝案例
#### 7.1 基础版本（一次读取全部）
```python
def copy_file(source_path, dest_path):
    """拷贝文件（一次性读取全部内容）"""
    with open(source_path, 'rb') as source:
        content = source.read()
    
    with open(dest_path, 'wb') as dest:
        dest.write(content)
    
    print(f"文件已从 {source_path} 拷贝到 {dest_path}")

# 使用示例
copy_file("D:/mv.png", "E:/mv.png")
```

**缺点**：大文件会占用大量内存

#### 7.2 优化版本（分块读取）
```python
def copy_file_chunk(source_path, dest_path, chunk_size=1024):
    """拷贝文件（分块读取，内存友好）"""
    with open(source_path, 'rb') as source:
        with open(dest_path, 'wb') as dest:
            while True:
                chunk = source.read(chunk_size)
                if not chunk:  # 读取完毕
                    break
                dest.write(chunk)
    
    print(f"文件已从 {source_path} 拷贝到 {dest_path}")

# 使用示例
copy_file_chunk("D:/mv.png", "E:/mv.png", chunk_size=4096)
```

#### 7.3 带进度显示版本
```python
import os

def copy_file_with_progress(source_path, dest_path, chunk_size=1024*1024):
    """拷贝文件并显示进度"""
    # 获取源文件大小
    total_size = os.path.getsize(source_path)
    copied_size = 0
    
    with open(source_path, 'rb') as source:
        with open(dest_path, 'wb') as dest:
            while True:
                chunk = source.read(chunk_size)
                if not chunk:
                    break
                dest.write(chunk)
                copied_size += len(chunk)
                progress = (copied_size / total_size) * 100
                print(f"\r拷贝进度: {progress:.1f}%", end="")
    
    print(f"\n文件拷贝完成！")
    print(f"源文件: {source_path} ({total_size} 字节)")
    print(f"目标文件: {dest_path}")

# 使用示例
copy_file_with_progress("D:/large_file.zip", "E:/large_file.zip")
```

#### 8.1 日志文件处理器
```python
import os
import time
from datetime import datetime

class LogProcessor:
    """日志文件处理器"""
    
    def __init__(self, log_dir="logs"):
        self.log_dir = log_dir
        self._ensure_log_dir()
    
    def _ensure_log_dir(self):
        """确保日志目录存在"""
        if not os.path.exists(self.log_dir):
            os.makedirs(self.log_dir)
    
    def write_log(self, level, message):
        """写入日志"""
        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        log_line = f"[{timestamp}] [{level}] {message}\n"
        
        # 按日期分割日志文件
        log_file = os.path.join(self.log_dir, f"{datetime.now().strftime('%Y%m%d')}.log")
        
        with open(log_file, "a", encoding="utf-8") as f:
            f.write(log_line)
    
    def read_logs(self, date=None, level=None):
        """读取日志"""
        if date is None:
            date = datetime.now().strftime("%Y%m%d")
        
        log_file = os.path.join(self.log_dir, f"{date}.log")
        
        if not os.path.exists(log_file):
            print(f"日志文件不存在: {log_file}")
            return []
        
        logs = []
        with open(log_file, "r", encoding="utf-8") as f:
            for line in f:
                if level is None or f"[{level}]" in line:
                    logs.append(line.rstrip())
        
        return logs
    
    def clean_old_logs(self, days=30):
        """清理旧日志"""
        now = time.time()
        deleted_count = 0
        
        for filename in os.listdir(self.log_dir):
            filepath = os.path.join(self.log_dir, filename)
            if os.path.isfile(filepath):
                # 获取文件修改时间
                mtime = os.path.getmtime(filepath)
                if (now - mtime) > (days * 24 * 3600):
                    os.remove(filepath)
                    deleted_count += 1
                    print(f"已删除旧日志: {filename}")
        
        print(f"共删除 {deleted_count} 个旧日志文件")

# 使用示例
log_processor = LogProcessor()

# 写入日志
log_processor.write_log("INFO", "程序启动")
log_processor.write_log("ERROR", "发生错误：连接超时")
log_processor.write_log("INFO", "程序关闭")

# 读取日志
print("=== 今日日志 ===")
logs = log_processor.read_logs()
for log in logs:
    print(log)

# 读取 ERROR 级别日志
print("\n=== ERROR 日志 ===")
error_logs = log_processor.read_logs(level="ERROR")
for log in error_logs:
    print(log)

# 清理30天前的日志
# log_processor.clean_old_logs(30)
```

#### 8.2 配置文件管理器
```python
import json
import yaml  # 需要安装: pip install pyyaml

class ConfigManager:
    """配置文件管理器"""
    
    def __init__(self, config_file="config.json"):
        self.config_file = config_file
        self.config = self.load()
    
    def load(self):
        """加载配置文件"""
        if not os.path.exists(self.config_file):
            print(f"配置文件不存在，创建默认配置")
            return self._get_default_config()
        
        ext = os.path.splitext(self.config_file)[1].lower()
        
        with open(self.config_file, "r", encoding="utf-8") as f:
            if ext == ".json":
                return json.load(f)
            elif ext in [".yaml", ".yml"]:
                return yaml.safe_load(f)
            else:
                # 简单的 key=value 格式
                config = {}
                for line in f:
                    line = line.strip()
                    if line and not line.startswith("#"):
                        key, value = line.split("=", 1)
                        config[key.strip()] = value.strip()
                return config
    
    def _get_default_config(self):
        """获取默认配置"""
        return {
            "app_name": "MyApp",
            "version": "1.0.0",
            "debug": True,
            "host": "localhost",
            "port": 8080,
            "database": {
                "host": "localhost",
                "port": 3306,
                "user": "root",
                "password": ""
            }
        }
    
    def save(self):
        """保存配置文件"""
        ext = os.path.splitext(self.config_file)[1].lower()
        
        with open(self.config_file, "w", encoding="utf-8") as f:
            if ext == ".json":
                json.dump(self.config, f, indent=4, ensure_ascii=False)
            elif ext in [".yaml", ".yml"]:
                yaml.dump(self.config, f, allow_unicode=True)
            else:
                for key, value in self.config.items():
                    if not isinstance(value, dict):
                        f.write(f"{key}={value}\n")
        
        print(f"配置已保存到: {self.config_file}")
    
    def get(self, key, default=None):
        """获取配置值（支持点号分隔的嵌套键）"""
        keys = key.split(".")
        value = self.config
        
        for k in keys:
            if isinstance(value, dict) and k in value:
                value = value[k]
            else:
                return default
        
        return value
    
    def set(self, key, value):
        """设置配置值（支持点号分隔的嵌套键）"""
        keys = key.split(".")
        target = self.config
        
        for k in keys[:-1]:
            if k not in target:
                target[k] = {}
            target = target[k]
        
        target[keys[-1]] = value
        self.save()

# 使用示例
config = ConfigManager("app_config.json")

print(f"应用名称: {config.get('app_name')}")
print(f"数据库主机: {config.get('database.host')}")

# 修改配置
config.set("debug", False)
config.set("database.password", "secret123")
```

### 十、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 忘记关闭文件 | 使用 `with` 语句自动管理 |
| 文本模式读取二进制文件 | 用 `'rb'` 模式读取二进制文件 |
| 编码不一致导致乱码 | 读写时使用相同的编码（推荐 UTF-8） |
| 使用相对路径不确定 | 使用 `os.path.abspath()` 获取绝对路径 |
| `w` 模式会清空文件 | 需要追加内容用 `'a'` 模式 |
| 文件不存在时 `'r'` 模式报错 | 使用 `'w'` 或 `'a'` 会自动创建 |
| `read()` 大文件内存溢出 | 分块读取或逐行读取 |
| 路径拼接使用字符串加法 | 使用 `os.path.join()` |


```python
# 错误示例
# 1. 忘记关闭文件
f = open("test.txt", "w")
f.write("hello")
# 没有 close()，数据可能未写入磁盘

# 2. 编码问题
# with open("test.txt", "r") as f:  # 默认使用系统编码，可能出错
#     content = f.read()

# 正确做法
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()

# 3. 路径拼接错误
path = "E:/folder/" + "subfolder/" + "file.txt"  # 可能多或少斜杠

# 正确做法
import os
path = os.path.join("E:", "folder", "subfolder", "file.txt")
```

os.path.join("dir", "file.txt")  # 拼接路径
os.path.exists("file.txt")       # 判断存在
os.path.isfile("file.txt")       # 判断文件
os.path.isdir("dir")             # 判断目录
os.makedirs("new_dir", exist_ok=True)  # 创建目录

# 目录遍历
for root, dirs, files in os.walk("."):
    for file in files:
        print(os.path.join(root, file))
