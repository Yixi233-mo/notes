## Python 字典（Dictionary）学习笔记
### 一、字典概述
**字典（Dictionary）** 是 Python 中一种重要的键值对（key-value pair）数据结构，具有以下特点：

| 特性 | 说明 |
| --- | --- |
| **键值对存储** | 每个元素由键（key）和值（value）组成 |
| **键的唯一性** | 键必须是唯一的，不能重复 |
| **值可重复** | 值可以重复 |
| **无序性** | 传统上无序，Python 3.7+ 保持插入顺序 |
| **可变性** | ✅ 可以添加、修改、删除键值对 |
| **键的类型** | 键必须是**不可变**类型（字符串、数字、元组） |
| **值的类型** | 值可以是任意类型 |


**核心概念**：

+ **键（Key）**：唯一标识，相当于"索引"
+ **值（Value）**：存储的数据，通过键来访问
+ **键值对（Key-Value Pair）**：字典的基本单元

### 二、创建字典
#### 1. 基本创建方式
```python
# 使用花括号创建
dict1 = {}  # 空字典
dict2 = {"name": "Alice", "age": 18, "gender": "male"}

# 使用 dict() 构造函数
dict3 = dict()  # 空字典
dict4 = dict(name="Bob", age=20, gender="female")  # 关键字参数
dict5 = dict([("name", "Tom"), ("age", 22), ("gender", "male")])  # 从列表创建

# 使用 dict.fromkeys() 创建
keys = ["name", "age", "gender"]
dict6 = dict.fromkeys(keys)  # 值默认为 None
dict7 = dict.fromkeys(keys, "unknown")  # 指定默认值

print(dict1)  # {}
print(dict2)  # {'name': 'Alice', 'age': 18, 'gender': 'male'}
print(dict3)  # {}
print(dict4)  # {'name': 'Bob', 'age': 20, 'gender': 'female'}
print(dict5)  # {'name': 'Tom', 'age': 22, 'gender': 'male'}
print(dict6)  # {'name': None, 'age': None, 'gender': None}
print(dict7)  # {'name': 'unknown', 'age': 'unknown', 'gender': 'unknown'}
```

#### 2. 字典推导式
```python
# 基础字典推导式
squares = {x: x**2 for x in range(4)}
print(squares)  # {0: 0, 1: 1, 2: 4, 3: 9}

# 带条件的字典推导式
even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
print(even_squares)  # {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

# 从两个列表创建字典
keys = ["name", "age", "city"]
values = ["Alice", 25, "New York"]
person = {k: v for k, v in zip(keys, values)}
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'New York'}
```

### 三、访问字典元素
#### 1. 使用方括号 `[]` 访问
```python
dict1 = {"name": "Alice", "age": 18, "gender": "male"}

print(dict1["name"])    # Alice
print(dict1["age"])     # 18
print(dict1["gender"])  # male

# 访问不存在的键会报错
# print(dict1["address"])  # ❌ KeyError: 'address'
```

#### 2. 使用 `get()` 方法（推荐）
```python
dict1 = {"name": "Alice", "age": 18, "gender": "male"}

# 存在时返回对应的值
print(dict1.get("name"))     # Alice

# 不存在时返回 None（不会报错）
print(dict1.get("address"))  # None

# 可以指定默认值
print(dict1.get("address", "earth"))  # earth
print(dict1.get("age", 0))            # 18（存在，返回原值）

# 链式调用
name = dict1.get("name", "Unknown")
print(f"Name: {name}")
```

#### 3. 使用 `setdefault()` 方法
`setdefault()` 类似于 `get()`，但如果键不存在，会插入键值对。

```python
dict1 = {"name": "Alice", "age": 18}

# 键存在：返回对应的值，不修改字典
value1 = dict1.setdefault("name", "Unknown")
print(value1)  # Alice
print(dict1)   # {'name': 'Alice', 'age': 18}

# 键不存在：插入新键值对，返回默认值
value2 = dict1.setdefault("city", "New York")
print(value2)  # New York
print(dict1)   # {'name': 'Alice', 'age': 18, 'city': 'New York'}
```

### 四、添加和修改元素
#### 1. 添加新元素
```python
dict1 = {"name": "Alice", "age": 18}

# 为不存在的 key 赋值会添加新键值对
dict1["address"] = "earth"
dict1["gender"] = "male"

print(dict1)  
# {'name': 'Alice', 'age': 18, 'address': 'earth', 'gender': 'male'}
```

#### 2. 修改元素
```python
dict1 = {"name": "Alice", "age": 18, "gender": "male"}

# 通过 key 修改对应的 value
dict1["name"] = "Bob"
dict1["age"] = 20

print(dict1)  
# {'name': 'Bob', 'age': 20, 'gender': 'male'}
```

#### 3. 使用 `update()` 方法批量更新
```python
dict1 = {"name": "Alice", "age": 18}
dict2 = {"age": 20, "city": "New York"}

# 更新：存在的键会覆盖，不存在的会添加
dict1.update(dict2)
print(dict1)  
# {'name': 'Alice', 'age': 20, 'city': 'New York'}

# 也可以使用关键字参数
dict1.update(gender="female", country="USA")
print(dict1)  
# {'name': 'Alice', 'age': 20, 'city': 'New York', 'gender': 'female', 'country': 'USA'}
```

### 五、删除字典元素
| 方法 | 说明 | 键不存在时 |
| --- | --- | --- |
| `del dict[key]` | 删除指定键值对 | ❌ 抛出 `KeyError` |
| `pop(key[, default])` | 删除并返回值 | 无 default 时抛出 `KeyError` |
| `popitem()` | 删除并返回最后插入的键值对 | 字典为空时抛出 `KeyError` |
| `clear()` | 清空所有键值对 | ✅ 清空字典 |


```python
my_dict = {'Name': 'Tom', 'Age': 17, 'City': 'Beijing'}

# del：删除指定键
del my_dict['Name']
print(my_dict)  # {'Age': 17, 'City': 'Beijing'}

# pop()：删除并返回值
age = my_dict.pop('Age')
print(age)      # 17
print(my_dict)  # {'City': 'Beijing'}

# pop() 指定默认值（不报错）
city = my_dict.pop('City', 'Unknown')
print(city)     # Beijing
country = my_dict.pop('Country', 'Unknown')
print(country)  # Unknown

# popitem()：删除最后插入的键值对（Python 3.7+）
my_dict = {'a': 1, 'b': 2, 'c': 3}
item = my_dict.popitem()
print(item)     # ('c', 3)
print(my_dict)  # {'a': 1, 'b': 2}

# clear()：清空字典
my_dict.clear()
print(my_dict)  # {}
```

### 六、检查成员

```python
dict1 = {"name": "Alice", "age": 81, "gender": "male"}

print("Bob" in dict1.values())     # False

# 检查键值对是否存在
print(("name", "Alice") in dict1.items())   # True
```

### 八、遍历字典
#### 1. 遍历所有键（keys）
```python
my_dict = {'Name': 'Tom', 'Age': 17, 'City': 'Beijing'}

# 方法1：直接遍历字典（默认遍历键）
for key in my_dict:
    print(key)

# 方法2：使用 keys() 方法
for key in my_dict.keys():
    print(key)
```

#### 2. 遍历所有值（values）
```python
my_dict = {'Name': 'Tom', 'Age': 17, 'City': 'Beijing'}

for value in my_dict.values():
    print(value)

# values() 返回视图对象，可以转换为列表
value_list = list(my_dict.values())
print(value_list)  # ['Tom', 17, 'Beijing']
```

#### 3. 遍历键值对（items）
```python
my_dict = {'Name': 'Tom', 'Age': 17, 'City': 'Beijing'}

# 方法1：使用 items() 解包
for key, value in my_dict.items():
    print(f"{key}: {value}")

# 方法2：获取键值对元组
for item in my_dict.items():
    print(item)  # ('Name', 'Tom'), ('Age', 17), ...
    print(f"{item[0]}: {item[1]}")
```

#### 4. 完整遍历示例
```python
my_dict = {'Name': 'Tom', 'Age': 17}

# 遍历所有key
print("=== 遍历所有键 ===")
for k in my_dict.keys():
    print(k)

# 遍历所有value
print("\n=== 遍历所有值 ===")
for v in my_dict.values():
    print(v)

# 遍历键值对
print("\n=== 遍历键值对 ===")
for k, v in my_dict.items():
    print(k + "---" + str(v))

# 使用 enumerate 获取序号
print("\n=== 带序号的遍历 ===")
for i, (k, v) in enumerate(my_dict.items()):
    print(f"{i}: {k} = {v}")
```

### 九、字典的视图对象（keys、values、items）
`keys()`、`values()`、`items()` 返回的是**视图对象（view object）**，不是列表。视图对象会**动态反映字典的变化**。

```python
my_dict = {'a': 1, 'b': 2, 'c': 3}

# 获取视图
keys_view = my_dict.keys()
values_view = my_dict.values()
items_view = my_dict.items()

print(keys_view)   # dict_keys(['a', 'b', 'c'])
print(values_view) # dict_values([1, 2, 3])
print(items_view)  # dict_items([('a', 1), ('b', 2), ('c', 3)])

# 视图对象是动态的
my_dict['d'] = 4
print(keys_view)   # dict_keys(['a', 'b', 'c', 'd'])（自动更新）

# 转换为列表（如果需要固定快照）
keys_list = list(my_dict.keys())
print(keys_list)   # ['a', 'b', 'c', 'd']
```

### 十、字典的常用函数速查表

| **删除并返回** | `dict.pop(key[, default])` | 删除并返回值 |

| **清空** | `dict.clear()` | 清空所有键值对 |

### 十一、字典的嵌套
字典可以嵌套使用，创建复杂的数据结构。

```python
# 嵌套字典示例：学生信息
students = {
    "1001": {
        "name": "张三",
        "age": 20,
        "scores": {"math": 85, "english": 90, "chinese": 88}
    },
    "1002": {
        "name": "李四",
        "age": 21,
        "scores": {"math": 92, "english": 85, "chinese": 90}
    }
}

# 访问嵌套数据
print(students["1001"]["name"])              # 张三
print(students["1001"]["scores"]["math"])    # 85

# 修改嵌套数据
students["1002"]["scores"]["math"] = 95
print(students["1002"]["scores"]["math"])    # 95

# 添加嵌套数据
students["1001"]["scores"]["physics"] = 82
print(students["1001"]["scores"])  
# {'math': 85, 'english': 90, 'chinese': 88, 'physics': 82}

# 遍历嵌套字典
for stu_id, info in students.items():
    print(f"学号: {stu_id}")
    for key, value in info.items():
        if key == "scores":
            print(f"  成绩: {value}")
        else:
            print(f"  {key}: {value}")
```

```python
# 使用字典实现斐波那契数列缓存
fib_cache = {}

def fibonacci(n):
    if n in fib_cache:
        return fib_cache[n]
    
    if n <= 1:
        result = n
    else:
        result = fibonacci(n-1) + fibonacci(n-2)
    
    fib_cache[n] = result
    return result

print(fibonacci(10))  # 55
print(fibonacci(20))  # 6765
print(f"缓存大小: {len(fib_cache)}")
```

### 十三、字典的复制（浅拷贝 vs 深拷贝）
```python
import copy

# 原始字典（包含嵌套结构）
original = {
    "name": "Alice",
    "age": 25,
    "scores": [85, 90, 88]
}

# 浅拷贝方式
shallow_copy1 = original.copy()
shallow_copy2 = dict(original)
shallow_copy3 = original

# 修改浅拷贝的外层数据
shallow_copy1["name"] = "Bob"
print(original["name"])  # Alice（不受影响）

# 修改浅拷贝的嵌套数据
shallow_copy1["scores"].append(95)
print(original["scores"])  # [85, 90, 88, 95]（受影响！）

# 深拷贝（创建完全独立的副本）
deep_copy = copy.deepcopy(original)
deep_copy["scores"].append(100)
print(original["scores"])  # [85, 90, 88, 95]（不受影响）
```

| 复制方式 | 代码 | 外层独立 | 嵌套对象独立 |
| --- | --- | --- | --- |
| 赋值 | `dict2 = dict1` | ❌ | ❌ |
| 浅拷贝 | `dict1.copy()` | ✅ | ❌ |
| 深拷贝 | `copy.deepcopy(dict1)` | ✅ | ✅ |


### 十四、字典 vs 其他数据结构对比
| 特性 | 字典（Dict） | 列表（List） | 元组（Tuple） | 集合（Set） |
| --- | --- | --- | --- | --- |
| **有序性** | Python 3.7+ 有序 | ✅ 有序 | ✅ 有序 | ❌ 无序 |
| **可变性** | ✅ 可变 | ✅ 可变 | ❌ 不可变 | ✅ 可变 |
| **访问方式** | 通过键 | 通过索引 | 通过索引 | 不支持索引 |
| **元素唯一性** | 键唯一 | 允许重复 | 允许重复 | 元素唯一 |
| **键的类型** | 不可变类型 | 无键概念 | 无键概念 | 可哈希类型 |
| **存储结构** | 哈希表 | 动态数组 | 动态数组 | 哈希表 |
| **查找复杂度** | O(1) | O(n) | O(n) | O(1) |
| **定义符号** | `{}` 或 `dict()` | `[]` 或 `list()` | `()` 或 `tuple()` | `{}` 或 `set()` |


### 十五、Python 3.7+ 字典特性
从 Python 3.7 开始，字典**保持插入顺序**（之前是无序的）。

```python
# Python 3.7+ 保持插入顺序
dict1 = {"b": 2, "a": 1, "c": 3}
print(dict1)  # {'b': 2, 'a': 1, 'c': 3}（保持插入顺序）

# 遍历时按插入顺序
for key in dict1:
    print(key)  # b, a, c

# popitem() 删除最后插入的元素
dict1 = {"first": 1, "second": 2, "third": 3}
print(dict1.popitem())  # ('third', 3)
```

### 十六、完整综合示例
```python
# 学生成绩管理系统（使用字典）

class StudentManager:
    def __init__(self):
        self.students = {}  # 学号 -> 学生信息
    
    def add_student(self, student_id, name, age, scores=None):
        """添加学生"""
        if student_id in self.students:
            print(f"学生 {student_id} 已存在")
            return False
        
        self.students[student_id] = {
            "name": name,
            "age": age,
            "scores": scores or {}
        }
        print(f"成功添加学生: {name}")
        return True
    
    def update_score(self, student_id, subject, score):
        """更新学生成绩"""
        if student_id not in self.students:
            print(f"学生 {student_id} 不存在")
            return False
        
        student = self.students[student_id]
        student["scores"][subject] = score
        print(f"成功更新 {student['name']} 的 {subject} 成绩为 {score}")
        return True
    
    def get_student_info(self, student_id):
        """获取学生信息"""
        if student_id not in self.students:
            print(f"学生 {student_id} 不存在")
            return None
        
        student = self.students[student_id]
        return {
            "id": student_id,
            "name": student["name"],
            "age": student["age"],
            "scores": student["scores"],
            "average": sum(student["scores"].values()) / len(student["scores"]) if student["scores"] else 0
        }
    
    def get_all_students(self):
        """获取所有学生信息"""
        return [
            self.get_student_info(sid) 
            for sid in self.students
        ]
    
    def get_top_students(self, n=3):
        """获取平均分最高的前 n 名学生"""
        students_with_avg = []
        for sid, info in self.students.items():
            if info["scores"]:
                avg = sum(info["scores"].values()) / len(info["scores"])
                students_with_avg.append((sid, info["name"], avg))
        
        # 按平均分降序排序
        students_with_avg.sort(key=lambda x: x[2], reverse=True)
        return students_with_avg[:n]
    
    def remove_student(self, student_id):
        """删除学生"""
        if student_id in self.students:
            name = self.students[student_id]["name"]
            del self.students[student_id]
            print(f"成功删除学生: {name}")
            return True
        print(f"学生 {student_id} 不存在")
        return False
    
    def display_all(self):
        """显示所有学生信息"""
        if not self.students:
            print("暂无学生信息")
            return
        
        print("\n" + "=" * 60)
        print(f"{'学号':<12}{'姓名':<10}{'年龄':<6}{'平均分':<8}{'成绩详情'}")
        print("=" * 60)
        
        for sid, info in self.students.items():
            avg = sum(info["scores"].values()) / len(info["scores"]) if info["scores"] else 0
            scores_str = ", ".join([f"{sub}:{score}" for sub, score in info["scores"].items()])
            print(f"{sid:<12}{info['name']:<10}{info['age']:<6}{avg:<8.1f}{scores_str}")
        print("=" * 60)


# 使用示例
manager = StudentManager()

# 添加学生
manager.add_student("2024001", "张三", 20, {"数学": 85, "英语": 90})
manager.add_student("2024002", "李四", 21, {"数学": 92, "英语": 88})
manager.add_student("2024003", "王五", 19, {"数学": 78, "英语": 85})

# 更新成绩
manager.update_score("2024001", "数学", 90)

# 获取学生信息
info = manager.get_student_info("2024001")
if info:
    print(f"\n学生详情: {info}")

# 显示所有学生
manager.display_all()

# 获取成绩最好的学生
print("\n=== 成绩优异学生 ===")
top_students = manager.get_top_students(2)
for sid, name, avg in top_students:
    print(f"{name} ({sid}): 平均分 {avg:.1f}")
```

### 十七、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 字典是无序的 | Python 3.7+ 保持插入顺序 |
| 使用 `{}` 创建空字典 | ✅ 正确，空集合要用 `set()` |
| 任何类型都可以作为键 | ❌ 键必须是**不可变**类型 |
| `dict[key]` 访问不存在的键 | ❌ 抛出 `KeyError`，用 `get()` 安全 |
| 使用 `in` 检查值 | `key in dict` 检查键，值用 `value in dict.values()` |
| 遍历时修改字典 | ❌ 可能导致错误，需转换为列表 |
| `dict.copy()` 是深拷贝 | ❌ 是浅拷贝，嵌套对象会被共享 |
| `update()` 覆盖整个字典 | ✅ 更新/添加，不会删除未指定的键 |


```python
# 错误示例合集
# 1. 键的类型
d = {}
# d[[1, 2]] = "value"  # ❌ TypeError: unhashable type: 'list'
d[(1, 2)] = "value"    # ✅ 元组可以

# 2. 访问不存在的键
d = {"a": 1}
# print(d["b"])        # ❌ KeyError
print(d.get("b"))      # ✅ 返回 None

# 3. 遍历时修改
d = {"a": 1, "b": 2, "c": 3}
# for k in d:
#     if k == "b":
#         del d[k]      # ❌ RuntimeError: dictionary changed size during iteration
for k in list(d.keys()):  # ✅ 转换为列表
    if k == "b":
        del d[k]

# 4. 字典比较
d1 = {"a": 1, "b": 2}
d2 = {"b": 2, "a": 1}
print(d1 == d2)  # True（内容相同，顺序不影响）
```

> 📌 **核心记忆**：
>
> + 字典是**键值对**集合，用 `{}` 定义，空字典用 `{}`
> + 键必须**唯一**、**不可变**（字符串、数字、元组）
> + 通过**键**访问值，不用索引
> + 安全访问用 `get()`，避免 `KeyError`
> + Python 3.7+ 保持**插入顺序**
> + 视图对象（`keys()`、`values()`、`items()`）是**动态**的
> + 适用于**映射关系**、**快速查找**、**配置管理**、**数据分组**
