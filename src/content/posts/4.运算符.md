### 一、算术运算符
| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `+` | 加 | `20 + 10` | 30 |
| `-` | 减/取负 | `20 - 10` | 10 |
| `*` | 乘 | `20 * 10` | 200 |
| `/` | 除（返回浮点数） | `20 / 10` | 2.0 |
| `//` | 整除（向下取整） | `10 // 3` | 3 |
| `%` | 取模（求余数） | `20 % 10` | 0 |
| `**` | 幂运算 | `2 ** 3` | 8 |


#### 代码示例
```python
a, b = 20, 10
print(f"{a} + {b} = {a + b}")   # 20 + 10 = 30
print(f"{a} - {b} = {a - b}")   # 20 - 10 = 10
print(f"{a} * {b} = {a * b}")   # 20 * 10 = 200
print(f"{a} / {b} = {a / b}")   # 20 / 10 = 2.0（注意结果是浮点数）
print(f"{a} % {b} = {a % b}")   # 20 % 10 = 0
print(f"2 ** 3 = {2 ** 3}")     # 8
print(f"10 // 3 = {10 // 3}")   # 3（向下取整）

# 字符串复用技巧
print("-" * 30)  # 输出 30 个 "-"
```

#### ⚠️ 注意事项
| 易错点 | 说明 | 正确理解 |
| --- | --- | --- |
| `/` 运算 | 结果总是浮点数 | `10 / 2` → `5.0` |
| `//` 整除 | 向下取整（不是截断） | `-10 // 3` → `-4`（因为 -3.33 向下取整为 -4） |
| `%` 取模 | 结果的符号与除数相同 | `-10 % 3` → `2` |


### 二、赋值运算符
| 运算符 | 说明 | 等价形式 |
| --- | --- | --- |
| `=` | 赋值 | `a = 1` |
| `+=` | 加法赋值 | `a += 2` → `a = a + 2` |
| `-=` | 减法赋值 | `a -= 2` → `a = a - 2` |
| `*=` | 乘法赋值 | `a *= 2` → `a = a * 2` |
| `/=` | 除法赋值 | `a /= 2` → `a = a / 2` |
| `//=` | 整除赋值 | `a //= 2` → `a = a // 2` |
| `%=` | 模赋值 | `a %= 2` → `a = a % 2` |
| `**=` | 幂赋值 | `a **= 2` → `a = a ** 2` |
| `:=` | 海象运算符（Python 3.8+） | 表达式中同时赋值并返回值 |


#### 代码示例
```python
# 普通赋值
num3, num4 = 30, 40
num5 = num3 + num4
print(num5)  # 70

# 复合赋值
num3 += 50   # num3 = num3 + 50
print(num3)  # 80

# 海象运算符（Python 3.8+）
num1 = 20
print((num2 := 3 ** 2) > num1)  # 先赋值 num2=9，再判断 9 > 20
print(num2)  # 9
```

### 三、比较运算符
| 运算符 | 说明 | 示例 |
| --- | --- | --- |
| `==` | 相等 | `10 == 20` → `False` |
| `!=` | 不等 | `10 != 20` → `True` |
| `>` | 大于 | `10 > 20` → `False` |
| `<` | 小于 | `10 < 20` → `True` |
| `>=` | 大于等于 | `10 >= 20` → `False` |
| `<=` | 小于等于 | `10 <= 20` → `True` |


#### 代码示例
```python
num1, num2 = 10, 20
print(num1 == num2)   # False
print(num1 != num2)   # True
print(num1 > num2)    # False
print(num1 < num2)    # True

# 字符串比较（按 ASCII 码逐字符比较）
print('5' > '6')      # False（'5'的ASCII码 53 < '6'的ASCII码 54）
print('15' > '6')     # False（先比较 '1' 和 '6'，'1' 的 ASCII 码更小）
```

#### ⚠️ 重点理解
| 要点 | 说明 |
| --- | --- |
| 不同类型比较 | Python 3 中不同类型不能比较大小（会抛出 `TypeError`） |
| 字符串比较 | 从左到右逐字符比较 ASCII/Unicode 码 |
| 链式比较 | `10 < 20 < 30` 等价于 `10 < 20 and 20 < 30` |


### 四、逻辑运算符
| 运算符 | 说明 | 短路规则 |
| --- | --- | --- |
| `and` | 与：两边都为 True 才返回 True | 左为 False 则返回左值，不计算右值 |
| `or` | 或：一边为 True 就返回 True | 左为 True 则返回左值，不计算右值 |
| `not` | 非：取反 | 返回布尔值 |


#### 代码示例
```python
b1, b2 = False, True
print(b1 and b2)   # False
print(b1 or b2)    # True
print(not b1)      # True

# 短路逻辑与返回值（重要！）
print(5 and 8)     # 8（5 为 True，返回右值 8）
print(0 and 8)     # 0（0 为 False，返回左值 0）
print(5 or 8)      # 5（5 为 True，返回左值 5）
print(0 or 8)      # 8（0 为 False，返回右值 8）
print(not 5)       # False（5 为 True，取反得 False）
```

#### 💡 核心理解：逻辑运算符的返回值
```python
# and: 左为假返回左，否则返回右
x and y  →  if x is False: return x else: return y

# or: 左为真返回左，否则返回右
x or y   →  if x is True: return x else: return y

# not: 总是返回布尔值
not x    →  return False if x else True
```

### 五、位运算符
| 运算符 | 说明 | 示例 |
| --- | --- | --- |
| `&` | 按位与 | `17 & 13` |
| `|` | 按位或 | `a | b` |
| `^` | 按位异或 | `17 ^ 13` |
| `~` | 按位取反 | `~17` |
| `<<` | 左移 | `17 << 1` |
| `>>` | 右移 | `17 >> 3` |


#### 1. 原码、反码、补码（预备知识）
| 概念 | 正数（如 +7） | 负数（如 -7） |
| --- | --- | --- |
| **原码** | 00000111 | 10000111 |
| **反码** | 00000111 | 11111000 |
| **补码** | 00000111 | 11111001 |


> 💡 **位运算规则**：计算机中位运算以**补码**形式进行
>

#### 2. 正数位运算示例
```python
num1, num2 = 17, 13
# 17 = 00010001
# 13 = 00001101

print(f"与运算: {num1} & {num2} = {num1 & num2:08b}")   # 00000001
print(f"或运算: {num1} | {num2} = {num1 | num2:08b}")   # 00011101
print(f"异或: {num1} ^ {num2} = {num1 ^ num2:08b}")     # 00011100
print(f"取反: ~{num1} = {~num1:08b}")                   # 补码表示
```

**运算图解**：

```plain
    17:  00010001
    13:  00001101
    ─────────────
&   结果: 00000001  (1)
|   结果: 00011101  (29)
^   结果: 00011100  (28)
~17 结果: 11101110  (-18 的补码表示)
```

#### 3. 移位运算
```python
num1 = 17   # 00010001

# 左移（右补0）
print(f"17 << 1 = {17 << 1}")   # 34 (00100010)
print(f"17 << 2 = {17 << 2}")   # 68 (01000100)

# 右移（正数左补0，负数左补1）
print(f"17 >> 3 = {17 >> 3}")   # 2 (00000010)
```

#### ⚠️ 负数位运算注意事项
```python
# 负数右移结果是向负无穷取整（向下取整）
num2 = -12
print(f"-12 >> 3 = {-12 >> 3}")   # -2（不是 -1）
# 因为 -12 / 8 = -1.5，向下取整得 -2
```

### 六、成员运算符
| 运算符 | 说明 | 示例 |
| --- | --- | --- |
| `in` | 在序列中找到值返回 `True` | `1 in [1,2,3]` → `True` |
| `not in` | 在序列中找不到值返回 `True` | `5 not in [1,2,3]` → `True` |


```python
test_list = [1, 2, 3, 4, 5]
print(1 in test_list)      # True
print(20 not in test_list) # True

# 字符串成员运算
print('he' in 'hello')     # True
print('hi' not in 'hello') # True
```

### 七、身份运算符
| 运算符 | 说明 |
| --- | --- |
| `is` | 判断两个变量是否引用**同一个对象**（内存地址相同） |
| `is not` | 判断两个变量是否引用**不同对象** |


#### `is` vs `==` 的区别（重点！）
| 运算符 | 比较内容 | 说明 |
| --- | --- | --- |
| `is` | **内存地址** | 是否为同一个对象 |
| `==` | **值** | 内容是否相等 |


```python
# 场景1：变量指向同一个对象
a = [1, 2, 3]
b = a                    # b 和 a 指向同一个列表
print(b is a)            # True（同一对象）
print(b == a)            # True（值相等）

# 场景2：切片创建新对象
b = a[:]                 # 创建副本
print(b is a)            # False（不同对象）
print(b == a)            # True（值相等）

# 小整数池影响
m, n = 20, 20
print(m is n)            # True（小整数池复用）
```

#### 💡 使用建议
| 场景 | 推荐使用 | 原因 |
| --- | --- | --- |
| 比较值 | `==` | 关注内容是否相等 |
| 与 `None` 比较 | `is` | None 是单例，`is None` 是规范 |
| 判断对象身份 | `is` | 需要确认是否为同一对象 |


```python
# 推荐写法
if x is None:
    pass

# 不推荐（虽然可行）
if x == None:
    pass
```

### 八、运算符优先级（从高到低）
| 优先级 | 运算符 | 说明 |
| --- | --- | --- |
| 1 | `()` | 括号 |
| 2 | `**` | 幂运算 |
| 3 | `~` `+` `-` | 按位取反、正号、负号 |
| 4 | `*` `/` `//` `%` | 乘、除、整除、取模 |
| 5 | `+` `-` | 加、减 |
| 6 | `<<` `>>` | 左移、右移 |
| 7 | `&` | 按位与 |
| 8 | `^` `|`  | <font style="color:rgb(15, 17, 21);">按位异或、按位或</font> |
| 9 | `==` `!=` `>` `>=` `<` `<=` | 比较运算符 |
| 10 | `is` `is not` `in` `not in` | 身份/成员运算符 |
| 11 | `not` | 逻辑非 |
| 12 | `and` | 逻辑与 |
| 13 | `or` | 逻辑或 |
| 14 | `=` `+=` `-=` 等 | 赋值运算符 |
| 15 | `:=` | 海象运算符 |


#### 优先级示例
```python
# 算术优先
result = 2 + 3 * 4     # 14（先乘后加）
result = (2 + 3) * 4   # 20（括号优先）

# 比较与逻辑
result = 5 > 3 and 2 < 4   # True（比较优先于逻辑）

# 复杂示例
result = not 5 > 3 or 2 == 2 and 3 < 5
# 等价于 (not (5 > 3)) or ((2 == 2) and (3 < 5))
# = False or (True and True) = True
```

### 九、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| `/` 结果类型 | 总是浮点数，不是整数 |
| `//` 整除方向 | 向下取整，负数时注意 |
| `%` 符号规则 | 结果符号与除数相同 |
| `and`/`or` 返回值 | 返回操作数的值，不一定是布尔值 |
| `is` 与 `==` 混淆 | `is` 比较地址，`==` 比较值 |
| 位运算符号 | `&` 不是 `and`，` |
| 负数右移 | 向负无穷取整，不是简单截断 |


### 十、速查卡片
```python
# 算术运算
+, -, *, /, //, %, **

# 赋值运算
=, +=, -=, *=, /=, //=, %=, **=, :=

# 比较运算
==, !=, >, <, >=, <=

# 逻辑运算
and, or, not  # 注意短路逻辑

# 位运算
&, |, ^, ~, <<, >>

# 成员运算
in, not in

# 身份运算
is, is not

# 常用技巧
print("-" * 30)                    # 字符串复用
result = 1 in [1, 2, 3]            # 成员判断
if x is None: pass                 # None 判断规范
```

> 📌 **核心记忆**：
>
> + `/` 得浮点，`//` 向下取整
> + `and`/`or` 返回操作数，有短路特性
> + `is` 比地址，`==` 比值
> + 与 `None` 比较用 `is`
> + 不确定优先级时**加括号**最安全
>
