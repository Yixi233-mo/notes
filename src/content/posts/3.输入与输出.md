### 一、输入函数 input()
#### 基本语法
```python
变量 = input("提示信息")
```

#### 核心特性
| 特性 | 说明 |
| --- | --- |
| 返回值类型 | **永远是字符串 (str)** |
| 执行行为 | 程序暂停，等待用户输入，按回车结束 |
| 提示信息 | 可选参数，会在输入前显示 |


#### 示例代码
```python
input_str = input("请输入：")
print("input_str数据类型为:", type(input_str))  # <class 'str'>
print(input_str)  # 输出用户输入的内容
```

#### ⚠️ 重要注意事项
```python
age = input("请输入年龄：")  # 用户输入 18
print(age + 1)  # ❌ TypeError! age 是字符串，不能与整数相加

# 正确做法：类型转换
age = int(input("请输入年龄："))  # 转换为整数
print(age + 1)  # ✅ 输出 19
```

> 💡 **记忆口诀**：input 进，字符串出；要计算，先转换。
>

### 二、输出函数 print()
#### 1. 基本用法
```python
# 输出单个内容
print("Hello Python")

# 输出多个内容（用逗号分隔，自动添加空格）
print("Hello", "Python")   # Hello Python

# 输出多个内容（自定义分隔符）
print("Hello", "Python", sep="---")  # Hello---Python
```

#### 2. end 参数控制结尾
| end 值 | 效果 | 示例 |
| --- | --- | --- |
| `end="\n"` | 换行结尾（默认） | `print("a"); print("b")` → a\nb |
| `end=""` | 无结尾，下一行继续 | `print("a", end=""); print("b")` → ab |
| `end=" "` | 空格结尾 | `print("a", end=" "); print("b")` → a b |


```python
print("使用\\n结尾", end="\n")  # 等同于 print("使用\\n结尾")
print('使用空字符串结尾', end="")
print("Hello")                  # 直接跟在上一行后面
```

### 三、格式化输出（四种方式）
#### 方式一：% 格式化（C语言风格）
```python
int1 = 10
float1 = 3.14159
str1 = "int1 = %d, float1 = %f" % (int1, float1)
print(str1)  # int1 = 10, float1 = 3.141590
```

**格式符号表**：

| 格式符号 | 说明 | 示例 |
| --- | --- | --- |
| `%d` | 十进制整数 | `%d` % 10 → '10' |
| `%f` | 浮点数 | `%f` % 3.14 → '3.140000' |
| `%.nf` | 指定小数位数 | `%.2f` % 3.14159 → '3.14' |
| `%s` | 字符串 | `%s` % 'hello' → 'hello' |
| `%o` | 八进制整数 | `%o` % 10 → '12' |
| `%x` | 十六进制整数 | `%x` % 255 → 'ff' |
| `%e` | 科学计数法 | `%e` % 1000 → '1.000000e+03' |


```python
# 指定小数位数的示例
pi = 3.1415926
print("圆周率保留2位小数：%.2f" % pi)   # 圆周率保留2位小数：3.14
```

#### 方式二：str.format() 方法
**① 默认顺序（推荐入门使用）**

```python
int1, float1, bool1 = 10, 3.14159, True
str2 = "int1 = {}, float1 = {}, bool1 = {}".format(int1, float1, bool1)
print(str2)  # int1 = 10, float1 = 3.14159, bool1 = True
```

**② 指定位置（可重复使用）**

```python
str2 = "int1 = {0}, float1 = {1}, bool1 = {2}".format(int1, float1, bool1)
print(str2)  # int1 = 10, float1 = 3.14159, bool1 = True

# 可以重复使用同一个位置
print("{0} {0} {1}".format("hello", "world"))  # hello hello world
```

**③ 参数名称（可读性最好）**

```python
str2 = "int1 = {i1}, float1 = {f1}, bool1 = {b1}".format(i1=int1, f1=float1, b1=bool1)
print(str2)  # int1 = 10, float1 = 3.14159, bool1 = True
```

**④ 数字格式化（高级用法）**

```python
float1 = 31415.9
str2 = "{:*^20,.2f}".format(float1)
print(str2)  # *****31,415.90******
```

**格式化语法解析**：

```plain
{ : * ^ 20 , .2f }
  │ │ │  │  │   └── 小数后2位
  │ │ │  │  └────── 千位分隔符
  │ │ │  └───────── 总宽度20个字符
  │ │ └──────────── 居中对齐（<左对齐，>右对齐）
  │ └────────────── 填充字符为 *
  └──────────────── 格式说明符开始
```

**对齐与填充示例**：

```python
num = 3.14
print("{:*<10}".format(num))   # 3.14******
print("{:*>10}".format(num))   # ******3.14
print("{:*^10}".format(num))   # ***3.14***
```

**⑤ 转义大括号**

```python
# 使用 {{ 和 }} 来输出字面量 {}
print("{0} 对应的位置是 {0}".format("hello"))   # hello 对应的位置是 hello
print("{} 对应的位置是 {{0}}".format("hello"))  # hello 对应的位置是 {0}
```

#### 方式三：f-string（Python 3.6+，推荐）
> ⭐ **最现代、最简洁、最推荐的方式**
>

```python
int1 = 10
float1 = 3.14159
bool1 = True

# 基础用法
str3 = f"int1 = {int1}, float1 = {float1}, bool1 = {bool1}"
print(str3)  # int1 = 10, float1 = 3.14159, bool1 = True
```

**① 使用 **`=`** 输出变量名和值（调试神器）**

```python
int1 = 10
float1 = 3.14159
str3 = f"{int1 = }, {float1 = }"
print(str3)  # int1 = 10, float1 = 3.14159
# 注意：= 左右会自动添加空格
```

**② 表达式计算**

```python
a, b = 5, 3
print(f"{a} + {b} = {a + b}")   # 5 + 3 = 8
print(f"{a} > {b}: {a > b}")    # 5 > 3: True
```

**③ 格式化数字（同 format 语法）**

```python
pi = 3.1415926
print(f"{pi:.2f}")        # 3.14
print(f"{pi:10.2f}")      #       3.14
print(f"{pi:>10.2f}")     #       3.14
print(f"{pi:<10.2f}")     # 3.14      
print(f"{pi:^10.2f}")     #    3.14   

# 千位分隔符
num = 12345678
print(f"{num:,}")   # 12,345,678
print(f"{num:_}")   # 12_345_678

# 百分比
ratio = 0.256
print(f"{ratio:.1%}")   # 25.6%
```

**④ 转义大括号**

```python
int1 = 10
str3 = f"{{int1 = }}, {{float1 = }}"
print(str3)  # {int1 = }, {float1 = }
```

#### 方式四：四种方式对比总结
| 方式 | 语法示例 | 优点 | 缺点 | 推荐度 |
| --- | --- | --- | --- | --- |
| % 格式化 | `"%d" % num` | 简洁，C语言习惯 | 类型不匹配易出错 | ⭐⭐ |
| format() | `"{}".format(num)` | 功能强大，灵活性高 | 写法稍长 | ⭐⭐⭐ |
| f-string | `f"{num}"` | 简洁直观，性能好 | Python 3.6+ 可用 | ⭐⭐⭐⭐⭐ |


```python
# 四种方式对比
name = "张三"
age = 18

# 1. % 格式化
print("姓名：%s，年龄：%d" % (name, age))

# 2. format()
print("姓名：{}，年龄：{}".format(name, age))

# 3. f-string（推荐）
print(f"姓名：{name}，年龄：{age}")

# 4. 普通拼接（不推荐）
print("姓名：" + name + "，年龄：" + str(age))
```

### 四、完整案例演示
```python
# 综合示例：用户信息录入与格式化输出

# 输入部分
name = input("请输入姓名：")
age = int(input("请输入年龄："))  # 注意类型转换
score = float(input("请输入成绩："))

# 输出部分 - 使用 f-string（推荐）
print("\n" + "="*30)
print(f"用户信息")
print("="*30)
print(f"姓名：{name}")
print(f"年龄：{age}")
print(f"成绩：{score:.1f}")

# 调试输出（使用 = 符号）
print(f"\n调试信息：{name = }, {age = }, {score = }")

# 格式化表格输出
print("\n" + "-"*30)
print(f"{'姓名':<10}{'年龄':<10}{'成绩':<10}")
print("-"*30)
print(f"{name:<10}{age:<10}{score:<10.1f}")
```

### 五、易错点总结
| 易错点 | 正确做法 |
| --- | --- |
| 忘记 input 返回的是字符串 | 使用 `int(input())` 转换类型 |
| `print(a, b)` 误以为无空格 | 默认有空格，用 `sep=""` 去除 |
| f-string 忘记加 `f` 前缀 | `f"{var}"` 不能写成 `"{var}"` |
| 格式化时花括号数量不匹配 | `{​{` 转义为 `{`，`}​}` 转义为 `}` |
| `%` 格式化时变量用元组包裹 | `"%d %s" % (num, str)` |


### 六、速查卡片
```python
# 输入
var = input("提示")        # 返回 str
num = int(input("提示"))   # 转换为整数

# 输出基础
print(value)               # 输出并换行
print(a, b, c)             # 多个值，空格分隔
print(a, b, sep="")        # 自定义分隔符
print(a, end="")           # 不换行

# 格式化（推荐 f-string）
print(f"{var}")            # 变量
print(f"{var = }")         # 变量名=值（调试）
print(f"{num:.2f}")        # 2位小数
print(f"{num:,}")          # 千位分隔符
print(f"{num:>10}")        # 右对齐，宽度10
print(f"{{花括号}}")       # 输出字面量 {}
```

> 📌 **核心记忆**：  
>
> + `input()` 永远返回字符串，计算前先转换  
> + `print()` 默认换行，用 `end` 控制  
> + **f-string** 是最推荐的格式化方式，简洁又强大
>
