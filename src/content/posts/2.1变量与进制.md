## 📌 一、变量基础
### 1.1 变量的本质
> **变量**：可以理解为可以赋给值的标签，指向特定的值
>

### 1.2 变量赋值与交换
```python
# 变量交换（Python特色）
a, b = 10, 15
a, b = b, a  # 直接交换，无需临时变量
```

### 1.3 变量的内存地址
```python
result = x + y
print(result, id(result))  # id()查看内存地址
```

## 📌 二、变量的创建方式
### 2.1 单变量创建
```python
name = 'xiaoming'
age = 18
```

### 2.2 多变量同时创建
```python
name, age, weight = '小米', 18, 89  # 数量必须一致
```

### 2.3 多个变量相同值
```python
a = b = c = 10  # 三个变量都指向10
```

## 📌 四、变量与常量的区别
| 类型 | 定义 | 命名规范 | 示例 |
| --- | --- | --- | --- |
| **变量** | 值可以改变 | 小写+下划线 | `msg = 'hello'` |
| **常量** | 值不可改变 | 全大写 | `PI = 3.1415926` |


```python
# 变量可修改
msg = 'hello world'
msg = 'hello python'  # 地址会改变

# 常量（约定俗成，Python无真正常量）
PI = 3.1415926
E = 2.71828
```

## 📌 五、进制转换
### 5.1 进制的表示方法
| 进制 | 前缀 | 示例 | 十进制值 |
| --- | --- | --- | --- |
| 二进制 | `0b` | `0b10` | 2 |
| 八进制 | `0o` | `0o10` | 8 |
| 十进制 | 无 | `10` | 10 |
| 十六进制 | `0x` | `0xA` | 10 |


### 5.2 进制转换函数
| 函数 | 功能 | 示例 |
| --- | --- | --- |
| `bin()` | 转二进制 | `bin(506)` → `'0b111111010'` |
| `oct()` | 转八进制 | `oct(1356)` → `'0o2514'` |
| `hex()` | 转十六进制 | `hex(1356)` → `'0x54c'` |
| `int()` | 转十进制 | `int('1010',2)` → `10` |


### 5.3 实战示例
```python
# 二进制 → 十进制
print(0b11011)    # 输出: 27

# 十进制 → 二进制
print(bin(506))   # 输出: 0b111111010

# 十六进制 → 十进制
print(0x3E4)      # 输出: 996

# 十进制 → 十六进制
print(hex(1356))  # 输出: 0x54c

# 十进制 → 八进制
print(oct(11111111))  # 输出: 0o52330607

# 十六进制 → 二进制
print(bin(0xBBB))     # 输出: 0b101110111011

# 八进制 → 二进制
print(bin(0o777))     # 输出: 0b111111111
```
