### 一、函数的概念
**函数**是带名字的代码块，用于完成具体的任务，可以重复使用。

| 特性 | 说明 |
| --- | --- |
| **可重用性** | 避免重复编写相同代码 |
| **模块化** | 将复杂任务分解为小任务 |
| **可维护性** | 修改一处，全局生效 |
| **可测试性** | 可独立测试每个函数 |


```python
# 没有函数时，代码冗余
row = 2
while row > 0:
    print("*" * 3)
    row -= 1

# 再次需要时，必须重复编写
row = 2
while row > 0:
    print("*" * 3)
    row -= 1

# 使用函数，代码复用
def printStar():
    row = 2
    while row > 0:
        print("*" * 3)
        row -= 1

printStar()  # 调用一次
printStar()  # 再调用一次
```

### 二、函数的定义与调用
#### 2.1 语法
```python
def 函数名(参数列表):
    """函数说明文档（可选）"""
    函数体
    [return 返回值]
```

#### 2.2 定义规则
| 规则 | 说明 |
| --- | --- |
| `def` 关键字 | 函数定义以 `def` 开头 |
| 函数名 | 遵循标识符命名规则，动词，小写+下划线 |
| 参数列表 | 放在圆括号 `()` 中，多个参数用逗号分隔 |
| 冒号 `:` | 参数列表后必须有冒号 |
| 缩进 | 函数体必须缩进（通常4个空格） |
| 文档字符串 | 可选，用三个引号括起来 |
| `return` | 可选，返回值，无 return 返回 `None` |


```python
def calculate_area(length, width):
    """计算矩形的面积"""
    area = length * width
    return area

# 调用函数
result = calculate_area(5, 3)
print(result)  # 15
```

#### 2.3 注意事项
+ 函数必须**先定义，后调用**
+ 定义时只是声明，不执行代码
+ 调用时才会真正执行

```python
# 正确：先定义后调用
def say_hello():
    print("Hello!")

say_hello()  # Hello!

# 错误：先调用后定义
# say_goodbye()  # NameError: name 'say_goodbye' is not defined
# def say_goodbye():
#     print("Goodbye!")
```

### 三、函数的参数
#### 3.1 形参和实参
| 术语 | 说明 | 位置 |
| --- | --- | --- |
| **形参**（形式参数） | 定义函数时指定的参数 | 函数定义中 |
| **实参**（实际参数） | 调用函数时传递的参数 | 函数调用中 |


```python
def add(a, b):    # a, b 是形参
    return a + b

result = add(3, 5)  # 3, 5 是实参
```

**内存理解**：

+ 形参在函数定义时没有分配存储空间
+ 函数调用时，在栈区分配空间，实参的值赋给形参
+ 函数执行结束，栈空间释放

#### 3.2 参数传递机制
Python 中，变量是对象的**引用**。参数传递取决于对象的可变性。

**不可变类型**（数字、字符串、元组）：

+ 类似于值传递
+ 函数内部修改不影响外部

```python
def change_int(a):
    print(f"函数内修改前: {id(a)}")
    a = 10
    print(f"函数内修改后: {id(a)}")

b = 2
print(f"函数外: {id(b)}")  # 例如: 140711474555352
change_int(b)              # 函数内修改前地址相同，修改后地址改变
print(b)                   # 2（外部不变）
```

**可变类型**（列表、字典、集合）：

+ 类似于引用传递
+ 函数内部修改会影响外部

```python
def change_list(my_list):
    my_list[0] = 100
    print(f"函数内: {my_list}")

original = [1, 2, 3]
change_list(original)
print(original)  # [100, 2, 3]（外部被修改）
```

`var1 *= 2`** 与 **`var1 = var1 * 2`** 的区别**：

| 操作 | 效果 |
| --- | --- |
| `var1 *= 2` | 使用原地址（原地操作） |
| `var1 = var1 * 2` | 开辟新空间 |


```python
def test_operation(var1):
    print(f"原始 id: {id(var1)}")
    var1 *= 2
    print(f"var1 *= 2 后 id: {id(var1)}")
    var1 = var1 * 2
    print(f"var1 = var1 * 2 后 id: {id(var1)}")

list1 = [1, 2, 3]
test_operation(list1)
# 列表是可变类型，*= 原地修改，地址不变；= * 创建新列表，地址改变
```

#### 3.3 参数类型详解
##### 1. 必须参数（位置参数）
调用时参数的数量和位置必须与定义一致。

```python
def func(a, b, c):
    print(a, b, c)

func(1, 2, 3)   # 1 2 3
# func(1, 2)    # TypeError: missing 1 required positional argument
```

##### 2. 关键字参数
调用时使用参数名指定值，顺序可以不同。

```python
def print_info(name, age):
    print(f"姓名: {name}, 年龄: {age}")

print_info(name="张三", age=18)
print_info(age=20, name="李四")  # 顺序可调换
```

##### 3. 默认值参数
为形参指定默认值，调用时可省略。

```python
def print_info(name, age=18):
    print(f"姓名: {name}, 年龄: {age}")

print_info("张三")      # 年龄使用默认值 18
print_info("李四", 25)  # 年龄使用传入的 25
```

> ⚠️ **注意**：默认参数必须放在非默认参数之后
>

```python
# 正确
def func(a, b=1, c=2): pass

# 错误
# def func(a=1, b, c=2): pass  # SyntaxError
```

##### 4. 不定长参数
**单星号 **`*args`：接收多余的位置参数，打包为元组

```python
def print_info(num, *args):
    print(f"num: {num}")
    print(f"args: {args}")

print_info(70, 60, 50, 40)
# num: 70
# args: (60, 50, 40)

# 没有传入额外参数时，args 为空元组
print_info(70)  # args: ()
```

**双星号 **`**kwargs`：接收多余的关键字参数，打包为字典

```python
def print_info(num, **kwargs):
    print(f"num: {num}")
    print(f"kwargs: {kwargs}")

print_info(10, name="张三", age=18, city="北京")
# num: 10
# kwargs: {'name': '张三', 'age': 18, 'city': '北京'}
```

**混合使用**：

```python
def func(a, b=1, *args, **kwargs):
    print(f"a={a}, b={b}, args={args}, kwargs={kwargs}")

func(1, 2, 3, 4, x=5, y=6)
# a=1, b=2, args=(3, 4), kwargs={'x': 5, 'y': 6}
```

> ⚠️ **注意**：`*args` 后面的参数必须使用关键字参数传递
>

```python
def func(a, *args, b):
    print(a, args, b)

func(1, 2, 3, b=4)   # 1 (2, 3) 4
# func(1, 2, 3, 4)   # TypeError: missing keyword-only argument 'b'
```

#### 3.4 解包传参
使用 `*` 和 `**` 对序列/字典解包后传递。

```python
def func(a, b, c):
    return a + b + c

# 列表/元组解包
args = (1, 2, 3)
print(func(*args))  # 6

# 字典解包（键名必须与参数名匹配）
kwargs = {"a": 1, "b": 2, "c": 3}
print(func(**kwargs))  # 6
```

#### 3.5 强制位置参数和关键字参数
+ `/` 前的参数必须使用位置传参
+ `*` 后的参数必须使用关键字传参

```python
def func(a, b, /, c, d, *, e, f):
    print(a, b, c, d, e, f)

# 正确调用
func(1, 2, 3, d=4, e=5, f=6)

# 错误调用示例
# func(1, 2, 3, 4, 5, 6)     # e, f 必须用关键字
# func(a=1, b=2, c=3, d=4, e=5, f=6)  # a, b 必须用位置
```

#### 3.6 防止函数修改列表
传递副本而非原对象。

```python
import copy

def modify_list(lst):
    lst.append(100)
    print(f"函数内: {lst}")

original = [1, 2, 3]

# 传递深拷贝，原列表不受影响
modify_list(copy.deepcopy(original))
print(f"函数外: {original}")  # [1, 2, 3]

# 传递副本（浅拷贝）
modify_list(original[:])
print(f"函数外: {original}")  # [1, 2, 3]
```

### 五、返回值
`return` 用于结束函数并返回值。

| 情况 | 返回值 |
| --- | --- |
| `return 表达式` | 返回表达式的值 |
| `return` | 返回 `None` |
| 无 `return` | 返回 `None` |


```python
# 返回单个值
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 8

# 返回多个值（打包为元组）
def get_min_max(numbers):
    return min(numbers), max(numbers)

result = get_min_max([1, 2, 3, 4, 5])
print(result)        # (1, 5)
print(type(result))  # <class 'tuple'>

# 解包返回值
min_val, max_val = get_min_max([1, 2, 3, 4, 5])
print(min_val, max_val)  # 1 5
```

### 六、函数嵌套调用
一个函数内部调用另一个函数。

```python
def function_a():
    print("\t函数 A 开始执行")
    print("\t函数 A 执行中...")
    print("\t函数 A 结束执行")

def function_b():
    print("函数 B 开始执行")
    print("函数 B 执行中...")
    function_a()        # 嵌套调用
    print("函数 B 执行中...")
    print("函数 B 结束执行")

function_b()
```

**执行流程**：

```plain
函数 B 开始执行
函数 B 执行中...
        函数 A 开始执行
        函数 A 执行中...
        函数 A 结束执行
函数 B 执行中...
函数 B 结束执行
```

### 七、变量的作用域
#### 7.1 作用域分类
Python 有 4 种作用域，查找顺序：**L → E → G → B**

| 层级 | 名称 | 说明 |
| --- | --- | --- |
| L | Local（局部） | 函数内部定义 |
| E | Enclosing（嵌套） | 闭包函数外的函数中 |
| G | Global（全局） | 模块级别定义 |
| B | Built-in（内建） | Python 内置的（如 `int`、`print`） |


```python
a = int(2.9)      # B: 内建作用域
b = 0             # G: 全局作用域

def outer():
    c = 1         # E: 嵌套作用域
    def inner():
        d = 2     # L: 局部作用域
        print(d, c, b, a)
    return inner

inner_func = outer()
inner_func()  # 可以访问所有层级的变量
```

#### 7.2 全局变量与局部变量
```python
# 全局变量
total = 100

def add(a, b):
    # 局部变量（与全局变量同名，但不同）
    total = a + b
    print(f"函数内局部变量: {total}")
    return total

add(10, 20)        # 函数内局部变量: 30
print(f"函数外全局变量: {total}")  # 函数外全局变量: 100
```

#### 7.3 `global` 关键字
在函数内修改全局变量时使用。

```python
counter = 0

def increment():
    global counter  # 声明使用全局变量
    counter += 1

increment()
print(counter)  # 1
```

**可变类型全局变量**：修改内容时无需 `global`

```python
my_list = [1, 2, 3]

def add_element():
    my_list.append(4)    # 修改列表内容，无需 global
    print(f"函数内: {my_list}")

add_element()            # 函数内: [1, 2, 3, 4]
print(f"函数外: {my_list}")  # 函数外: [1, 2, 3, 4]

def reassign_list():
    my_list = [5, 6, 7]   # 重新赋值（改变指向），需要 global
    print(f"函数内: {my_list}")

reassign_list()           # 函数内: [5, 6, 7]
print(f"函数外: {my_list}")  # 函数外: [1, 2, 3, 4]（未改变）
```

#### 7.4 `nonlocal` 关键字
修改嵌套作用域中的变量。

```python
def outer():
    x = 10
    print(f"outer 中 x: {x}")
    
    def inner():
        nonlocal x      # 声明使用外层函数的变量
        x = 20
        print(f"inner 中 x: {x}")
    
    inner()
    print(f"outer 中 x 被 inner 修改后: {x}")

outer()
# outer 中 x: 10
# inner 中 x: 20
# outer 中 x 被 inner 修改后: 20
```

#### 7.5 特殊说明
`if`、`for`、`while`、`try/except` 等代码块**不会**创建新的作用域。

```python
# 在 if 块中定义的变量，外部可以访问
if True:
    msg = "Hello"

print(msg)  # Hello（可以访问）

# 在函数中定义的变量，外部不能访问
def test():
    inner_msg = "World"

# print(inner_msg)  # NameError: name 'inner_msg' is not defined
```

### 八、递归函数
#### 8.1 概念
函数调用自身，将大问题分解为相似的小问题。

#### 8.2 递归三要素
1. **递归公式**：问题分解的规律
2. **终止条件**：递归何时结束
3. **问题规模递减**：每次递归向终止条件靠近

#### 8.3 阶乘示例
```python
# 循环实现
def factorial_iterative(n):
    result = 1
    for i in range(1, n + 1):
        result *= i
    return result

# 递归实现
def factorial_recursive(n):
    """n! = n * (n-1)!，且 1! = 1"""
    return n * factorial_recursive(n - 1) if n > 1 else 1

print(factorial_iterative(5))  # 120
print(factorial_recursive(5))  # 120
```

#### 8.4 递归执行流程（以 `factorial(5)` 为例）
```plain
factorial(5) = 5 * factorial(4)
factorial(4) = 4 * factorial(3)
factorial(3) = 3 * factorial(2)
factorial(2) = 2 * factorial(1)
factorial(1) = 1  ← 终止条件

→ 逐层返回：
factorial(2) = 2 * 1 = 2
factorial(3) = 3 * 2 = 6
factorial(4) = 4 * 6 = 24
factorial(5) = 5 * 24 = 120
```

#### 8.5 递归优缺点
| 优点 | 缺点 |
| --- | --- |
| 代码简洁，逻辑清晰 | 函数调用有开销，效率较低 |
| 适合树形结构问题 | 递归深度过大可能导致栈溢出 |


```python
import sys
print(sys.getrecursionlimit())  # 默认递归深度限制（通常1000）
# sys.setrecursionlimit(10000)  # 可调整深度
```

### 九、匿名函数（lambda）
#### 9.1 语法
```python
lambda 参数列表: 表达式
```

| 特点 | 说明 |
| --- | --- |
| 简洁 | 单行表达式，无需 `return` |
| 匿名 | 没有函数名 |
| 受限 | 只能包含一个表达式 |
| 命名空间 | 不能访问参数列表外的变量 |


```python
# 普通函数
def add(x, y):
    return x + y

# lambda 函数
lambda_add = lambda x, y: x + y

print(add(3, 5))        # 8
print(lambda_add(3, 5)) # 8
```

#### 9.2 作为参数传递
```python
def operate(a, b, func):
    return func(a, b)

# 使用普通函数
def add(a, b):
    return a + b

print(operate(3, 5, add))  # 8

# 使用 lambda
print(operate(3, 5, lambda x, y: x + y))  # 8
```

#### 9.3 与内置函数配合
`sorted()`：按指定规则排序

```python
students = [
    {"name": "张三", "age": 36},
    {"name": "李四", "age": 14},
    {"name": "王五", "age": 27}
]

# 按年龄排序
sorted_students = sorted(students, key=lambda x: x["age"])
print(sorted_students)
# [{'name': '李四', 'age': 14}, {'name': '王五', 'age': 27}, {'name': '张三', 'age': 36}]
```

`map()`：对序列中每个元素应用函数

```python
numbers = [1, 2, 3, 4, 5]
squared = map(lambda x: x ** 2, numbers)
print(list(squared))  # [1, 4, 9, 16, 25]
```

`filter()`：过滤序列中满足条件的元素

```python
numbers = [-2, -1, 0, 1, 2, 3]
positive = filter(lambda x: x > 0, numbers)
print(list(positive))  # [1, 2, 3]
```

`reduce()`：对序列进行累积操作（需要导入）

```python
from functools import reduce

numbers = [1, 2, 3, 4, 5]
product = reduce(lambda x, y: x * y, numbers)
print(product)  # 120 = 1*2*3*4*5
```

### 十一、函数使用的最佳实践
| 最佳实践 | 说明 |
| --- | --- |
| 函数名使用动词 | `calculate_sum()`、`get_user_name()` |
| 保持函数简短 | 一个函数只做一件事 |
| 避免修改全局变量 | 使用参数传递和返回值 |
| 添加文档字符串 | 说明函数功能、参数和返回值 |
| 参数数量不宜过多 | 超过 5 个考虑使用字典或对象 |
| 注意可变对象的副作用 | 如需修改，明确说明或传递副本 |


### 十二、完整综合示例
```python
"""
学生成绩管理系统 - 函数版
"""
import copy

# 全局学生列表
students = []

def add_student(name: str, scores: dict) -> bool:
    """添加学生信息"""
    student = {"name": name, "scores": copy.deepcopy(scores)}
    students.append(student)
    print(f"成功添加学生: {name}")
    return True

def calculate_average(scores: dict) -> float:
    """计算平均分"""
    if not scores:
        return 0.0
    return sum(scores.values()) / len(scores)

def get_student_by_name(name: str) -> dict:
    """根据姓名查找学生"""
    for student in students:
        if student["name"] == name:
            return student
    return None

def update_score(name: str, subject: str, score: int) -> bool:
    """更新学生成绩"""
    student = get_student_by_name(name)
    if not student:
        print(f"未找到学生: {name}")
        return False
    
    student["scores"][subject] = score
    print(f"成功更新 {name} 的 {subject} 成绩为 {score}")
    return True

def get_top_students(n: int = 3) -> list:
    """获取平均分最高的前 n 名学生"""
    student_avg = []
    for student in students:
        avg = calculate_average(student["scores"])
        student_avg.append((student["name"], avg))
    
    student_avg.sort(key=lambda x: x[1], reverse=True)
    return student_avg[:n]

def display_all_students():
    """显示所有学生信息"""
    if not students:
        print("暂无学生信息")
        return
    
    print("\n" + "=" * 50)
    print(f"{'姓名':<10}{'平均分':<10}{'成绩详情'}")
    print("=" * 50)
    
    for student in students:
        avg = calculate_average(student["scores"])
        scores_str = ", ".join([f"{k}:{v}" for k, v in student["scores"].items()])
        print(f"{student['name']:<10}{avg:<10.1f}{scores_str}")
    print("=" * 50)

# 主程序
if __name__ == "__main__":
    # 添加学生
    add_student("张三", {"数学": 85, "英语": 90})
    add_student("李四", {"数学": 92, "英语": 88})
    add_student("王五", {"数学": 78, "英语": 85})
    
    # 更新成绩
    update_score("张三", "数学", 95)
    
    # 显示所有学生
    display_all_students()
    
    # 获取优秀学生
    print("\n=== 成绩优异学生 TOP 2 ===")
    top = get_top_students(2)
    for rank, (name, avg) in enumerate(top, 1):
        print(f"{rank}. {name}: {avg:.1f}分")
```

### 十三、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 函数定义后自动执行 | ❌ 需要显式调用 |
| 形参和实参混淆 | 形参定义时用，实参调用时用 |
| 默认参数使用可变对象 | ❌ 会共享，使用 `None` 替代 |
| 函数内修改全局变量 | 需要使用 `global` 声明 |
| `*args` 后参数可位置传参 | ❌ 必须关键字传参 |
| 递归没有终止条件 | 会导致无限递归，栈溢出 |
| lambda 中修改外部变量 | ❌ lambda 只能包含表达式 |
| 函数注解强制类型检查 | ❌ 注解只是提示，不强制 |


```python
# 默认参数陷阱
def add_item(item, my_list=[]):  # 危险！
    my_list.append(item)
    return my_list

print(add_item(1))  # [1]
print(add_item(2))  # [1, 2]（共享同一个列表！）

# 正确做法
def add_item_safe(item, my_list=None):
    if my_list is None:
        my_list = []
    my_list.append(item)
    return my_list
```

> 📌 **核心记忆**：
>
> + 函数用 `def` 定义，**先定义后调用**
> + 形参是占位符，实参是实际值
> + Python 参数传递取决于对象**可变性**
> + 可变类型（列表、字典）修改影响外部
> + 不可变类型（数字、字符串、元组）修改不影响外部
> + 使用 `global` 修改全局变量，`nonlocal` 修改嵌套变量
> + 递归需要**终止条件**，避免栈溢出
> + lambda 是**单表达式**的匿名函数
> + 默认参数避免使用**可变对象**
