### 一、面向过程 vs 面向对象
#### 1.1 两种编程范式
| 范式 | 核心思想 | 关注点 | 适合场景 |
| --- | --- | --- | --- |
| **面向过程编程** | 以过程（函数）为中心 | 解决问题的步骤和流程 | 简单、线性任务 |
| **面向对象编程** | 以对象为中心 | 系统中的对象及对象间的交互 | 复杂、大型系统 |


**Python 是混合型语言**：既支持面向过程，也支持面向对象。

#### 1.2 面向过程举例：做晚餐
```python
def buy():
    print("去超市购买食材。")

def wash():
    print("清洗蔬菜。")

def cut():
    print("切菜。")

def cook():
    print("开始烹饪。")

def serve():
    print("上菜啦！")

# 按顺序调用
buy()
wash()
cut()
cook()
serve()
```

**缺点**：步骤固定，缺乏通用性；代码会越来越臃肿。

#### 1.3 面向对象举例：做晚餐
```python
class Dish:
    """菜品类"""
    def __init__(self, name):
        self.name = name
    
    def prepare(self):
        pass

class Salad(Dish):
    """沙拉类"""
    def prepare(self):
        print(f"为 {self.name} 购买食材。")
        print(f"清洗 {self.name} 的蔬菜。")
        print(f"切 {self.name} 的蔬菜。")

class Stew(Dish):
    """炖菜类"""
    def prepare(self):
        print(f"为 {self.name} 购买食材。")
        print(f"切 {self.name} 的肉。")
        print(f"烹饪 {self.name}。")

# 创建菜品对象并准备
salad = Salad("蔬菜沙拉")
stew = Stew("炖肉")

salad.prepare()
stew.prepare()
```

**优势**：

+ **封装**：将数据（菜名）和操作（制作过程）组织在一起
+ **可扩展**：添加新菜品只需创建新子类，不影响现有代码

#### 1.4 面向对象历史
+ 1960年代，**Simula 67** 语言首次引入"类"和"对象"的概念
+ 由奥利-约翰·达尔和克利斯登·奈加特为模拟环境设计
+ 目的是将真实世界的对象映射到程序中的抽象对象

### 二、类与对象的基本概念
#### 2.1 类（Class）
| 定义 | 说明 |
| --- | --- |
| **是什么** | 对大量对象共性的抽象 |
| **作用** | 创建对象的模板 |
| **本质** | 客观事物在人脑中的主观反映 |


**类的成员**：

+ **属性**：对象"有什么"（特征、状态）
+ **方法**：对象"能做什么"（行为）

#### 2.2 对象（Object）
| 定义 | 说明 |
| --- | --- |
| **是什么** | 客观存在的事物 |
| **与类的关系** | 对象是类的**实例**（Instance），类是抽象的，对象是具体的 |
| **特征** | 每个对象有自己的状态（属性）、行为（方法）和唯一的标识（内存地址） |


```python
# 类：图纸
class House:
    pass

# 对象：根据图纸建造的房子
house1 = House()  # 实例1
house2 = House()  # 实例2
# house1 和 house2 是不同的对象，有不同的内存地址
```

### 三、定义类
#### 3.1 语法
```python
class 类名:
    """类说明文档"""
    类体
```

**命名规范**：类名使用**大驼峰命名法**（如 `Person`、`StudentInfo`）

**类体可以包含**：

+ 类属性（类变量）
+ 实例属性（实例变量）
+ 方法（实例方法、类方法、静态方法）

#### 3.2 案例
```python
class Person:
    """人的类"""
    
    # 类属性（所有实例共享）
    home = "earth"
    
    # 构造方法（初始化实例属性）
    def __init__(self):
        self.age = 0  # 实例属性
    
    # 实例方法
    def eat(self):
        print("eating...")
    
    def drink(self):
        print("drinking...")
```

### 四、类的操作
#### 4.1 成员引用（通过类名访问）
```python
class Person:
    home = "earth"
    
    def __init__(self):
        self.age = 0
    
    def eat(self):
        print("eating...")

# 访问类属性
print(Person.home)           # earth

# 获取方法对象（未绑定）
eat_func = Person.eat
print(eat_func)              # <function Person.eat at 0x...>

# 获取文档字符串
doc = Person.__doc__
print(doc)                   # 人的类
```

#### 4.2 实例化（创建对象）
```python
# 创建实例
p = Person()

# 通过实例访问
print(p.home)    # earth（继承自类）
print(p.age)     # 0
p.eat()          # eating...
p.drink()        # drinking...
```

### 五、`__init__()` 构造方法
`__init__()` 是一个特殊方法，在创建对象时自动调用，用于**初始化实例属性**。

| 特点 | 说明 |
| --- | --- |
| **调用时机** | 创建对象时自动调用 |
| **主要作用** | 初始化实例属性 |
| **返回值** | 只能返回 `None`，不能返回其他值 |
| **是否必需** | 否，若未定义则使用默认空构造 |


```python
class Person:
    def __init__(self, name, age):
        self.name = name   # 初始化实例属性
        self.age = age

# 创建对象时传入参数
p = Person("张三", 18)
print(p.name)  # 张三
print(p.age)   # 18
```

### 六、`self` 参数
#### 6.1 `self` 是什么
`self` 代表类的**实例对象本身**，是实例方法的第一个参数。

```python
class Person:
    def eat(self):
        print("eating...")

p = Person()
p.eat()              # 等价于 Person.eat(p)
```

#### 6.2 通过 `self` 访问成员
在实例方法内部，通过 `self` 可以访问：

+ 实例属性
+ 类属性（也可通过类名访问）
+ 其他实例方法

```python
class Person:
    home = "earth"
    
    def __init__(self, name):
        self.name = name
    
    def eat(self):
        print(f"{self.name} is eating...")
    
    def drink(self):
        print(f"{self.name} is drinking...")
    
    def eat_and_drink(self):
        print(f"当前人: {self.name}")   # 访问实例属性
        self.eat()                       # 调用实例方法
        self.drink()                     # 调用实例方法

p = Person("张三")
p.eat_and_drink()
```

### 七、属性详解
#### 7.1 类属性（类变量）
**定义位置**：在类中、方法外定义。

**特点**：

+ 所有实例**共享**同一个类属性
+ 可通过 `类名.属性名` 或 `实例名.属性名` 访问
+ 通过 `类名.属性名` 修改会影响所有实例
+ 通过 `实例名.属性名` 赋值会创建**实例属性**（不推荐同名）

```python
class Person:
    home = "earth"  # 类属性

# 访问
print(Person.home)  # earth

p1 = Person()
p2 = Person()
print(p1.home)      # earth
print(p2.home)      # earth

# 通过类名修改（影响所有实例）
Person.home = "mars"
print(p1.home)      # mars
print(p2.home)      # mars

# 通过实例名修改（创建实例属性，不推荐）
p1.home = "venus"   # 创建实例属性，不影响类
print(Person.home)  # mars（类属性不变）
print(p1.home)      # venus（实例属性）
print(p2.home)      # mars（仍使用类属性）
```

**推荐**：使用 `类名.属性名` 访问和修改类属性，避免与实例属性同名。

#### 7.2 实例属性（实例变量）
**定义位置**：通常在 `__init__()` 方法中通过 `self.属性名` 定义。

**特点**：

+ 每个实例**独有一份**，互不影响
+ 只能通过实例访问，不能通过类名访问
+ 可以在实例上动态添加

```python
class Person:
    def __init__(self, name, age):
        self.name = name   # 实例属性
        self.age = age     # 实例属性

p1 = Person("张三", 18)
p2 = Person("李四", 20)

print(p1.name, p1.age)    # 张三 18
print(p2.name, p2.age)    # 李四 20

# 每个实例的属性独立
p1.age = 25
print(p1.age)             # 25
print(p2.age)             # 20

# 通过类名无法访问实例属性
# print(Person.name)      # AttributeError
```

### 八、方法详解
Python 类中有三种方法：**实例方法**、**类方法**、**静态方法**。

| 方法类型 | 定义装饰器 | 第一个参数 | 调用方式 | 可访问内容 |
| --- | --- | --- | --- | --- |
| **实例方法** | 无 | `self`（实例） | 实例调用 | 实例属性、类属性、其他方法 |
| **类方法** | `@classmethod` | `cls`（类） | 类或实例调用 | 类属性、其他类方法 |
| **静态方法** | `@staticmethod` | 无 | 类或实例调用 | 仅依赖传入参数 |


#### 8.1 实例方法
```python
class Person:
    home = "earth"
    
    def __init__(self, name):
        self.name = name
    
    def instance_method(self):
        """实例方法：第一个参数为 self"""
        print(f"实例方法被调用: {self.name}")
        print(f"访问类属性: {self.home}")  # 或 Person.home

p = Person("张三")
p.instance_method()  # 通过实例调用
# Person.instance_method(p)  # 等价形式
```

#### 8.2 类方法
```python
class Person:
    home = "earth"
    
    @classmethod
    def class_method(cls):
        """类方法：第一个参数为 cls（类本身）"""
        print(f"类方法被调用")
        print(f"访问类属性: {cls.home}")

# 通过类调用（推荐）
Person.class_method()

# 通过实例调用（也可以）
p = Person()
p.class_method()
```

**使用场景**：工厂方法、操作类属性、创建类的实例等。

#### 8.3 静态方法
```python
class Person:
    @staticmethod
    def static_method(x, y):
        """静态方法：无特殊参数"""
        print(f"静态方法被调用: {x + y}")
        # 不能访问实例属性或类属性（除非通过类名）

# 通过类调用（推荐）
Person.static_method(3, 5)

# 通过实例调用（也可以）
p = Person()
p.static_method(3, 5)
```

**使用场景**：工具函数、与类逻辑相关但不依赖实例或类的功能。

#### 8.4 三种方法对比示例
```python
class Example:
    class_var = "类变量"
    
    def __init__(self, name):
        self.name = name
    
    def instance_method(self):
        """实例方法：可以访问实例属性和类属性"""
        print(f"实例方法: name={self.name}, class_var={self.class_var}")
    
    @classmethod
    def class_method(cls):
        """类方法：只能访问类属性"""
        print(f"类方法: class_var={cls.class_var}")
    
    @staticmethod
    def static_method():
        """静态方法：不能访问实例属性或类属性"""
        print("静态方法: 只能使用传入的参数")

# 调用
obj = Example("test")
obj.instance_method()   # 实例方法: name=test, class_var=类变量
Example.class_method()  # 类方法: class_var=类变量
Example.static_method() # 静态方法: 只能使用传入的参数
```

#### 8.5 在类外定义方法
可以将外部函数赋值给类的成员：

```python
# 在类外定义的函数
def f1(self, x, y):
    print(x & y)

class C:
    f = f1  # 将函数赋值给类成员

C().f(6, 13)  # 4
```

#### 8.6 特殊方法（魔法方法）
| 方法 | 说明 | 触发时机 |
| --- | --- | --- |
| `__new__(cls, ...)` | 创建实例 | 实例化时第一个被调用 |
| `__init__(self, ...)` | 初始化实例 | `__new__` 之后调用 |
| `__del__(self)` | 销毁对象 | 对象被垃圾回收时 |
| `__str__(self)` | 字符串表示 | `str(obj)` 或 `print(obj)` |
| `__repr__(self)` | 开发态字符串表示 | `repr(obj)`，通常可用于重建对象 |
| `__getattribute__(self, name)` | 属性访问拦截 | 访问任何属性时 |


```python
class Person:
    def __init__(self, name):
        self.name = name
    
    def __str__(self):
        """人类可读的输出"""
        return f"Person: {self.name}"
    
    def __repr__(self):
        """机器可读的输出，通常可用于重建对象"""
        return f"Person('{self.name}')"

p = Person("张三")
print(p)           # Person: 张三（调用 __str__）
print(repr(p))     # Person('张三')（调用 __repr__）
```

### 九、动态添加属性与方法
Python 支持动态地为对象或类添加属性和方法。

#### 9.1 动态给对象添加属性
```python
class Person:
    def __init__(self, name):
        self.name = name

p = Person("张三")
print(p.name)      # 张三

# 动态添加属性
p.age = 18
p.city = "北京"
print(p.age)       # 18
print(p.city)      # 北京
```

#### 9.2 动态给类添加属性
```python
class Person:
    pass

# 动态添加类属性
Person.home = "earth"
print(Person.home)  # earth

# 类属性对所有实例生效
p = Person()
print(p.home)       # earth
```

#### 9.3 动态给实例添加方法
**添加普通函数**（无 `self` 参数）：

```python
class Person:
    def __init__(self, name):
        self.name = name

def eat():
    print("吃饭")

p = Person("张三")
p.eat = eat
p.eat()  # 吃饭
```

**添加实例方法**（需要 `self`，使用 `MethodType`）：

```python
import types

class Person:
    def __init__(self, name):
        self.name = name

def eat(self):
    print(f"{self.name}在吃饭")

p = Person("张三")
# 将函数绑定到实例
p.eat = types.MethodType(eat, p)
p.eat()  # 张三在吃饭

# 其他实例不受影响
p2 = Person("李四")
# p2.eat()  # 会报错，因为 p2 没有 eat 方法
```

#### 9.4 动态给类添加方法
```python
class Person:
    home = "earth"

# 添加类方法
@classmethod
def come_from(cls):
    print(f"来自{cls.home}")

Person.come_from = come_from
Person.come_from()  # 来自earth

# 添加静态方法
@staticmethod
def utility():
    print("工具函数")

Person.utility = utility
Person.utility()  # 工具函数
```

#### 9.5 动态删除属性与方法
```python
class Person:
    def __init__(self, name):
        self.name = name
        self.age = 18

p = Person("张三")

# 方式1：del 语句
del p.age
# print(p.age)  # AttributeError

# 方式2：delattr 函数
delattr(p, "name")
# print(p.name)  # AttributeError
```

#### 9.6 `__slots__`：限制实例属性
`__slots__` 可以限制实例能动态添加的属性和实例方法。

```python
import types

class Person:
    __slots__ = ("name", "age", "eat")  # 只允许这些实例属性/方法
    
    def __init__(self, name):
        self.name = name

p = Person("张三")

# 允许添加
p.age = 18
print(p.age)  # 18

# 允许添加实例方法（如果在 __slots__ 中）
def eat(self):
    print(f"{self.name}在吃饭")

p.eat = types.MethodType(eat, p)
p.eat()  # 张三在吃饭

# 禁止添加不在 __slots__ 中的属性
# p.weight = 100  # AttributeError: 'Person' object has no attribute 'weight'
```

**注意事项**：

+ `__slots__` 仅对当前类生效，对其**子类不生效**
+ 可以继续添加类属性、类方法和静态方法
+ 使用 `__slots__` 可以节省内存（因为每个实例不需要 `__dict__`）

### 十、综合示例：学生管理系统
```python
class Student:
    """学生类"""
    
    # 类属性：学校名称（所有学生共享）
    school = "第一中学"
    
    def __init__(self, name, student_id, score=0):
        """初始化实例属性"""
        self.name = name
        self.id = student_id
        self.score = score
    
    # 实例方法
    def study(self, hours):
        print(f"{self.name} 学习了 {hours} 小时")
        self.score += hours * 5
        print(f"成绩提升到: {self.score}")
    
    def display_info(self):
        """显示学生信息"""
        print(f"姓名: {self.name}")
        print(f"学号: {self.id}")
        print(f"成绩: {self.score}")
        print(f"学校: {self.school}")
    
    # 类方法
    @classmethod
    def change_school(cls, new_school):
        """修改学校名称"""
        cls.school = new_school
        print(f"学校已更名为: {cls.school}")
    
    # 静态方法
    @staticmethod
    def is_pass(score):
        """判断是否及格"""
        return score >= 60


# 使用示例
# 创建学生
stu1 = Student("张三", "2024001", 60)
stu2 = Student("李四", "2024002", 55)

# 显示信息
print("=== 初始信息 ===")
stu1.display_info()
stu2.display_info()

# 学习
print("\n=== 学习后 ===")
stu1.study(2)
stu2.study(1)

# 使用类方法
Student.change_school("第一高级中学")

# 使用静态方法
print(f"\n张三是否及格: {Student.is_pass(stu1.score)}")
print(f"李四是否及格: {Student.is_pass(stu2.score)}")

# 动态添加属性
stu1.hobby = "篮球"
print(f"\n张三的爱好: {stu1.hobby}")
```

### 十一、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 类名使用小写 | 类名使用**大驼峰命名法** |
| 实例方法忘记写 `self` | 实例方法第一个参数必须是 `self` |
| 类属性通过实例修改 | 通过实例修改类属性会创建实例属性，不修改类属性 |
| `__init__` 返回非 `None` | `__init__` 只能返回 `None` |
| 混淆类方法和静态方法 | 类方法有 `@classmethod` 和 `cls` 参数，可访问类属性；静态方法无特殊参数 |
| 忽略 `__slots__` 对子类的影响 | `__slots__` 仅对当前类生效，对子类无效 |


### 十二、快速参考卡片
```python
# 定义类
class ClassName:
    class_var = "类属性"           # 类属性
    
    def __init__(self, param):     # 构造方法
        self.instance_var = param  # 实例属性
    
    def instance_method(self):     # 实例方法
        print(self.instance_var)
    
    @classmethod
    def class_method(cls):         # 类方法
        print(cls.class_var)
    
    @staticmethod
    def static_method():           # 静态方法
        print("静态方法")

# 创建对象
obj = ClassName("值")

# 访问成员
obj.instance_method()           # 调用实例方法
ClassName.class_method()        # 调用类方法
ClassName.static_method()       # 调用静态方法
print(ClassName.class_var)      # 访问类属性
print(obj.instance_var)         # 访问实例属性

# 动态操作
obj.new_attr = "新属性"          # 动态添加属性
del obj.new_attr                # 删除属性

# 限制实例属性
class Restricted:
    __slots__ = ("allowed1", "allowed2")  # 只允许这些属性
```

> 📌 **核心记忆**：
>
> + **类**是模板（图纸），**对象**是实例（房子）
> + 类名用**大驼峰**，实例方法第一个参数是 `self`
> + `__init__` 是构造方法，用于初始化实例属性
> + **类属性**所有实例共享，**实例属性**每个实例独有
> + 类方法用 `@classmethod`（参数 `cls`），静态方法用 `@staticmethod`（无特殊参数）
> + Python 支持**动态添加**属性和方法
> + `__slots__` 可以限制实例属性，节省内存
>
