### 一、核心概念：变量无类型，对象有类型
> **关键理解**：Python 变量本身没有类型，它只是指向内存中某个对象的"标签"。我们所说的"类型"是指变量所指向的对象 的类型。
>

```python
a = 10      # a 指向 int 对象
a = "hello" # a 现在指向 str 对象（变量可以改变类型）
```

### 二、Python 数据类型分类
| 类别 | 类型 | 是否可变 |
| --- | --- | --- |
| **数值** | int（整型）、float（浮点型）、complex（复数）、bool（布尔） | ❌ 不可变 |
| **字符串** | str | ❌ 不可变 |
| **容器类型** | list（列表）、tuple（元组）、set（集合）、dict（字典） | list/set/dict 可变，tuple 不可变 |
| **特殊类型** | None | ❌ 不可变 |


**记忆卡片**：

```plain
不可变（3个）：Number、String、Tuple
可变（3个）：List、Dictionary、Set
```

### 三、数值类型详解
#### 1. 整型（int）
**特点**：Python 可以处理任意大小的整数（无上限，仅受内存限制）

##### ① 整数分隔符（Python 3.6+）
```python
num1 = 1_000_000_000_000_000
print(num1)   # 1000000000000000
```

| 要点 | 说明 |
| --- | --- |
| 作用 | 提高大数字可读性 |
| 存储 | Python 自动忽略下划线 |
| 适用范围 | 整数和浮点数 |
| 版本要求 | Python 3.6 及以上 |


##### ② 类型判断：`type()` vs `isinstance()`
```python
num1 = True
num2 = 10

# type() - 精确类型
print(type(num1))        # <class 'bool'>
print(type(num2))        # <class 'int'>
print(type(num1) == type(num2))   # False

# isinstance() - 考虑继承关系（bool 是 int 的子类）
print(isinstance(num1, bool))     # True
print(isinstance(num1, int))      # True
```

| 函数 | 特点 | 推荐场景 |
| --- | --- | --- |
| `type()` | 精确类型，不考虑继承 | 严格区分子类和父类时 |
| `isinstance()` | 考虑继承关系 | **推荐**，更符合多态原则 |


##### ③ 小整数池 vs 大整数池
```plain
小整数池：[-5, 256] 范围内的整数
├── 程序启动时预先创建
├── 全局共享同一个对象
└── 不会被垃圾回收

大整数池：超出 [-5, 256] 的范围
├── 初始为空
├── 创建时存入池中
└── 不同实现细节可能不同
```

```python
a = 100
b = 100
print(a is b)   # True（小整数池复用）

c = 1000
d = 1000
print(c is d)   # 交互式环境可能是 False，脚本中可能为 True（优化机制）
```

> ⚠️ **注意事项**：不同 Python 实现（CPython、Jython 等）的小整数池范围可能不同。CPython 默认为 [-5, 256]。
>

#### 2. 浮点型（float）
```python
num1 = 0.1
num2 = 0.2
print(num1 + num2)   # 0.30000000000000004 （精度误差！）
```

**原因图解**：

```plain
0.1（十进制）→ 二进制无限循环小数 → 近似存储 → 累积误差
```

**解决方案**：

```python
from decimal import Decimal

num3 = Decimal('1.0')
num4 = Decimal('0.9')
print(num3 - num4)   # 0.1（精确计算）
```

**科学计数法**：

```python
num1 = 1.3e7    # 13000000.0（正指数，向右移）
num2 = 1.3e-7   # 0.00000013（负指数，向左移）
```

#### 3. 布尔型（bool）
```python
bool1 = True
bool2 = False
```

**关键特性**：

| 特性 | 说明 |
| --- | --- |
| 子类关系 | `bool` 是 `int` 的子类 |
| 数值等价 | `True == 1`、`False == 0` |
| 对象身份 | `True is not 1`、`False is not 0` |
| 可与数字运算 | `True + 5 = 6`、`False + 5 = 5` |


`==`** vs **`is`** 对比**：

```python
print(True == 1)    # True（值相等）
print(True is 1)    # False（不是同一个对象）

print(False == 0)   # True
print(False is 0)   # False
```

**会被判定为 **`False`** 的值**：

```python
# 假值列表
None
False
0
0.0
""          # 空字符串
[]          # 空列表
()          # 空元组
{}          # 空字典
set()       # 空集合
```

> 💡 其他所有值都被视为 `True`
>

### 四、字符串（str）
#### 创建方式
```python
# 单引号/双引号
str1 = 'This is a "string"'
str2 = "This is a 'string' too"

# 三引号（多行字符串）
str3 = """hello world
HELLO WORLD"""
```

#### 转义字符表
| 转义符 | 说明 | 转义符 | 说明 |
| --- | --- | --- | --- |
| `\n` | 换行 | `\\` | 反斜杠 |
| `\t` | 制表符 | `\'` | 单引号 |
| `\r` | 回车 | `\"` | 双引号 |
| `\b` | 退格 | `\`（行尾） | 续行符 |


#### intern 机制（字符串驻留）
```plain
创建字符串时：
    ↓
检查内存中是否已存在相同内容的字符串
    ↓
存在 → 直接返回已有对象的引用（共享）
不存在 → 创建新对象
```

**触发条件**：不包含空格或特殊符号的字符串

```python
a = "hello"
b = "hello"
print(a is b)   # True（intern 机制）

c = "hello world"
d = "hello world"
print(c is d)   # 可能为 False（包含空格）
```

**字符串缓冲池**：长度为 1 的 ASCII 字符会被自动 interned

### 五、类型转换
#### 1. 自动类型转换（隐式转换）
**规则**：小范围 → 大范围（避免数据丢失）

```python
# 整型 + 浮点型 = 浮点型
num1 = 2
num2 = 3.0
print(num1 + num2)   # 5.0

# 整型除法 → 浮点型
print(9 / 1)         # 9.0

# 整型 + 字符串 → 报错（无法自动转换）
print(123 + "456")   # TypeError!
```

#### 2. 强制类型转换（显式转换）
| 函数 | 说明 | 示例 |
| --- | --- | --- |
| `int(x)` | 转整数 | `int("123")` → 123 |
| `float(x)` | 转浮点数 | `float("3.14")` → 3.14 |
| `str(x)` | 转字符串 | `str(123)` → "123" |
| `bool(x)` | 转布尔 | `bool(1)` → True |
| `list(s)` | 转列表 | `list("abc")` → ['a','b','c'] |
| `tuple(s)` | 转元组 | `tuple([1,2])` → (1,2) |
| `set(s)` | 转集合 | `set([1,2,2])` → {1,2} |
| `ord(c)` | 字符 → ASCII | `ord('A')` → 65 |
| `chr(n)` | ASCII → 字符 | `chr(65)` → 'A' |
| `bin(x)` | 转二进制字符串 | `bin(10)` → '0b1010' |
| `hex(x)` | 转十六进制字符串 | `hex(255)` → '0xff' |


**完整示例**：

```python
num_int = 123
num_str = "456"

print(type(num_str))        # <class 'str'>
num_str = int(num_str)      # 强制转换
print(type(num_str))        # <class 'int'>

num_sum = num_int + num_str
print(num_sum)              # 579
print(type(num_sum))        # <class 'int'>
```

### 六、None 特殊类型
| 属性 | 说明 |
| --- | --- |
| 含义 | 空值或缺失值 |
| 唯一性 | 只有一个 None 对象 |
| 典型场景 | 函数无返回值时、变量未赋值时 |
| 类型 | `NoneType` |


```python
def func():
    pass

result = func()
print(result)   # None
print(result is None)   # True
```

### 七、易错点速查表
| 易错点 | 正确理解 |
| --- | --- |
| 变量有类型 | ❌ 变量无类型，对象有类型 |
| `0.1 + 0.2 == 0.3` | ❌ 浮点精度问题，结果约为 0.30000000000000004 |
| `type()` 和 `isinstance()` 没区别 | ❌ `isinstance()` 考虑继承关系 |
| `True is 1` 为 True | ❌ `is` 比较身份，`==` 比较值 |
| 空容器 = None | ❌ 空列表是 `[]`，None 是单独的类型 |
| 所有字符串都 intern | ❌ 只有不包含空格的字符串默认 intern |
| 类型转换自动进行 | ❌ 某些转换（如 str + int）必须显式进行 |

### 九、速查卡片
```python
# 类型获取
type(obj)           # 获取精确类型
isinstance(obj, type)   # 判断是否为某类型（含继承）

# 类型转换
int()   float()   str()   bool()
list()  tuple()   set()   dict()

# 数值相关
1_000_000      # 分隔符
Decimal('0.1') # 精确小数

# 字符串
s.encode('utf-8')   # str → bytes
b.decode('utf-8')   # bytes → str

# None
x is None    # 推荐判断方式
x == None    # 不推荐
```

> 📌 **核心记忆**：变量是标签，对象有类型；类型分可变与不可变；浮点计算用 Decimal；空值判断用 is None。
>
