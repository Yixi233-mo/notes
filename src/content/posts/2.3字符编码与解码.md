### 一、基本概念
| 术语 | 说明 | 方向 |
| --- | --- | --- |
| **编码 (encode)** | 字符串 → 字节数据 | str → bytes |
| **解码 (decode)** | 字节数据 → 字符串 | bytes → str |


```python
str1 = '你好中国'        # 字符串类型 (str)
byte1 = str1.encode()    # 编码：str → bytes
str2 = byte1.decode()    # 解码：bytes → str
```

### 二、代码解析
```python
# 1. 创建字符串
str1 = '你好中国'
print(str1, type(str1))   # 你好中国 <class 'str'>

# 2. 编码：字符串 → 字节
byte1 = str1.encode('utf-8')
print(byte1, type(byte1))  
# b'\xe4\xbd\xa0\xe5\xa5\xbd\xe4\xb8\xad\xe5\x9b\xbd' <class 'bytes'>

# 3. 解码：字节 → 字符串
str2 = byte1.decode(encoding='utf8')
print(str2)   # 你好中国
```

### 三、UTF-8 编码规则（重点）
#### 字节数计算
```python
byte1 = str1.encode('utf-8')
print(len(byte1))   # 12个字节
```

| 字符 | UTF-8 字节数 | 说明 |
| --- | --- | --- |
| 英文字母/数字/常见符号 | 1 字节 | ASCII 兼容 |
| 希腊字母、西里尔字母等 | 2 字节 | 扩展拉丁文 |
| 中文、日文、韩文（CJK） | 3 字节 | 常用汉字 |
| 生僻字、Emoji | 4 字节 | 辅助平面字符 |


**示例分析**：

+ `'你好中国'` 共 4 个中文字符
+ 每个中文 → 3 字节（UTF-8）
+ 总计：4 × 3 = **12 字节**

### 四、常用编码格式对比
| 编码 | 特点 | 中文占用 | 适用场景 |
| --- | --- | --- | --- |
| **UTF-8** | 变长编码，兼容 ASCII | 3 字节 | **国际通用，首选** |
| **GBK/GB2312** | 中文字符集，定长 | 2 字节 | 纯中文环境 |
| **ASCII** | 仅英文、数字、符号 | 不支持 | 纯英文 |
| **UTF-16** | 变长编码 | 2 或 4 字节 | Windows 内部 |
| **ISO-8859-1** | 西欧语言 | 不支持 | 欧洲语言 |


```python
# 不同编码对比示例
text = '你好'

print(len(text.encode('utf-8')))   # 6 字节 (3×2)
print(len(text.encode('gbk')))     # 4 字节 (2×2)
```

### 五、编码/解码常见错误与处理
#### 1. 编码错误（UnicodeEncodeError）
```python
# 字符串包含当前编码无法表示的字符
text = '你好'
text.encode('ascii')   # ❌ UnicodeEncodeError: 'ascii' codec can't encode...
```

**解决方法**：

```python
text.encode('ascii', errors='ignore')   # 忽略无法编码的字符
text.encode('ascii', errors='replace')  # 用 ? 替换
text.encode('ascii', errors='xmlcharrefreplace')  # 转为 XML 实体
```

#### 2. 解码错误（UnicodeDecodeError）
```python
# 用错误的编码解码字节数据
byte1 = '你好'.encode('utf-8')
byte1.decode('gbk')    # ❌ 乱码或报错
```

### 六、bytes 类型详解
```python
byte1 = '你好'.encode('utf-8')
# b'\xe4\xbd\xa0\xe5\xa5\xbd'
```

**特点**：

+ 以 `b''` 表示
+ 元素是 0-255 之间的整数
+ 不可变类型

```python
print(byte1[0])        # 228 (0xe4)
print(byte1[0:2])      # b'\xe4\xbd'
print(list(byte1))     # [228, 189, 160, 229, 165, 189]
```

### 七、实际应用场景
| 场景 | 说明 |
| --- | --- |
| **网络传输** | socket 发送数据需先 encode 为 bytes |
| **文件读写** | 二进制模式（图片、音视频）用 bytes |
| **加密/解密** | 加密函数通常输入 bytes |
| **JSON 交互** | API 传输需指定编码 |
| **数据库存储** | 确认数据库编码设置 |


```python
# 文件写入示例
with open('test.txt', 'wb') as f:    # 二进制写入
    f.write('你好'.encode('utf-8'))

with open('test.txt', 'rb') as f:    # 二进制读取
    data = f.read()
    print(data.decode('utf-8'))      # 你好
```

### 八、易错点总结
| 易错点 | 正确做法 |
| --- | --- |
| 编码和解码使用不同编码 | 必须使用**相同编码** |
| 忘记指定 encoding 参数 | 建议显式指定：`.encode('utf-8')` |
| 混淆 str 和 bytes | `type()` 检查类型 |
| 网络传输直接发 str | 需先 `.encode()` 为 bytes |
| 打印 bytes 直接看内容 | bytes 以十六进制显示，需 decode 才能看原文 |

>
> + **编码**：人读的 → 机器读的（str → bytes）  
> + **解码**：机器读的 → 人读的（bytes → str）  
> + **UTF-8** 是首选，中文占 **3 字节**
