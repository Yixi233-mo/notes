### 一、作用域（Scope）
**作用域**是Python代码中的一段文本区域，从这个区域可以直接访问该命名空间。

#### 1.1 四种作用域（LEGB规则）
| 层级 | 名称 | 说明 | 搜索顺序 |
| --- | --- | --- | --- |
| **L** | Local（局部） | 函数内部定义的名称 | 第1位 |
| **E** | Enclosing（嵌套） | 外层闭包函数中的名称 | 第2位 |
| **G** | Global（全局） | 当前模块的全局名称 | 第3位 |
| **B** | Built-in（内置） | Python内置名称 | 第4位 |


```plain
LEGB 查找规则：
L → E → G → B
（从最内层逐层向外搜索）
```

#### 1.2 作用域示例
```python
x = "global"        # G: 全局作用域

def outer():
    x = "enclosing"   # E: 嵌套作用域
    
    def inner():
        x = "local"   # L: 局部作用域
        print(x)      # 查找顺序: local → enclosing → global → built-in
    
    inner()

outer()   # 输出: local
```

#### 1.3 `global` 和 `nonlocal` 关键字
```python
# global：在局部作用域中修改全局变量
count = 0

def increment():
    global count
    count += 1

increment()
print(count)   # 1

# nonlocal：在内部作用域中修改嵌套作用域的变量
def outer():
    x = 10
    
    def inner():
        nonlocal x
        x += 5
    
    inner()
    print(x)   # 15

outer()
```

> ⚠️ **注意**：在不使用 `global` 或 `nonlocal` 的情况下，在局部作用域中对外部变量进行赋值，会在局部作用域中创建一个新的局部变量，外部变量保持不变。
>

```python
x = 10

def func():
    x = 20   # 创建局部变量 x，不影响全局 x
    print(f"函数内: {x}")

func()       # 函数内: 20
print(x)     # 10（全局变量未被修改）
```

### 二、闭包（Closure）
#### 2.1 什么是闭包
**闭包**是指一个函数对象，它保留了对其定义时所在作用域中变量的引用，即使该作用域已经执行完毕。

| 特点 | 说明 |
| --- | --- |
| **数据持久化** | 函数内的数据在函数调用后仍然存在 |
| **数据隐藏** | 避免使用全局变量，隐藏数据 |
| **工厂函数** | 可以创建定制化的函数 |


#### 2.2 构建闭包的条件
1. **外部函数**内定义一个**内部函数**
2. **内部函数**使用了**外部函数的变量**
3. **外部函数**将**内部函数**作为**返回值**返回

#### 2.3 闭包示例
```python
def linear(a, b):
    """外部函数：创建一次线性函数"""
    def inner(x):
        """内部函数：计算 a*x + b"""
        return a * x + b
    return inner

# 创建不同的线性函数
y1 = linear(1, 1)   # y = x + 1
y2 = linear(2, 3)   # y = 2x + 3

print(y1(5))   # 6（1*5 + 1）
print(y2(5))   # 13（2*5 + 3）
```

**执行流程**：

```plain
调用 linear(1, 1) 后：
1. a=1, b=1 保存在内存中
2. 返回 inner 函数
3. 调用 y1(5) 时，inner 仍然记得 a=1, b=1
```

#### 2.4 查看闭包中的值
所有函数对象都有 `__closure__` 属性：

```python
def linear(a, b):
    def inner(x):
        return a * x + b
    return inner

y1 = linear(1, 2)
closure_cells = y1.__closure__

print(closure_cells)                    # 单元格元组
print(closure_cells[0].cell_contents)   # 1（a 的值）
print(closure_cells[1].cell_contents)   # 2（b 的值）
```

### 三、装饰器（Decorator）
#### 3.1 什么是装饰器
**装饰器**允许在不修改原有函数代码的基础上，动态地增加或修改函数的功能。

| 本质 | 说明 |
| --- | --- |
| **输入** | 一个函数 |
| **输出** | 一个新的包装后的函数 |
| **目的** | 在不修改原函数的情况下增强功能 |


#### 3.2 基本装饰器实现
```python
from math import sqrt

# 装饰器函数
def decorator(func):
    def inner(x):
        x = abs(x)      # 添加功能：取绝对值
        return func(x)  # 调用原函数
    return inner

# 手动应用装饰器
def sqrt_func(x):
    return sqrt(x)

sqrt_func = decorator(sqrt_func)
print(sqrt_func(-4))   # 2.0
```

#### 3.3 使用 `@` 语法糖
```python
from math import sqrt

def decorator(func):
    def inner(x):
        x = abs(x)
        return func(x)
    return inner

@decorator
def sqrt_func(x):
    """开根号"""
    return sqrt(x)

print(sqrt_func(-4))   # 2.0
```

> 💡 `@decorator` 等价于 `sqrt_func = decorator(sqrt_func)`
>

### 四、多层装饰器
多个装饰器的执行顺序：**从下往上**（离函数最近的先执行）。

```python
from math import sqrt

# 装饰器1：将参数转换为整数
def get_integer(func):
    def inner(x):
        x = int(x)
        return func(x)
    return inner

# 装饰器2：将参数转换为非负数
def get_absolute(func):
    def inner(x):
        x = abs(x)
        return func(x)
    return inner

@get_integer      # 后执行（外层）
@get_absolute     # 先执行（内层）
def sqrt_func(x):
    return sqrt(x)

print(sqrt_func("-4"))   # 2.0
```

**执行顺序**：

```plain
1. 先应用 @get_absolute：处理绝对值
2. 再应用 @get_integer：处理整数转换
3. 最后执行 sqrt_func
```

### 五、带参数的装饰器
```python
from math import sqrt

# 带参数的装饰器
def times(n):
    """返回一个装饰器，对结果应用 n 次函数"""
    def get_absolute(func):
        def inner(x):
            x = abs(x)
            for _ in range(n):
                x = func(x)
            return x
        return inner
    return get_absolute

@times(2)   # 对结果开两次根号
def sqrt_func(x):
    return sqrt(x)

print(sqrt_func(-16))   # 2.0（sqrt(sqrt(16)) = sqrt(4) = 2）
```

**结构层次**：

```plain
times(n)          # 最外层：接收参数
    ↓ 返回
get_absolute      # 中层：接收函数
    ↓ 返回
inner             # 内层：接收参数并执行
```

### 六、类装饰器
类装饰器通过实现 `__call__()` 方法，使得类的实例可以像函数一样被调用。

```python
from math import sqrt

class DecoratorClass:
    def __init__(self, func):
        self.func = func
    
    def __call__(self, x):
        """当实例被调用时执行"""
        x = abs(x)           # 添加功能
        return self.func(x)  # 调用原函数

@DecoratorClass
def sqrt_func(x):
    return sqrt(x)

print(sqrt_func(-4))   # 2.0
```

**执行流程**：

```plain
@DecoratorClass
↓
sqrt_func = DecoratorClass(sqrt_func)
↓
调用 sqrt_func(-4) 时执行 __call__()
```

### 七、装饰器综合示例
```python
"""
日志装饰器：记录函数调用信息
"""

import time
from functools import wraps

def logger(level="INFO"):
    """带参数的日志装饰器"""
    def decorator(func):
        @wraps(func)   # 保留原函数的元信息
        def wrapper(*args, **kwargs):
            # 记录调用时间
            timestamp = time.strftime("%Y-%m-%d %H:%M:%S")
            
            # 记录参数
            args_str = ", ".join(map(str, args))
            kwargs_str = ", ".join([f"{k}={v}" for k, v in kwargs.items()])
            params = args_str
            if kwargs_str:
                params += f", {kwargs_str}" if params else kwargs_str
            
            # 执行前日志
            print(f"[{timestamp}] [{level}] 调用 {func.__name__}({params})")
            
            # 执行原函数
            start_time = time.time()
            result = func(*args, **kwargs)
            end_time = time.time()
            
            # 执行后日志
            print(f"[{timestamp}] [{level}] {func.__name__} 执行完成，耗时 {end_time - start_time:.4f}s")
            print(f"[{timestamp}] [{level}] 返回值: {result}")
            print("-" * 50)
            
            return result
        return wrapper
    return decorator


# 使用装饰器
@logger(level="DEBUG")
def calculate(a, b, operator="+"):
    """计算器函数"""
    if operator == "+":
        return a + b
    elif operator == "-":
        return a - b
    elif operator == "*":
        return a * b
    elif operator == "/":
        return a / b
    else:
        raise ValueError(f"不支持的运算符: {operator}")

@logger(level="INFO")
def fibonacci(n):
    """斐波那契数列"""
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)


if __name__ == "__main__":
    calculate(10, 5, "*")
    calculate(100, 3, operator="/")
    fibonacci(5)
```

**输出示例**：

```plain
[2024-01-15 10:30:45] [DEBUG] 调用 calculate(10, 5, operator=*)
[2024-01-15 10:30:45] [DEBUG] calculate 执行完成，耗时 0.0001s
[2024-01-15 10:30:45] [DEBUG] 返回值: 50
--------------------------------------------------
[2024-01-15 10:30:45] [DEBUG] 调用 calculate(100, 3, operator=/)
[2024-01-15 10:30:45] [DEBUG] calculate 执行完成，耗时 0.0001s
[2024-01-15 10:30:45] [DEBUG] 返回值: 33.333333333333336
--------------------------------------------------
[2024-01-15 10:30:45] [INFO] 调用 fibonacci(5)
[2024-01-15 10:30:45] [INFO] fibonacci 执行完成，耗时 0.0002s
[2024-01-15 10:30:45] [INFO] 返回值: 5
--------------------------------------------------
```

### 八、`functools.wraps` 的作用
使用 `@wraps` 可以保留原函数的元信息（如 `__name__`、`__doc__`）：

```python
from functools import wraps

def decorator(func):
    @wraps(func)   # 保留原函数信息
    def inner(*args, **kwargs):
        return func(*args, **kwargs)
    return inner

@decorator
def my_function():
    """这是我的函数"""
    pass

print(my_function.__name__)   # my_function（保留原名）
print(my_function.__doc__)    # 这是我的函数（保留文档）
```

如果不使用 `@wraps`：

```python
def bad_decorator(func):
    def inner(*args, **kwargs):
        return func(*args, **kwargs)
    return inner

@bad_decorator
def my_function():
    """这是我的函数"""
    pass

print(my_function.__name__)   # inner（被替换）
print(my_function.__doc__)    # None（文档丢失）
```

### 九、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 装饰器在函数定义时执行 | ✅ 装饰器在 `@` 语法糖处立即执行 |
| 闭包和装饰器是同一概念 | ❌ 闭包是基础，装饰器是闭包的应用 |
| 多层装饰器执行顺序从上到下 | ❌ 从下往上执行（离函数最近的先执行） |
| 装饰器会修改原函数 | ❌ 装饰器不修改原函数，而是返回新的函数 |
| 忘记使用 `functools.wraps` | 会导致函数元信息丢失 |


### 十、快速参考卡片
```python
# === 作用域 ===
# LEGB 规则：Local → Enclosing → Global → Built-in
global x    # 在函数内声明全局变量
nonlocal x  # 在嵌套函数内声明外层变量

# === 闭包 ===
def outer(a, b):
    def inner(x):
        return a * x + b
    return inner

# === 基本装饰器 ===
def decorator(func):
    def wrapper(*args, **kwargs):
        # 执行前
        result = func(*args, **kwargs)
        # 执行后
        return result
    return wrapper

@decorator
def func():
    pass

# === 带参数装饰器 ===
def decorator(param):
    def outer(func):
        def wrapper(*args, **kwargs):
            return func(*args, **kwargs)
        return wrapper
    return outer

@decorator("参数")
def func():
    pass

# === 类装饰器 ===
class Decorator:
    def __init__(self, func):
        self.func = func
    
    def __call__(self, *args):
        return self.func(*args)

# === 保留元信息 ===
from functools import wraps

def decorator(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        return func(*args, **kwargs)
    return wrapper
```

> 📌 **核心记忆**：
>
> + **作用域**：LEGB 规则（Local → Enclosing → Global → Built-in）
> + **闭包**：内部函数引用外部变量，外部函数返回内部函数
> + **装饰器**：不修改原函数，动态增强功能
> + `@`** 语法糖**：`@decorator` 等价于 `func = decorator(func)`
> + **多层装饰器**：从下往上执行
> + `@wraps`：保留原函数的元信息（名称、文档等）
>
