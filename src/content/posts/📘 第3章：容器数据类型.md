**任务**：实现一个简单的数据集管理工具，模拟AI训练中的数据加载和预处理。

**背景**：AI训练中需要加载数据集、划分训练集和测试集、进行batch采样。

**要求**：

    - `shuffle(self)`：打乱数据顺序

**使用示例**：

### 实战关参考答案
```python
import random

class Dataset:
    """简单数据集管理"""
    
    def __init__(self, data, labels):
        if len(data) != len(labels):
            raise ValueError("data和labels长度必须一致")
        self.data = data
        self.labels = labels
        self.size = len(data)
    
    def shuffle(self):
        """打乱数据"""
        combined = list(zip(self.data, self.labels))
        random.shuffle(combined)
        self.data, self.labels = zip(*combined)
        self.data = list(self.data)
        self.labels = list(self.labels)
        return self
    
    def split(self, ratio=0.8):
        """划分训练集和测试集"""
        split_idx = int(self.size * ratio)
        return (self.data[:split_idx], self.labels[:split_idx],
                self.data[split_idx:], self.labels[split_idx:])
    
    def get_batch(self, batch_size, shuffle=True):
        """获取一个batch"""
        if shuffle:
            self.shuffle()
        
        batch_x = self.data[:batch_size]
        batch_y = self.labels[:batch_size]
        return batch_x, batch_y
    
    def label_distribution(self):
        """统计标签分布"""
        return dict(zip(*zip(*[(label, 1) for label in set(self.labels)])))


# 测试
if __name__ == "__main__":

    batch_x, batch_y = ds.get_batch(3)
    print(f"Batch X: {batch_x}")
    print(f"Batch Y: {batch_y}")
```

### 挑战关（60分钟）
**任务**：实现一个简单的词表构建工具，用于文本数据的预处理。

**背景**：NLP任务中需要构建词表（vocabulary），将文本转换为ID序列。

**要求**：

1. 实现`Vocabulary`类：
    - `__init__(self)`：初始化词表
    - `build_from_texts(texts, min_freq=1)`：从文本列表构建词表
    - `add_word(word)`：添加单个词
    - `word_to_id(word)`：词转ID，不存在则返回`<UNK>`的ID
    - `id_to_word(idx)`：ID转词
    - `encode(text, max_len=None)`：文本转ID序列
    - `decode(ids)`：ID序列转文本
2. 特殊标记：
    - `[PAD]`：填充标记，ID=0
    - `[UNK]`：未知词，ID=1
3. 分词：使用`str.split()`简单分词
4. 使用`Counter`统计词频

**使用示例**：

```python
texts = [
    "hello world",
    "hello python",
    "world of AI",
    "hello AI world"
]

vocab = Vocabulary()
vocab.build_from_texts(texts)

print(vocab.word_to_id("hello"))  # 2
print(vocab.word_to_id("AI"))     # 4
print(vocab.word_to_id("unknown"))  # 1 (UNK)

ids = vocab.encode("hello AI python", max_len=5)
print(ids)  # [2, 4, 3, 0, 0]

text = vocab.decode([2, 4, 3])
print(text)  # "hello AI python"
```

### 挑战关参考答案
```python
from collections import Counter

class Vocabulary:
    """词表管理器"""
    
    def __init__(self):
        self.word_to_id = {}
        self.id_to_word = {}
        self.vocab_size = 0
        
        # 特殊标记
        self.pad_token = "[PAD]"
        self.unk_token = "[UNK]"
        
        # 添加特殊标记
        self._add_word(self.pad_token)
        self._add_word(self.unk_token)
    
    def _add_word(self, word):
        """添加单词（内部方法）"""
        if word not in self.word_to_id:
            self.word_to_id[word] = self.vocab_size
            self.id_to_word[self.vocab_size] = word
            self.vocab_size += 1
    
    def build_from_texts(self, texts, min_freq=1):
        """从文本列表构建词表"""
        # 分词并统计词频
        words = []
        for text in texts:
            words.extend(text.split())
        
        word_freq = Counter(words)
        
        # 按频率排序添加（过滤低频词）
        for word, freq in sorted(word_freq.items(), key=lambda x: x[1], reverse=True):
            if freq >= min_freq:
                self._add_word(word)
    
    def add_word(self, word):
        """添加单个词"""
        self._add_word(word)
    
    def word_to_id(self, word):
        """词转ID"""
        return self.word_to_id.get(word, self.word_to_id[self.unk_token])
    
    def id_to_word(self, idx):
        """ID转词"""
        return self.id_to_word.get(idx, self.unk_token)
    
    def encode(self, text, max_len=None):
        """文本转ID序列"""
        words = text.split()
        ids = [self.word_to_id(word) for word in words]
        
        if max_len is not None:
            if len(ids) > max_len:
                ids = ids[:max_len]
            else:
                # 填充到max_len
                pad_id = self.word_to_id[self.pad_token]
                ids = ids + [pad_id] * (max_len - len(ids))
        
        return ids
    
    def decode(self, ids, remove_pad=True):
        """ID序列转文本"""
        words = []
        pad_id = self.word_to_id[self.pad_token]
        
        for idx in ids:
            if remove_pad and idx == pad_id:
                continue
            words.append(self.id_to_word(idx))
        
        return " ".join(words)


# 测试
if __name__ == "__main__":
    texts = [
        "hello world",
        "hello python",
        "world of AI",
        "hello AI world"
    ]
    
    vocab = Vocabulary()
    vocab.build_from_texts(texts, min_freq=1)
    
    print(f"词表大小: {vocab.vocab_size}")
    print(f"词表: {list(vocab.word_to_id.keys())}")
    
    print(f"hello -> ID: {vocab.word_to_id('hello')}")
    print(f"AI -> ID: {vocab.word_to_id('AI')}")
    print(f"unknown -> ID: {vocab.word_to_id('unknown')}")
    
    ids = vocab.encode("hello AI python", max_len=5)
    print(f"编码: {ids}")
    
    text = vocab.decode(ids)
    print(f"解码: {text}")
```



## 四、面试高频题
| 问题 | 参考答案 |
| --- | --- |
| 列表和元组的区别？ | 列表可变、元组不可变；列表用`[]`，元组用`()`；列表内存更大 |
| 为什么字典的键必须是不可变类型？ | 字典用哈希表实现，键必须可哈希，不可变类型才能保证哈希值不变 |
| `dict.get(key)`和`dict[key]`的区别？ | `get()`不存在返回None，`[]`不存在抛出KeyError |
| 如何快速去除列表中的重复元素？ | `list(set(lst))` |
| 列表推导式和生成器表达式区别？ | 列表推导式返回列表（立即计算），生成器返回生成器对象（惰性计算），内存效率更高 |
| `append()`和`extend()`的区别？ | `append`添加单个元素，`extend`合并另一个可迭代对象 |
| `copy()`和`deepcopy()`的区别？ | `copy`浅拷贝只复制外层，`deepcopy`递归复制所有层 |
| 如何遍历字典的键值对？ | `for key, value in dict.items():` |
| `zip()`的作用？ | 将多个可迭代对象对应位置打包成元组，用于并行遍历 |
