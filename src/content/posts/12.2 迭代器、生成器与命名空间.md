### 一、迭代器（Iterator）
**迭代**是遍历容器中元素的一种方式，而**迭代器**是一个可以记住遍历位置的对象。

| 特性 | 说明 |
| --- | --- |
| **访问方式** | 从第一个元素开始，逐个访问 |
| **遍历方向** | 只能向前，不能后退 |
| **结束标志** | 元素用尽时抛出 `StopIteration` 异常 |
| **适用对象** | 字符串、列表、元组等可迭代对象 |


### 二、可迭代对象（Iterable）
#### 2.1 什么是可迭代对象
可以直接作用于 `for` 循环的对象统称为**可迭代对象**。

**常见可迭代对象**：

+ 容器类型：`list`、`tuple`、`dict`、`set`、`str`
+ 生成器（Generator）
+ 文件对象

```python
# 以下都可以使用 for 循环遍历
for element in [1, 2, 3]:
    print(element)

for key in {"one": 1, "two": 2}:
    print(key)

for char in "123":
    print(char)

with open("myfile.txt", "w") as f:
    f.write("Hello\nWorld\n")
for line in open("myfile.txt"):
    print(line, end="")
```

#### 2.2 判断是否可迭代
使用 `isinstance()` 和 `Iterable` 判断：

```python
from collections.abc import Iterable

print(isinstance([], Iterable))      # True（列表）
print(isinstance((), Iterable))      # True（元组）
print(isinstance(set(), Iterable))   # True（集合）
print(isinstance({}, Iterable))      # True（字典）
print(isinstance("100", Iterable))   # True（字符串）
print(isinstance(100, Iterable))     # False（数字不可迭代）
```

#### 2.3 判断是否是迭代器
```python
from collections.abc import Iterator

print(isinstance([], Iterator))      # False（列表不是迭代器）
print(isinstance((), Iterator))      # False（元组不是迭代器）
print(isinstance("100", Iterator))   # False（字符串不是迭代器）
print(isinstance((x for x in range(10)), Iterator))  # True（生成器是迭代器）
```

### 三、使用迭代器
迭代器有两个基本方法：`iter()` 和 `next()`。

#### 3.1 工作原理
```plain
for 循环背后发生的事情：

1. 在容器对象上调用 iter() → 返回迭代器对象
2. 迭代器对象定义了 __next__() 方法
3. 每次循环调用 __next__() 获取下一个元素
4. 元素用尽时抛出 StopIteration 异常 → 循环结束
```

#### 3.2 使用 `iter()` 和 `next()`
```python
# 创建迭代器
my_list = [1, 2, 3]
it = iter(my_list)

# 使用 next() 遍历
print(next(it))   # 1
print(next(it))   # 2
print(next(it))   # 3
# print(next(it))   # StopIteration（元素已用完）
```

#### 3.3 使用 `for` 遍历迭代器
```python
my_list = [1, 2, 3]
it = iter(my_list)

for i in it:
    print(i)   # 1, 2, 3
```

> 💡 `for` 循环会自动捕获 `StopIteration` 异常，因此不需要手动处理。
>

### 四、创建自定义迭代器
通过实现 `__iter__()` 和 `__next__()` 方法，可以为类添加迭代器行为。

```python
class Reverse:
    """对一个序列执行反向循环的迭代器"""
    
    def __init__(self, data):
        self.data = data
        self.index = len(data)
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.index == 0:
            raise StopIteration
        self.index -= 1
        return self.data[self.index]

# 使用
rev = Reverse([2, 3, 5, 7, 11, 13, 17, 19])
for char in rev:
    print(char)   # 19, 17, 13, 11, 7, 5, 3, 2
```

### 五、生成器（Generator）
#### 5.1 什么是生成器
**生成器**是一个用于创建迭代器的简单而强大的工具，使用 `yield` 语句返回值。

| 特性 | 说明 |
| --- | --- |
| **惰性求值** | 按需生成值，不一次性计算所有结果 |
| **节省内存** | 避免一次性生成大量数据 |
| **状态保留** | 函数执行状态在每次 `yield` 后保留 |
| **无缝配合** | 可与 `for` 循环等迭代工具配合使用 |


#### 5.2 创建生成器
**方式1：使用推导式**

```python
# 列表推导式 vs 生成器推导式
list_comp = [x for x in range(5)]     # 列表
generator = (x for x in range(5))     # 生成器

print(list_comp)   # [0, 1, 2, 3, 4]
print(generator)   # <generator object at 0x...>

for x in generator:
    print(x)       # 0, 1, 2, 3, 4
```

**方式2：使用函数和 **`yield`

```python
def fibonacci():
    """斐波那契数列生成器"""
    a, b = 0, 1
    while True:
        yield b
        a, b = b, a + b

# 使用生成器
f = fibonacci()
print(next(f))   # 1
print(next(f))   # 1
print(next(f))   # 2
print(next(f))   # 3
print(next(f))   # 5
```

#### 5.3 捕获 `return` 值
生成器函数中的 `return` 值会在 `StopIteration` 异常中返回：

```python
def fibonacci(n):
    """生成前 n 个斐波那契数"""
    a, b, counter = 0, 1, 0
    while counter < n:
        yield b
        a, b, counter = b, a + b, counter + 1
    return "done"

f = fibonacci(5)
try:
    while True:
        print(next(f))
except StopIteration as result:
    print(f"生成器结束: {result.value}")  # 生成器结束: done
```

**输出**：

```plain
1
1
2
3
5
生成器结束: done
```

### 六、`send()` 方法
`send()` 方法可以向生成器发送值，该值作为当前 `yield` 表达式的结果。

#### 6.1 基本用法
```python
def gen():
    task_id = 0
    int_value = 0
    char_value = "A"
    
    while True:
        match task_id:
            case 0:
                task_id = yield int_value
                int_value += 1
            case 1:
                task_id = yield char_value
                char_value = chr(ord(char_value) + 1)
            case _:
                task_id = yield

g = gen()

# 启动生成器
print(next(g))         # 0（第一次 yield int_value）
print(g.send(1))       # A（发送 1，切换到 task_id=1）
print(g.send(0))       # 1（发送 0，切换到 task_id=0）
print(g.send(1))       # B
print(g.send(0))       # 2
```

#### 6.2 使用 `send(None)` 启动
当调用 `send()` 启动生成器时，必须以 `None` 作为参数：

```python
g = gen()
print(g.send(None))    # 0（启动生成器）
print(g.send(1))       # A
```

> 💡 `send(None)` 等价于 `next(g)`，用于启动生成器。
>

### 七、迭代器 vs 生成器对比
| 特性 | 迭代器（Iterator） | 生成器（Generator） |
| --- | --- | --- |
| **定义方式** | 实现 `__iter__` 和 `__next__` | 使用 `yield` 的函数或推导式 |
| **代码量** | 较多 | 较少 |
| **状态管理** | 手动管理 | 自动维护 |
| **适用场景** | 复杂迭代逻辑 | 简单序列生成 |
| **内存效率** | 高 | 高（惰性求值） |


```python
# 迭代器方式（较复杂）
class CounterIterator:
    def __init__(self, n):
        self.n = n
        self.i = 0
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.i >= self.n:
            raise StopIteration
        self.i += 1
        return self.i

# 生成器方式（更简洁）
def counter_generator(n):
    for i in range(1, n + 1):
        yield i
```

### 八、命名空间（Namespace）
#### 8.1 什么是命名空间
**命名空间**是从名称到对象的映射，类似于一个字典，键是名称，值是对象。

| 特点 | 说明 |
| --- | --- |
| **独立性** | 各个命名空间相互独立 |
| **无重名冲突** | 同一命名空间中不能有重名 |
| **跨空间可重名** | 不同命名空间可以有同名名称 |


#### 8.2 三种命名空间
| 命名空间 | 创建时机 | 生命周期 | 示例 |
| --- | --- | --- | --- |
| **内置名称** | Python 解释器启动时 | 永不删除 | `print`、`len`、`int` |
| **全局名称** | 读取模块定义时 | 直到解释器退出 | 模块中定义的变量、函数 |
| **局部名称** | 函数调用时 | 函数返回或抛出异常时删除 | 函数内的局部变量 |


```python
# 内置名称（Python 自带）
print(len)   # <built-in function len>

# 全局名称（模块级别）
global_var = 100

def my_function():
    # 局部名称（函数内部）
    local_var = 200
    print(local_var)

print(global_var)   # 100（全局命名空间）
my_function()       # 200（局部命名空间）
# print(local_var)  # ❌ NameError（局部变量外部无法访问）
```

#### 8.3 命名空间查找顺序
```plain
当访问一个变量时，Python 按以下顺序查找：

1. 局部命名空间（Local）
2. 嵌套命名空间（Enclosing）
3. 全局命名空间（Global）
4. 内置命名空间（Built-in）

L → E → G → B
```

```python
# 演示查找顺序
x = "global"   # 全局

def outer():
    x = "enclosing"   # 嵌套
    
    def inner():
        x = "local"   # 局部
        print(x)      # 查找顺序：局部 → 嵌套 → 全局 → 内置
    
    inner()

outer()   # 输出: local
```

### 九、综合示例
```python
"""
生成器应用：逐行读取大文件
"""
import os

def read_large_file(filepath, chunk_size=1024):
    """
    逐行读取大文件的生成器
    内存友好，适合处理超大文件
    """
    with open(filepath, 'r', encoding='utf-8') as f:
        line = f.readline()
        while line:
            yield line.rstrip('\n')
            line = f.readline()

def process_data_generator():
    """
    数据处理生成器示例
    模拟从数据源逐步获取数据
    """
    data = list(range(10))
    for item in data:
        # 模拟处理
        result = item ** 2
        yield f"处理结果: {result}"

# 使用示例
if __name__ == "__main__":
    # 创建测试文件
    test_file = "test_data.txt"
    with open(test_file, 'w', encoding='utf-8') as f:
        for i in range(10):
            f.write(f"第 {i+1} 行数据\n")
    
    # 使用生成器逐行读取
    print("=== 逐行读取文件 ===")
    for line in read_large_file(test_file):
        print(line)
    
    # 使用数据处理生成器
    print("\n=== 数据处理生成器 ===")
    processor = process_data_generator()
    for result in processor:
        print(result)
    
    # 清理测试文件
    os.remove(test_file)
```

### 十、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 所有可迭代对象都是迭代器 | ❌ 可迭代对象不等于迭代器，需通过 `iter()` 转换 |
| 迭代器可以重复遍历 | ❌ 迭代器只能遍历一次，耗尽后无法重置 |
| 生成器函数返回的是值 | ❌ 生成器函数返回的是生成器对象 |
| `yield` 和 `return` 类似 | ❌ `yield` 暂停函数并返回值，`return` 终止函数 |
| `send()` 可以随意发送任意值 | ❌ 首次调用必须用 `send(None)` 或 `next()` |
| 命名空间可以任意访问 | ❌ 局部命名空间的变量外部无法访问 |


```python
# 错误示例：迭代器只能遍历一次
my_list = [1, 2, 3]
it = iter(my_list)
print(list(it))   # [1, 2, 3]
print(list(it))   # [] ← 第二次为空！

# 正确做法：每次需要时重新创建迭代器
it = iter(my_list)   # 重新创建
```

> 📌 **核心记忆**：
>
> + **可迭代对象**：可使用 `for` 循环的对象（`list`、`tuple`、`str` 等）
> + **迭代器**：通过 `iter()` 创建，用 `next()` 访问，只能向前遍历
> + **生成器**：使用 `yield` 的函数或推导式，惰性求值，节省内存
> + `send()`：向生成器发送值，首次调用需用 `send(None)`
> + **命名空间**：名称到对象的映射，分内置、全局、局部三种
> + **查找顺序**：L → E → G → B（局部 → 嵌套 → 全局 → 内置）
>
