## Python 面向对象之封装与继承学习笔记
### 一、封装（Encapsulation）
**封装**是将变量（属性）和函数（方法）写入类中的操作，将数据和对数据的操作组织在一起。

| 目的 | 说明 |
| --- | --- |
| **隐藏细节** | 将内部实现细节隐藏起来 |
| **暴露接口** | 只暴露出必要的接口供调用者使用 |
| **保护数据** | 防止外部直接修改内部数据，保证数据安全 |


### 二、私有化（Privatization）
Python 通过命名约定来实现访问控制：

| 命名方式 | 含义 | 示例 |
| --- | --- | --- |
| **单下划线前缀** | "非公开API"的约定（不强制） | `_var` |
| **双下划线前缀** | 名称改写（Name Mangling） | `__var` |


#### 2.1 单下划线：非公开API约定
```python
class Person:
    def __init__(self, name):
        self._name = name   # 约定为"非公开"，但不强制

p = Person("张三")
print(p._name)   # 可以访问（仅约定，不强制）
```

> 💡 这是一种**约定**，告知其他开发者这些属性/方法不应直接使用，但 Python 并不强制。
>

#### 2.2 双下划线：名称改写（Name Mangling）
```python
class Person:
    def __init__(self, name):
        self.__name = name   # 双下划线前缀

p = Person("张三")

# 不能直接访问
# print(p.__name)   # ❌ AttributeError

# 可以通过改写后的名称访问（但不建议）
print(p._Person__name)  # 张三
```

**名称改写规则**：`__x` → `_类名__x`

### 三、私有属性
```python
class Person:
    def __init__(self, name):
        self.__name = name   # 私有属性
    
    # 提供公共方法访问私有属性
    def get_name(self):
        return self.__name
    
    def set_name(self, name):
        self.__name = name

p = Person("张三")
print(p.get_name())          # 张三（通过公共方法访问）
print(p._Person__name)       # 张三（名称改写后访问，不推荐）
# print(p.__name)            # ❌ AttributeError
```

### 四、私有方法
```python
class Person:
    # 私有方法
    def __private_method(self):
        print("private method")
    
    # 公共方法调用私有方法
    def do_something(self):
        self.__private_method()

p = Person()
p.do_something()                  # private method
p._Person__private_method()       # private method（不推荐）
# p.__private_method()            # ❌ AttributeError
```

### 五、property 装饰器
`@property` 可以将方法转换为属性调用，实现更优雅的访问控制。

#### 5.1 方法转换为属性
```python
class Person:
    def __init__(self, name):
        self.name = name
    
    @property
    def eat(self):
        print(f"{self.name} is eating...")

p = Person("张三")
p.eat   # 直接使用 .属性名，无需加 ()
# 张三 is eating...
```

#### 5.2 只读属性（只提供 getter）
```python
class Person:
    def __init__(self, name):
        self.__name = name
    
    @property
    def name(self):
        """只读属性：只能获取，不能修改"""
        return self.__name

p = Person("张三")
print(p.name)    # 张三
# p.name = "李四"  # ❌ AttributeError: can't set attribute
```

#### 5.3 读写属性（提供 getter 和 setter）
```python
class Person:
    def __init__(self, name):
        self.__name = name
    
    @property
    def name(self):
        return self.__name
    
    @name.setter
    def name(self, name):
        self.__name = name

p = Person("张三")
print(p.name)    # 张三

p.name = "李四"  # 调用 setter
print(p.name)    # 李四
```

#### 5.4 带验证的 setter
```python
class Person:
    def __init__(self, name):
        self.__name = name
    
    @property
    def name(self):
        return self.__name
    
    @name.setter
    def name(self, name):
        if name == "李四":
            print("不许叫李四")
        else:
            self.__name = name

p = Person("张三")
print(p.name)      # 张三

p.name = "李四"     # 不许叫李四
print(p.name)      # 张三（未被修改）

p.name = "王五"
print(p.name)      # 王五
```

#### 5.5 ⚠️ property 常见陷阱
```python
class Person:
    @property
    def name(self):
        return self.name   # ❌ 无限递归！因为 self.name 又调用了 property

p = Person()
# p.name  # RecursionError: maximum recursion depth exceeded
```

**正确做法**：property 方法中应返回私有属性（带双下划线）的值。

```python
class Person:
    def __init__(self, name):
        self.__name = name
    
    @property
    def name(self):
        return self.__name   # ✅ 返回私有属性
```

### 六、继承（Inheritance）
**继承**是子类（派生类）从父类（基类）获取属性和方法的机制，实现代码重用。

| 角色 | 说明 |
| --- | --- |
| **父类（基类）** | 被继承的类 |
| **子类（派生类）** | 继承父类的类 |
| **单继承** | 一个子类只继承一个父类 |
| **多继承** | 一个子类继承多个父类 |


#### 6.1 单继承
**语法**：`class 子类名(父类名):`

```python
# 定义父类
class Animal:
    def __init__(self, name):
        self.name = name
    
    def eat(self):
        print(f"{self.name} is eating...")
    
    def sleep(self):
        print(f"{self.name} is sleeping...")

# 定义子类，继承 Animal
class Dog(Animal):
    def bark(self):
        print(f"{self.name} is barking...")

# 使用子类
dog = Dog("旺财")
dog.eat()    # 继承自父类：旺财 is eating...
dog.sleep()  # 继承自父类：旺财 is sleeping...
dog.bark()   # 子类特有：旺财 is barking...
```

#### 6.2 方法重写（Override）
子类可以重写父类的方法，实现自己的行为。

```python
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        print("动物发出声音")

class Dog(Animal):
    def speak(self):           # 重写父类的 speak 方法
        print(f"{self.name} 汪汪叫")

class Cat(Animal):
    def speak(self):
        print(f"{self.name} 喵喵叫")

# 使用
dog = Dog("旺财")
cat = Cat("咪咪")

dog.speak()   # 旺财 汪汪叫
cat.speak()   # 咪咪 喵喵叫
```

#### 6.3 调用父类方法：`super()`
子类中可以使用 `super()` 调用父类的方法。

```python
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def eat(self):
        print("动物在吃东西")

class Dog(Animal):
    def __init__(self, name, age, breed):
        # 调用父类的 __init__ 方法
        super().__init__(name, age)
        self.breed = breed
    
    def eat(self):
        # 调用父类的 eat 方法
        super().eat()
        print(f"{self.name} 在啃骨头")

dog = Dog("旺财", 3, "金毛")
print(f"名字: {dog.name}, 年龄: {dog.age}, 品种: {dog.breed}")
dog.eat()
# 动物在吃东西
# 旺财 在啃骨头
```

#### 6.4 继承与私有成员
子类**不能继承**父类的私有属性和私有方法（双下划线）。

```python
class Parent:
    def __init__(self):
        self.__private_attr = "私有属性"
        self.public_attr = "公共属性"
    
    def __private_method(self):
        print("私有方法")
    
    def public_method(self):
        print("公共方法")

class Child(Parent):
    def test(self):
        # print(self.__private_attr)   # ❌ 不能访问父类私有属性
        print(self.public_attr)        # ✅ 可以访问父类公共属性
        # self.__private_method()      # ❌ 不能调用父类私有方法
        self.public_method()           # ✅ 可以调用父类公共方法

c = Child()
c.test()
# 公共属性
# 公共方法

# 通过名称改写可以访问父类私有成员（不推荐）
print(c._Parent__private_attr)  # 私有属性
c._Parent__private_method()     # 私有方法
```

> ⚠️ **注意**：虽然可以通过名称改写访问私有成员，但这**违背了封装原则**，不应该在生产代码中使用。
>

#### 6.5 多继承
Python 支持多继承，一个子类可以继承多个父类。

```python
class Flyable:
    def fly(self):
        print("Flying...")

class Swimmable:
    def swim(self):
        print("Swimming...")

class Duck(Flyable, Swimmable):
    def quack(self):
        print("Quack!")

duck = Duck()
duck.fly()    # Flying...
duck.swim()   # Swimming...
duck.quack()  # Quack!
```

**多继承的 MRO（方法解析顺序）**：

```python
class A:
    def method(self):
        print("A.method")

class B(A):
    def method(self):
        print("B.method")

class C(A):
    def method(self):
        print("C.method")

class D(B, C):
    pass

d = D()
d.method()  # B.method（按 MRO 顺序查找）

# 查看 MRO
print(D.__mro__)
# (<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>)
```

### 七、综合示例：宠物管理系统
```python
class Pet:
    """宠物基类"""
    
    def __init__(self, name, age):
        self.__name = name   # 私有属性
        self.age = age
    
    @property
    def name(self):
        """只读属性：姓名不能修改"""
        return self.__name
    
    def eat(self):
        print(f"{self.name} 在吃东西")
    
    def sleep(self):
        print(f"{self.name} 在睡觉")
    
    def speak(self):
        """子类重写的方法"""
        pass


class Dog(Pet):
    """狗类"""
    
    def __init__(self, name, age, breed):
        super().__init__(name, age)
        self.breed = breed
    
    def speak(self):
        print(f"{self.name} 汪汪叫")
    
    def guard(self):
        print(f"{self.name} 在看家护院")


class Cat(Pet):
    """猫类"""
    
    def __init__(self, name, age, color):
        super().__init__(name, age)
        self.__color = color   # 私有属性
    
    @property
    def color(self):
        return self.__color
    
    @color.setter
    def color(self, color):
        if color in ["白色", "黑色", "橘色", "灰色"]:
            self.__color = color
        else:
            print(f"不支持的毛色: {color}")
    
    def speak(self):
        print(f"{self.name} 喵喵叫")
    
    def climb(self):
        print(f"{self.name} 在爬树")


class Bird(Pet):
    """鸟类"""
    
    def speak(self):
        print(f"{self.name} 叽叽喳喳")
    
    def fly(self):
        print(f"{self.name} 在飞翔")


# 使用示例
print("=== 狗狗 ===")
dog = Dog("旺财", 3, "金毛")
dog.eat()          # 继承自 Pet
dog.sleep()        # 继承自 Pet
dog.speak()        # 重写父类方法
dog.guard()        # Dog 特有方法
print(f"品种: {dog.breed}")

print("\n=== 猫咪 ===")
cat = Cat("咪咪", 2, "橘色")
cat.speak()        # 重写父类方法
cat.climb()        # Cat 特有方法
print(f"毛色: {cat.color}")
cat.color = "灰色"  # 使用 setter
print(f"新毛色: {cat.color}")

print("\n=== 小鸟 ===")
bird = Bird("小叽", 1)
bird.speak()       # 重写父类方法
bird.fly()         # Bird 特有方法

print("\n=== 多态演示 ===")
pets = [dog, cat, bird]
for pet in pets:
    pet.speak()    # 同一个方法名，不同行为
```

### 八、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 单下划线和双下划线效果相同 | ❌ 单下划线是约定，双下划线触发表名改写 |
| 子类可以继承父类私有成员 | ❌ 子类不能继承私有属性和方法 |
| `@property` 方法中访问同名属性 | ❌ 会无限递归，应访问私有属性 |
| 多继承时同名方法优先级不确定 | 按 MRO（方法解析顺序）查找 |
| 父类 `__init__` 不会自动调用 | 子类中需用 `super()` 显式调用 |


```python
# 错误示例1：property 无限递归
class Bad:
    @property
    def name(self):
        return self.name   # ❌ 无限递归
# Bad().name  # RecursionError

# 正确写法
class Good:
    def __init__(self, name):
        self.__name = name
    
    @property
    def name(self):
        return self.__name   # ✅ 返回私有属性

# 错误示例2：忘记调用父类 __init__
class Child(Parent):
    def __init__(self):
        pass  # ❌ 父类的 __init__ 没有被调用，父类属性未初始化

# 正确写法
class Child(Parent):
    def __init__(self):
        super().__init__()  # ✅ 调用父类 __init__
```

### 九、快速参考卡片
```python
# === 私有属性和方法 ===
class Example:
    def __init__(self):
        self.__private = "私有"      # 双下划线：名称改写
        self._protected = "保护"     # 单下划线：约定（不强制）
        self.public = "公共"         # 公共属性
    
    def __private_method(self):     # 私有方法
        pass
    
    def public_method(self):
        self.__private_method()      # 内部可访问私有方法

# === property 使用 ===
class Person:
    def __init__(self, name):
        self.__name = name
    
    @property
    def name(self):                  # getter
        return self.__name
    
    @name.setter
    def name(self, value):           # setter
        self.__name = value
    
    @name.deleter
    def name(self):                  # deleter
        del self.__name

# === 继承 ===
class Parent:
    def __init__(self, name):
        self.name = name
    
    def method(self):
        pass

class Child(Parent):
    def __init__(self, name, age):
        super().__init__(name)       # 调用父类 __init__
        self.age = age
    
    def method(self):                # 方法重写
        super().method()             # 调用父类方法
        print("Child method")

# === 多继承 ===
class A: pass
class B: pass
class C(A, B): pass                  # 多继承
```

> 📌 **核心记忆**：
>
> + **封装**：将数据和操作组织在一起，通过私有化隐藏内部细节
> + **私有化**：`__var` 触发名称改写（`_类名__var`），`_var` 仅约定
> + `@property`：将方法转换为属性，提供 getter/setter 控制
> + **继承**：子类继承父类的**非私有**属性和方法
> + `super()`：在子类中调用父类的方法
> + **方法重写**：子类可以覆盖父类的方法
> + **多继承**：子类可以继承多个父类，MRO 决定方法查找顺序
>
