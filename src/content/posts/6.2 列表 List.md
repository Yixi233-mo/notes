### 一、列表概述
**列表（List）** 是 Python 中最常用的序列类型，具有以下特点：

| 特性 | 说明 |
| --- | --- |
| **可变性** | ✅ 可修改（增、删、改） |
| **有序性** | ✅ 元素有固定顺序 |
| **索引支持** | ✅ 支持正向和负向索引 |
| **元素类型** | ✅ 可包含不同类型的元素 |
| **重复元素** | ✅ 允许重复 |


**定义方式**：使用 `[]` 定义，元素之间用 `,` 分隔

```python
list1 = [100, 200, 300, 400, 500]           # 纯数字列表
list2 = [1, "hello", 3.14, True]            # 混合类型列表
list3 = []                                   # 空列表
```

### 二、创建列表
```python
# 方式1：直接使用方括号
list1 = [100, 200, 300, 400, 500]

# 方式2：使用 list() 构造函数
list2 = list((1, 2, 3))      # 从元组转换
list3 = list("hello")        # 从字符串转换 → ['h','e','l','l','o']

# 方式3：列表推导式
list4 = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]
```

### 三、访问列表元素
#### 1. 通过索引访问
```python
list1 = [100, 200, 300, 400, 500]
print(list1[0])    # 100（第一个元素）
print(list1[2])    # 300（第三个元素）
print(list1[-1])   # 500（最后一个元素）
print(list1[-2])   # 400（倒数第二个元素）
```

#### 2. 切片操作
```python
list1 = [100, 200, 300, 400, 500]

print(list1)        # [100, 200, 300, 400, 500]（全部）
print(list1[:])     # [100, 200, 300, 400, 500]（复制整个列表）
print(list1[2:4])   # [300, 400]（索引2到3）
print(list1[2:])    # [300, 400, 500]（索引2到末尾）
print(list1[:2])    # [100, 200]（开头到索引1）
print(list1[2:-1])  # [300, 400]（索引2到倒数第二个）
print(list1[::-1])  # [500, 400, 300, 200, 100]（倒序）
```

**切片规则总结**：

> `[start:stop:step]` — 左闭右开，步长方向
>

### 四、向列表添加元素
| 方法 | 说明 | 示例 |
| --- | --- | --- |
| `append(x)` | 在**末尾**追加一个元素 | `list1.append(600)` |
| `insert(i, x)` | 在**指定位置**插入元素 | `list1.insert(2, 700)` |
| `extend(iterable)` | 在末尾追加另一个序列的所有元素 | `list1.extend([800,900])` |


```python
list1 = [100, 200, 300, 400, 500]

# append：末尾追加
list1.append(600)
print(list1)   # [100, 200, 300, 400, 500, 600]

# insert：指定位置插入
list1.insert(2, 700)   # 在索引2处插入700
print(list1)   # [100, 200, 700, 300, 400, 500, 600]

# extend：合并列表
list1.extend([800, 900])
print(list1)   # [100, 200, 700, 300, 400, 500, 600, 800, 900]
```

### 五、列表相加与乘法
```python
# 列表相加（拼接）
list1 = [100, 200, 300]
list2 = ["a", "b", "c"]
print(list1 + list2)   # [100, 200, 300, 'a', 'b', 'c']

# 列表乘法（重复）
print(list1 * 2)       # [100, 200, 300, 100, 200, 300]
```

> ⚠️ `+` 和 `*` 都返回**新列表**，不修改原列表。
>

### 六、修改列表元素
#### 1. 通过下标修改
```python
list1 = [100, 200, 300, 400, 500]
list1[0] = -1
print(list1)   # [-1, 200, 300, 400, 500]
```

#### 2. 通过切片修改
```python
list1 = [100, 200, 300, 400, 500]
list1[2:4] = ["a", "b", "c"]   # 替换索引2~3的元素
print(list1)   # [100, 200, 'a', 'b', 'c', 500]

# 切片替换可以改变列表长度
list1[1:3] = [1, 2, 3, 4]      # 插入多个元素
print(list1)   # [100, 1, 2, 3, 4, 'c', 500]
```

### 七、删除列表元素
| 方法 | 说明 | 示例 |
| --- | --- | --- |
| `del list[i]` | 删除指定位置元素 | `del list1[2]` |
| `del list[i:j]` | 删除切片 | `del list1[1:3]` |
| `remove(x)` | 删除第一次出现的 x | `list1.remove(300)` |
| `pop(i)` | 删除并返回指定位置元素（默认末尾） | `list1.pop()` |
| `clear()` | 清空所有元素 | `list1.clear()` |


```python
list1 = [100, 200, 300, 400, 500]

# del 删除指定位置
del list1[2]
print(list1)   # [100, 200, 400, 500]

# remove 删除指定值（第一次出现）
list1.remove(200)
print(list1)   # [100, 400, 500]

# pop 删除并返回
last = list1.pop()     # 删除末尾
print(last, list1)     # 500 [100, 400]

# clear 清空
list1.clear()
print(list1)   # []
```

### 八、列表的常用操作
#### 1. 成员检查
```python
list1 = [100, 200, 300]
print(100 in list1)     # True
print(500 not in list1) # True
```

#### 2. 获取长度、最值、求和
```python
list1 = [100, 200, 300, 400, 500]

print(len(list1))   # 5（元素个数）
print(max(list1))   # 500（最大值）
print(min(list1))   # 100（最小值）
print(sum(list1))   # 1500（所有元素和）
```

#### 3. 查找元素位置和计数
```python
list1 = [10, 20, 30, 20, 40, 20]

print(list1.index(20))        # 1（第一次出现的位置）
print(list1.index(20, 2))     # 3（从索引2开始查找）
print(list1.count(20))        # 3（20出现的次数）
```

### 九、遍历列表
#### 方式1：直接遍历元素
```python
list1 = [100, 200, 300, 400, 500]
for val in list1:
    print(val)
```

#### 方式2：通过下标遍历
```python
list1 = [100, 200, 300, 400, 500]
for i in range(len(list1)):
    print(i, list1[i])
```

#### 方式3：使用 `enumerate()` 同时获取索引和值（推荐）
```python
list1 = [100, 200, 300, 400, 500]
for i, val in enumerate(list1):
    print(f"索引{i}: {val}")
```

### 十、列表排序与反转
| 方法 | 说明 | 是否修改原列表 |
| --- | --- | --- |
| `sorted(list)` | 返回排序后的新列表 | ❌ 不修改 |
| `list.sort()` | 就地排序 | ✅ 修改 |
| `list.reverse()` | 反转列表 | ✅ 修改 |
| `list[::-1]` | 返回反转的新列表 | ❌ 不修改 |


```python
list1 = [3, 1, 4, 1, 5, 9, 2]

# sorted() 返回新列表
new_list = sorted(list1)
print(new_list)      # [1, 1, 2, 3, 4, 5, 9]
print(list1)         # [3, 1, 4, 1, 5, 9, 2]（原列表不变）

# sort() 就地排序
list1.sort()
print(list1)         # [1, 1, 2, 3, 4, 5, 9]

# 降序排序
list1.sort(reverse=True)
print(list1)         # [9, 5, 4, 3, 2, 1, 1]

# 反转
list1.reverse()
print(list1)         # [1, 1, 2, 3, 4, 5, 9]
```

### 十一、嵌套列表
列表中的元素可以是另一个列表，形成多维数组结构。

```python
# 创建一个 3x3 的嵌套列表
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

# 遍历嵌套列表
for inner_list in matrix:
    print(inner_list)

# 访问嵌套列表的元素
print(matrix[0][0])    # 1（第一行第一列）
print(matrix[1][2])    # 6（第二行第三列）

# 修改嵌套列表元素
matrix[0][0] = 100
print(matrix)          # [[100, 2, 3], [4, 5, 6], [7, 8, 9]]
```

### 十二、列表推导式（List Comprehension）
列表推导式是 Python 中创建列表的**简洁优雅**的方式。

#### 1. 基础语法
```python
[表达式 for 变量 in 可迭代对象]
```

```python
# 传统写法
squares = []
for x in range(5):
    squares.append(x**2)

# 列表推导式
squares = [x**2 for x in range(5)]
print(squares)   # [0, 1, 4, 9, 16]
```

#### 2. 带条件的列表推导式
```python
[表达式 for 变量 in 可迭代对象 if 条件]
```

```python
# 筛选偶数并计算平方
squares = [x**2 for x in range(10) if x % 2 == 0]
print(squares)   # [0, 4, 16, 36, 64]
```

#### 3. 使用现有列表
```python
list1 = [1, 2, 3, 4, 5]
squares = [x**2 for x in list1]
print(squares)   # [1, 4, 9, 16, 25]
```

#### 4. 多个循环的列表推导式
```python
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']

# 笛卡尔积
pairs = [(i, j) for i in list1 for j in list2]
print(pairs)  # [(1,'a'), (1,'b'), (1,'c'), (2,'a'), ...]
```

### 十三、`zip()` 函数
`zip()` 将多个可迭代对象的对应元素打包成元组。

```python
list1 = [1, 2, 3, 4, 5]
list2 = ["a", "b", "c", "d", "e"]

zipped = zip(list1, list2)
print(list(zipped))  # [(1,'a'), (2,'b'), (3,'c'), (4,'d'), (5,'e')]

# 遍历配对元素
for num, letter in zip(list1, list2):
    print(f"{num} -> {letter}")

# 长度不一致时，以最短的为准
list3 = [1, 2, 3]
list4 = ['a', 'b', 'c', 'd', 'e']
print(list(zip(list3, list4)))  # [(1,'a'), (2,'b'), (3,'c')]
```

### 十四、列表的复制（深拷贝 vs 浅拷贝）
```python
# 浅拷贝方式
original = [1, 2, [3, 4]]

copy1 = original[:]           # 切片复制
copy2 = original.copy()       # copy()方法
copy3 = list(original)        # 构造函数

# 修改外层元素
copy1[0] = 100
print(original[0])   # 1（不受影响）

# 修改内层列表（重要！）
copy1[2][0] = 999
print(original[2])   # [999, 4]（受影响！）

# 深拷贝（创建完全独立的副本）
import copy
deep_copy = copy.deepcopy(original)
deep_copy[2][0] = 777
print(original[2])   # [999, 4]（不受影响）
```

| 复制方式 | 代码 | 内层可变对象是否独立 |
| --- | --- | --- |
| 浅拷贝 | `list[:]`、`list.copy()` | ❌ 共享 |
| 深拷贝 | `copy.deepcopy()` | ✅ 独立 |


### 十五、列表常用函数速查表
| 函数/方法 | 说明 | 示例 |
| --- | --- | --- |
| `list.append(x)` | 末尾追加 | `[1,2].append(3)` → `[1,2,3]` |
| `list.insert(i,x)` | 指定位置插入 | `[1,3].insert(1,2)` → `[1,2,3]` |
| `list.extend(iter)` | 扩展列表 | `[1].extend([2,3])` → `[1,2,3]` |
| `del list[i]` | 删除指定位置 | `del [1,2,3][1]` → `[1,3]` |
| `list.remove(x)` | 删除指定值 | `[1,2,3].remove(2)` → `[1,3]` |
| `list.pop(i)` | 删除并返回 | `[1,2,3].pop(1)` → `2` |
| `list.clear()` | 清空列表 | `[1,2,3].clear()` → `[]` |
| `list.index(x)` | 查找位置 | `[1,2,3].index(2)` → `1` |
| `list.count(x)` | 计数 | `[1,2,2,3].count(2)` → `2` |
| `list.sort()` | 排序（原地） | `[3,1,2].sort()` → `[1,2,3]` |
| `sorted(list)` | 排序（新列表） | `sorted([3,1,2])` → `[1,2,3]` |
| `list.reverse()` | 反转（原地） | `[1,2,3].reverse()` → `[3,2,1]` |
| `len(list)` | 长度 | `len([1,2,3])` → `3` |
| `max(list)` | 最大值 | `max([1,2,3])` → `3` |
| `min(list)` | 最小值 | `min([1,2,3])` → `1` |
| `sum(list)` | 求和 | `sum([1,2,3])` → `6` |
| `list.copy()` | 浅拷贝 | `[1,2,3].copy()` |
| `list(iterable)` | 转换为列表 | `list("abc")` → `['a','b','c']` |


### 十六、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| `append()` 一次添加多个元素 | 用 `extend()` 或 `+` |
| `list = list + [x]` 不会修改原列表 | 返回新列表，需重新赋值 |
| 切片 `[i:j]` 修改时长度可以不同 | 切片替换会调整列表长度 |
| 列表乘法 `[[]]*n` 创建的是引用 | 内层列表指向同一对象 |
| `sort()` 和 `sorted()` 混淆 | `sort()` 原地修改，`sorted()` 返回新列表 |
| 浅拷贝内层可变对象 | 用 `copy.deepcopy()` 创建完全独立副本 |


### 十七、完整综合示例
```python
# 学生成绩管理系统示例

# 创建学生列表
students = ["张三", "李四", "王五", "赵六"]
scores = [85, 92, 78, 88]

# 1. 添加新学生
students.append("小明")
scores.append(95)

# 2. 修改李四的成绩
index = students.index("李四")
scores[index] = 96

# 3. 删除王五
index = students.index("王五")
del students[index]
del scores[index]

# 4. 计算统计信息
print(f"学生列表: {students}")
print(f"成绩列表: {scores}")
print(f"平均分: {sum(scores)/len(scores):.2f}")
print(f"最高分: {max(scores)}")
print(f"最低分: {min(scores)}")

# 5. 使用 enumerate 遍历
print("\n成绩单:")
for i, (name, score) in enumerate(zip(students, scores)):
    print(f"{i+1}. {name}: {score}分")

# 6. 列表推导式：筛选及格学生
passing = [(name, score) for name, score in zip(students, scores) if score >= 90]
print(f"\n优秀学生: {passing}")
```

> 📌 **核心记忆**：
>
> + 列表是**可变、有序**的序列，用 `[]` 定义
> + 索引从 0 开始，支持负索引和切片
> + 常用操作：`append`、`insert`、`remove`、`pop`、`sort`、`reverse`
> + 遍历推荐 `for i, val in enumerate(list)`
> + 列表推导式是创建列表的**Pythonic**方式
> + 复制嵌套列表用 `copy.deepcopy()` 实现深拷贝
>
