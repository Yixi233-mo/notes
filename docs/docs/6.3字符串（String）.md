### 一、字符串概述
**字符串（String）** 是 Python 中表示文本数据的序列类型，具有以下特点：

| 特性 | 说明 |
| --- | --- |
| **不可变性** | ❌ 创建后不可修改 |
| **有序性** | ✅ 字符有固定顺序 |
| **索引支持** | ✅ 支持正向和负向索引 |
| **可迭代** | ✅ 可以使用 `for` 循环遍历 |


**定义方式**：使用单引号、双引号或三重引号

```python
str1 = 'hello world'      # 单引号
str2 = "hello world"      # 双引号
str3 = """hello
world"""                   # 三重引号（支持多行）
str4 = ''                  # 空字符串
```

### 二、创建字符串
```python
# 直接赋值
str1 = "hello world"

# 使用 str() 构造函数
str2 = str(123)           # "123"
str3 = str(3.14)          # "3.14"
str4 = str([1, 2, 3])     # "[1, 2, 3]"

# 多行字符串
str5 = """这是第一行
这是第二行
这是第三行"""
```

### 三、访问字符串元素
#### 1. 通过索引访问
```python
str1 = "hello world"
print(str1[0])     # 'h'（第一个字符）
print(str1[-1])    # 'd'（最后一个字符）
print(str1[4])     # 'o'
```

#### 2. 切片操作
```python
str1 = "hello world"
print(str1[4:-3])  # 'o w'
# 索引4 → 'o'，索引-3 → 'r'（不包含），取 'o', ' ', 'w'

print(str1[:5])    # 'hello'
print(str1[6:])    # 'world'
print(str1[::-1])  # 'dlrow olleh'（反转字符串）
```

> ⚠️ **重要**：字符串不可变，不能修改单个字符
>

```python
str1[0] = 'H'  # ❌ TypeError: 'str' object does not support item assignment
```

### 四、字符串拼接与重复
#### 1. 字符串相加（拼接）
```python
str1 = "hello "
str2 = "world"
print(str1 + str2)   # "hello world"

# 连续拼接多个字符串
result = "hello" + " " + "world"
```

#### 2. 字符串乘法（重复）
```python
str1 = "hello "
print(str1 * 3)      # "hello hello hello "

# 创建重复字符的字符串
print("-" * 30)      # "------------------------------"
print("=" * 20)      # "===================="
```

### 五、成员检查
```python
str1 = "hello world"
print("lo" in str1)      # True
print("hi" in str1)      # False
print("wo" not in str1)  # False

# 检查子字符串是否存在
if "world" in str1:
    print("包含 world")
```

### 六、原始字符串
原始字符串会**忽略转义字符**，按字面意思处理字符串中的每个字符。

```python
# 普通字符串：转义字符生效
print("hello\nworld")    # 输出两行（\n 被解释为换行）

# 原始字符串：转义字符失效
print(r"hello\nworld")   # 输出: hello\nworld

# 实际应用：文件路径、正则表达式
path = r"C:\Users\name\Desktop"     # 避免 \U、\n 被转义
regex = r"\d+\.\d+"                 # 正则表达式中的转义

# 大写 R 也可以
print(R"hello\tworld")
```

### 七、字符串常用函数详解
#### 1. 大小写转换
| 方法 | 说明 | 示例 |
| --- | --- | --- |
| `upper()` | 全部转为大写 | `"Hello".upper()` → `"HELLO"` |
| `lower()` | 全部转为小写 | `"Hello".lower()` → `"hello"` |
| `swapcase()` | 大小写互换 | `"Hello".swapcase()` → `"hELLO"` |
| `capitalize()` | 首字母大写，其余小写 | `"hello WORLD".capitalize()` → `"Hello world"` |
| `title()` | 每个单词首字母大写 | `"hello world".title()` → `"Hello World"` |
| `casefold()` | 更强的 lower（用于无大小写比较） | `"Straße".casefold()` → `"strasse"` |


```python
text = "hello WORLD"
print(text.upper())      # HELLO WORLD
print(text.lower())      # hello world
print(text.capitalize()) # Hello world
print(text.title())      # Hello World
```

#### 2. 查找与统计
| 方法 | 说明 | 返回值 |
| --- | --- | --- |
| `find(x[, start][, end])` | 查找子串位置 | 找到返回索引，找不到返回 `-1` |
| `rfind(x[, start][, end])` | 从右边查找 | 找到返回索引，找不到返回 `-1` |
| `index(x[, start][, end])` | 查找子串位置 | 找到返回索引，找不到**报错** |
| `rindex(x[, start][, end])` | 从右边查找 | 找到返回索引，找不到**报错** |
| `count(x[, start][, end])` | 统计子串出现次数 | 返回整数 |


```python
text = "hello world, hello python"

print(text.find("hello"))      # 0
print(text.rfind("hello"))     # 13
print(text.find("xyz"))        # -1（不存在）

print(text.index("world"))     # 6
# print(text.index("xyz"))     # ValueError: substring not found

print(text.count("hello"))     # 2
print(text.count("o"))         # 4

# 指定搜索范围
print(text.find("hello", 1))   # 13（从索引1开始查找）
print(text.find("hello", 1, 10))  # -1（在1~10范围内查找）
```

#### 3. 检查方法（返回布尔值）
| 方法 | 说明 |
| --- | --- |
| `startswith(x[, start][, end])` | 是否以指定前缀开头 |
| `endswith(x[, start][, end])` | 是否以指定后缀结尾 |
| `isspace()` | 是否只包含空白字符 |
| `isalnum()` | 是否只包含字母和数字 |
| `isalpha()` | 是否只包含字母 |
| `isdigit()` | 是否只包含数字 |
| `isdecimal()` | 是否只包含十进制数字 |
| `isnumeric()` | 是否只包含数值字符 |
| `isupper()` | 是否全部大写 |
| `islower()` | 是否全部小写 |
| `istitle()` | 是否符合标题格式 |
| `isascii()` | 是否只包含 ASCII 字符 |


```python
text = "hello123"
print(text.isalnum())    # True（字母+数字）
print(text.isalpha())    # False（包含数字）
print(text.isdigit())    # False

print("123".isdigit())   # True
print("3.14".isdigit())  # False（小数点不是数字）

print("Hello World".istitle())  # True（每个单词首字母大写）
print("HELLO".isupper())        # True
print("hello".islower())        # True

print("   ".isspace())          # True
print("hello\nworld".isprintable())  # False（包含换行符）
```

#### 4. 去除空白/字符
| 方法 | 说明 |
| --- | --- |
| `strip([x])` | 去除两端的空白字符或指定字符 |
| `lstrip([x])` | 去除左端的空白字符或指定字符 |
| `rstrip([x])` | 去除右端的空白字符或指定字符 |
| `removeprefix(prefix)` | 去除指定前缀（Python 3.9+） |
| `removesuffix(suffix)` | 去除指定后缀（Python 3.9+） |


```python
text = "  hello world  "
print(text.strip())           # "hello world"
print(text.lstrip())          # "hello world  "
print(text.rstrip())          # "  hello world"

# 去除指定字符
text2 = "***hello***"
print(text2.strip("*"))       # "hello"

# Python 3.9+ 新方法
url = "https://example.com"
print(url.removeprefix("https://"))  # "example.com"
print(url.removesuffix(".com"))      # "https://example"
```

#### 5. 分割与连接
| 方法 | 说明 |
| --- | --- |
| `split([sep][, maxsplit])` | 按分隔符分割，返回列表 |
| `rsplit([sep][, maxsplit])` | 从右边开始分割 |
| `splitlines([keepends])` | 按行分割，返回列表 |
| `join(iterable)` | 用字符串连接可迭代对象中的元素 |
| `partition(sep)` | 将字符串分割为三部分（头、分隔符、尾） |
| `rpartition(sep)` | 从右边开始分割为三部分 |


```python
# split() 分割
text = "apple,banana,orange"
print(text.split(","))           # ['apple', 'banana', 'orange']

text2 = "a b c d e"
print(text2.split())             # ['a', 'b', 'c', 'd', 'e']（默认按空白分割）
print(text2.split(" ", 2))       # ['a', 'b', 'c d e']（最多分割2次）

# rsplit() 从右边分割
print(text2.rsplit(" ", 2))      # ['a b c', 'd', 'e']

# splitlines() 按行分割
multi_line = "line1\nline2\r\nline3"
print(multi_line.splitlines())   # ['line1', 'line2', 'line3']

# join() 连接
words = ['hello', 'world']
print(" ".join(words))           # "hello world"
print(", ".join(words))          # "hello, world"
print("".join(words))            # "helloworld"

# partition() 分割为三部分
text = "name=value"
print(text.partition("="))       # ('name', '=', 'value')
print(text.partition("?"))       # ('name=value', '', '')（分隔符不存在）
```

#### 6. 替换

```python
text = "hello world, hello python"
```

#### 7. 格式化与对齐
| 方法 | 说明 |
| --- | --- |
| `center(width[, fillchar])` | 居中，用指定字符填充（默认空格） |
| `ljust(width[, fillchar])` | 左对齐 |
| `rjust(width[, fillchar])` | 右对齐 |
| `zfill(width)` | 右对齐，用 0 填充 |


```python
text = "hello"

print(text.center(11, "*"))   # "***hello***"
print(text.ljust(10, "-"))    # "hello-----"
print(text.rjust(10, "-"))    # "-----hello"
print(text.zfill(10))         # "00000hello"

# 实际应用：数字补零
print(str(42).zfill(5))       # "00042"
```

#### 8. 编码与解码
| 方法 | 说明 |
| --- | --- |
| `encode(encoding='utf-8', errors='strict')` | 将字符串编码为字节 |
| `decode(encoding='utf-8', errors='strict')` | 将字节解码为字符串（bytes 对象的方法） |


```python
text = "hello世界"
encoded = text.encode('utf-8')
print(encoded)   # b'hello\xe4\xb8\x96\xe7\x95\x8c'

decoded = encoded.decode('utf-8')
print(decoded)   # "hello世界"
```

#### 9. 翻译表
```python
# maketrans() 和 translate() 用于批量字符替换
text = "hello world"

# 创建翻译表：将 'h' 换成 'H'，'w' 换成 'W'
trans_table = str.maketrans({'h': 'H', 'w': 'W'})
print(text.translate(trans_table))   # "Hello World"

# 使用等长字符串创建翻译表
trans_table2 = str.maketrans("hw", "HW")
print(text.translate(trans_table2))  # "Hello World"

# 删除字符（第三个参数中的字符会被映射到 None）
trans_table3 = str.maketrans("", "", "l")
print(text.translate(trans_table3))  # "heo word"
```

### 九、字符串格式化（补充）
除了之前学习的格式化方式，字符串还有 `format()` 方法：

```python
# format() 方法
name = "Alice"
age = 25
print("{} is {} years old".format(name, age))      # 位置参数
print("{1} is {0} years old".format(name, age))    # 索引参数
print("{n} is {a} years old".format(n=name, a=age)) # 关键字参数

# 格式说明
print("{:.2f}".format(3.14159))    # "3.14"
print("{:>10}".format("hi"))       # "        hi"
print("{:*^20}".format("center"))  # "*******center*******"
```

### 十、字符串与其他类型的转换

### 十一、字符串拼接性能注意事项
```python
# ❌ 低效：在循环中使用 + 拼接字符串
result = ""
for i in range(1000):
    result += str(i)      # 每次创建新字符串，效率低

# ✅ 高效：使用 join()
result = "".join(str(i) for i in range(1000))
```

### 十二、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 字符串不可变 | `s[0] = 'a'` 会报错，需创建新字符串 |
| `find()` 和 `index()` 区别 | `find()` 返回 -1，`index()` 抛出异常 |
| `split()` 默认行为 | 按任意空白分割，会丢弃空字符串 |
| `split(' ')` 与 `split()` 区别 | `split(' ')` 会保留空字符串 |
| `strip()` 去除的不仅是空格 | 还去除 `\n`、`\t` 等空白字符 |
| 字符串比较是逐字符比较 | `"10" < "2"` 返回 `True`（'1' < '2'） |
| 原始字符串末尾不能有反斜杠 | `r"C:\"` 会报错 |
| 用 `+` 在循环中拼接字符串 | 效率低，推荐使用 `join()` |


### 十三、完整综合示例
```python
# 文本处理示例：清理和统计用户评论

def process_comment(comment):
    # 1. 去除首尾空白和标点符号
    cleaned = comment.strip().strip(".,!?;:")
    
    # 2. 转换为小写
    cleaned = cleaned.lower()
    
    # 3. 统计信息
    word_count = len(cleaned.split())
    char_count = len(cleaned)
    
    # 4. 检查是否为垃圾评论
    is_spam = "spam" in cleaned or "advertisement" in cleaned
    
    return {
        "original": comment,
        "cleaned": cleaned,
        "word_count": word_count,
        "char_count": char_count,
        "is_spam": is_spam
    }

# 测试
comments = [
    "  Hello World!  ",
    "This is a SPAM message!",
    "Python is awesome!!!",
]

for comment in comments:
    result = process_comment(comment)
    print(f"原文本: {result['original']}")
    print(f"清理后: {result['cleaned']}")
    print(f"单词数: {result['word_count']}, 字符数: {result['char_count']}")
    print(f"是否垃圾: {result['is_spam']}")
    print("-" * 30)
```

> 📌 **核心记忆**：
>
> + 字符串**不可变**，所有操作返回**新字符串**
> + 索引从 0 开始，支持切片 `[start:stop:step]`
> + 原始字符串 `r""` 让转义字符失效
> + 查找用 `find()`（安全），需要异常用 `index()`
> + 分割用 `split()`，连接用 `join()`
> + 循环拼接字符串用 `join()` 而非 `+`
> + 字符串格式化推荐 **f-string**
>
