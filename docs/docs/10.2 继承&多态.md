## Python 继承与多态学习笔记
### 一、单继承
**单继承**是指子类只继承一个父类。

#### 1.1 语法
```python
class 子类名(父类名):
    类体
```

#### 1.2 案例
```python
class Person:
    """人的类"""
    
    home = "earth"  # 类属性
    
    def __init__(self, name):
        self.name = name  # 实例属性
    
    def eat(self):
        print("eating...")

class YellowRace(Person):
    """黄种人"""
    color = "yellow"

class WhiteRace(Person):
    """白种人"""
    color = "white"

class BlackRace(Person):
    """黑种人"""
    color = "black"

# 创建黄种人实例
y1 = YellowRace("张三")
print(y1.home)   # earth（继承自 Person）
print(y1.color)  # yellow（YellowRace 自己的属性）
print(y1.name)   # 张三（继承自 Person）
y1.eat()         # eating...（继承自 Person）

# 创建白种人实例
w1 = WhiteRace("李四")
print(w1.color)  # white

# 创建黑种人实例
b1 = BlackRace("王五")
print(b1.color)  # black
```

### 二、多继承
**多继承**是指子类继承多个父类。

#### 2.1 语法
```python
class 子类名(父类1, 父类2, ...):
    类体
```

#### 2.2 方法查找顺序
调用方法时，先在子类中查找，若不存在则**从左到右**依次在父类中查找。

#### 2.3 案例
```python
class Person:
    home = "earth"
    
    def __init__(self, name):
        self.name = name
    
    def eat(self):
        print("eating...")

class YellowRace(Person):
    color = "yellow"
    
    def run(self):
        print("running...")

class Student(Person):
    def __init__(self, name, grade):
        self.name = name
        self.grade = grade
    
    def study(self):
        print("studying...")

# 多继承：先继承 Student，再继承 YellowRace
class ChineseStudent(Student, YellowRace):
    country = "中国"

# 创建实例
y1 = ChineseStudent("张三", "三年级")

# 可以访问所有父类的属性和方法
print(y1.home)      # earth（来自 Person）
print(y1.color)     # yellow（来自 YellowRace）
print(y1.country)   # 中国（自己的属性）
print(y1.name)      # 张三（来自 Student/Person）
print(y1.grade)     # 三年级（来自 Student）

y1.eat()            # eating...（来自 Person）
y1.run()            # running...（来自 YellowRace）
y1.study()          # studying...（来自 Student）
```

### 三、复用父类方法
子类中可以通过两种方式调用父类的方法：

| 方式 | 语法 | 说明 |
| --- | --- | --- |
| `super()` | `super().方法名()` | 推荐，自动处理 MRO |
| 父类名直接调用 | `父类名.方法名(self)` | 需要手动传入 self |


#### 3.1 使用 `super()`（推荐）
```python
class Person:
    def __init__(self, name):
        self.name = name
    
    def eat(self):
        print("eating...")

class Student(Person):
    def __init__(self, name, grade):
        super().__init__(name)  # 调用父类 __init__
        self.grade = grade
    
    def study(self):
        print("先吃再学")
        super().eat()           # 调用父类 eat 方法
        print("studying...")

class ChineseStudent(Student, YellowRace):
    country = "中国"

y1 = ChineseStudent("张三", "三年级")
y1.study()
# 输出：
# 先吃再学
# eating...
# studying...
```

#### 3.2 使用父类名直接调用
```python
class Student(Person):
    def __init__(self, name, grade):
        Person.__init__(self, name)  # 直接调用父类 __init__
        self.grade = grade
    
    def study(self):
        print("先吃再学")
        Person.eat(self)             # 直接调用父类 eat 方法
        print("studying...")
```

> 💡 **推荐使用 **`super()`：
>
> + 在单继承中效果相同
> + 在多继承中能正确处理 MRO
> + 代码更简洁，更易于维护
>

### 四、方法解析顺序（MRO）
**MRO（Method Resolution Order）** 决定了在多继承时方法的查找顺序。

#### 4.1 查看 MRO
使用 `类名.__mro__` 查看继承链：

```python
class Person:
    pass

class YellowRace(Person):
    pass

class Student(Person):
    pass

class ChineseStudent(Student, YellowRace):
    pass

print(ChineseStudent.__mro__)
# (<class '__main__.ChineseStudent'>, 
#  <class '__main__.Student'>, 
#  <class '__main__.YellowRace'>, 
#  <class '__main__.Person'>, 
#  <class 'object'>)
```

#### 4.2 MRO 查找规则
```plain
调用方法时，按 MRO 顺序查找：
1. 先在当前类中查找
2. 然后从左到右依次在父类中查找
3. 最后在 object 中查找
```

#### 4.3 菱形继承问题
```python
class A:
    def method(self):
        print("A.method")

class B(A):
    def method(self):
        print("B.method")
        super().method()

class C(A):
    def method(self):
        print("C.method")
        super().method()

class D(B, C):
    def method(self):
        print("D.method")
        super().method()

d = D()
d.method()
# 输出：
# D.method
# B.method
# C.method
# A.method

print(D.__mro__)
# D -> B -> C -> A -> object
```

### 五、方法重写（Override）
子类可以重写父类的方法，调用时使用子类自己的版本。

#### 5.1 基本重写
```python
class Person:
    def eat(self):
        print("eating...")

class Chinese(Person):
    # 重写父类 eat 方法
    def eat(self):
        print("用筷子吃")

y1 = Chinese()
y1.eat()  # 用筷子吃（调用子类重写的方法）
```

#### 5.2 重写 `__init__()` 方法
⚠️ **重要**：子类重写 `__init__()` 时，父类的 `__init__()` **不会自动执行**。

```python
class Person:
    def __init__(self, name):
        self.name = name
        print(f"Person.__init__: {name}")

class Chinese(Person):
    def __init__(self, name, area):
        # 必须手动调用父类的 __init__
        super().__init__(name)
        self.area = area
        print(f"Chinese.__init__: {name}, {area}")

y1 = Chinese("张三", "北京")
print(y1.name)  # 张三
print(y1.area)  # 北京
```

**如果忘记调用 **`super().__init__()`：

```python
class BadChild(Person):
    def __init__(self, name, extra):
        self.extra = extra
        # 没有调用 super().__init__(name)

# bad = BadChild("张三", "extra")
# print(bad.name)  # ❌ AttributeError: 'BadChild' object has no attribute 'name'
```

### 六、多态（Polymorphism）
**多态**是指同一事物在不同场景下呈现不同的状态。不同的对象调用同一个方法，产生不同的行为。

#### 6.1 多态示例
```python
class Animal:
    def go(self):
        pass

class Dog(Animal):
    def go(self):
        print("跑")

class Fish(Animal):
    def go(self):
        print("游")

class Bird(Animal):
    def go(self):
        print("飞")

# 多态函数：接收 Animal 类型，调用 go 方法
def make_go(animal):
    animal.go()  # 不同实例执行不同的 go 方法

# 使用
dog = Dog()
fish = Fish()
bird = Bird()

make_go(dog)   # 跑
make_go(fish)  # 游
make_go(bird)  # 飞
```

#### 6.2 多态的本质
| 概念 | 说明 |
| --- | --- |
| **同一个方法名** | 不同的子类有同名但行为不同的方法 |
| **同一个接口** | 调用者无需关心具体类型，只需调用统一的方法 |
| **运行时绑定** | 方法的具体执行在运行时根据对象类型决定 |


```python
# 多态的优势：可以统一处理不同类型的对象
animals = [Dog(), Fish(), Bird()]

for animal in animals:
    animal.go()
# 输出：
# 跑
# 游
# 飞
```

### 七、综合示例
```python
"""
宠物商店系统 - 演示继承与多态
"""

class Pet:
    """宠物基类"""
    
    total_pets = 0  # 类属性：宠物总数
    
    def __init__(self, name, price):
        self.name = name
        self.price = price
        Pet.total_pets += 1
    
    def speak(self):
        """发出声音（由子类重写）"""
        pass
    
    def info(self):
        """显示宠物信息"""
        print(f"名称: {self.name}, 价格: {self.price}元")


class Dog(Pet):
    """狗类"""
    
    def __init__(self, name, price, breed):
        super().__init__(name, price)
        self.breed = breed
    
    def speak(self):
        print(f"{self.name} 汪汪叫")
    
    def info(self):
        super().info()
        print(f"品种: {self.breed}")
        print("技能: 看家护院")


class Cat(Pet):
    """猫类"""
    
    def __init__(self, name, price, color):
        super().__init__(name, price)
        self.color = color
    
    def speak(self):
        print(f"{self.name} 喵喵叫")
    
    def info(self):
        super().info()
        print(f"毛色: {self.color}")
        print("技能: 捉老鼠")


class Bird(Pet):
    """鸟类"""
    
    def __init__(self, name, price, can_talk):
        super().__init__(name, price)
        self.can_talk = can_talk
    
    def speak(self):
        if self.can_talk:
            print(f"{self.name} 说: 你好！")
        else:
            print(f"{self.name} 叽叽喳喳")
    
    def info(self):
        super().info()
        print(f"会说话: {'是' if self.can_talk else '否'}")
        print("技能: 飞翔")


def adopt_pet(pet):
    """领养宠物（多态演示）"""
    print("\n" + "=" * 30)
    print("领养信息:")
    pet.info()
    print("声音测试:")
    pet.speak()


# 使用
dog = Dog("旺财", 500, "金毛")
cat = Cat("咪咪", 300, "橘色")
bird = Bird("小八", 1000, True)

# 展示多态
adopt_pet(dog)
adopt_pet(cat)
adopt_pet(bird)

# 遍历处理
print("\n" + "=" * 30)
print("所有宠物声音:")
pets = [dog, cat, bird]
for pet in pets:
    print(f"{pet.name}: ", end="")
    pet.speak()

print(f"\n宠物总数: {Pet.total_pets}")
```

### 八、易错点总结
| 易错点 | 正确理解 |
| --- | --- |
| 子类重写 `__init__` 后父类 `__init__` 自动执行 | ❌ 需要手动调用 `super().__init__()` |
| 多继承时方法从左到右查找 | ✅ 按 MRO 顺序，不一定是简单的从左到右 |
| 子类能继承父类私有成员 | ❌ 私有成员（`__var`）不能继承 |
| 子类不调用 `super()` 会丢失父类属性 | ✅ 务必在 `__init__` 中调用 `super().__init__()` |
| `super()` 只适用于单继承 | ❌ `super()` 在多继承中更强大 |


### 九、快速参考卡片
```python
# === 单继承 ===
class Child(Parent):
    pass

# === 多继承 ===
class Child(Parent1, Parent2):
    pass

# === 调用父类方法（推荐 super()）===
class Child(Parent):
    def __init__(self):
        super().__init__()          # 调用父类 __init__
    
    def method(self):
        super().method()            # 调用父类方法

# === 查看 MRO ===
print(Child.__mro__)

# === 方法重写 ===
class Child(Parent):
    def method(self):               # 重写父类方法
        super().method()            # 可选：调用父类方法
        print("Child method")

# === 多态 ===
def process(obj):
    obj.method()                    # 不同对象不同行为
```

> 📌 **核心记忆**：
>
> + **单继承**：子类继承一个父类，用 `class 子类(父类):`
> + **多继承**：子类继承多个父类，用 `class 子类(父1, 父2):`
> + `super()`：在子类中调用父类方法，推荐使用
> + **MRO**：方法解析顺序，决定多继承时方法的查找顺序
> + **重写**：子类重新定义父类的方法，覆盖原有行为
> + **多态**：不同对象调用同一方法，产生不同行为
>
