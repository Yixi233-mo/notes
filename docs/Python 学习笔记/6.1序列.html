Python 序列（Sequence）学习笔记 ​

一、什么是序列 ​

序列（Sequence） 是一种基本且核心的数据结构，它允许我们以有序的方式存储和操作数据。

特性说明
有序性元素按照一定顺序排列，每个元素都有固定位置
索引访问支持通过索引（下标）访问元素
可迭代可以使用 for 循环遍历
可包含不同类型序列可以包含不同类型的元素

常见序列类型 ​

序列类型是否可变示例
列表（List）✅ 可变[1, 2, 'a', True]
元组（Tuple）❌ 不可变(1, 2, 'a', True)
字符串（String）❌ 不可变"hello"

二、序列的通用操作 ​

序列类型（列表、元组、字符串）都支持以下通用操作：

1. 索引（Indexing） ​

通过索引访问序列中的单个元素，索引从 0 开始。

python
sequence = [10, 20, 30, 40, 50]
print(sequence[0])   # 10（第一个元素）
print(sequence[2])   # 30（第三个元素）
print(sequence[-1])  # 50（最后一个元素，负索引）
print(sequence[-2])  # 40（倒数第二个元素）

2. 切片（Slicing） ​

从序列中提取一个子序列（子集），返回新的序列。

语法：sequence[start:stop:step]

参数说明是否包含
start起始索引✅ 包含
stop结束索引❌ 不包含
step步长（默认1）—
python
seq = [10, 20, 30, 40, 50, 60, 70]

print(seq[1:4])      # [20, 30, 40]（索引1到3）
print(seq[:3])       # [10, 20, 30]（从开头到索引2）
print(seq[3:])       # [40, 50, 60, 70]（从索引3到结尾）
print(seq[:])        # [10, 20, 30, 40, 50, 60, 70]（整个序列的副本）
print(seq[::2])      # [10, 30, 50, 70]（步长为2）
print(seq[::-1])     # [70, 60, 50, 40, 30, 20, 10]（反转序列）

3. 序列相加（Concatenation） ​

使用 + 运算符将两个相同类型的序列连接成一个新序列。

python
# 列表相加
list1 = [1, 2, 3]
list2 = [4, 5, 6]
print(list1 + list2)   # [1, 2, 3, 4, 5, 6]

# 元组相加
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
print(tuple1 + tuple2)  # (1, 2, 3, 4, 5, 6)

# 字符串相加
str1 = "Hello"
str2 = "World"
print(str1 + str2)      # "HelloWorld"

# ⚠️ 不同类型不能相加
# print([1, 2] + (3, 4))  # TypeError

4. 序列乘法（Repetition） ​

使用 * 运算符将序列重复指定次数。

python
# 列表乘法
print([1, 2] * 3)    # [1, 2, 1, 2, 1, 2]

# 元组乘法
print((1, 2) * 3)    # (1, 2, 1, 2, 1, 2)

# 字符串乘法
print("Hi" * 3)      # "HiHiHi"

# 初始化固定长度的列表
zeros = [0] * 5      # [0, 0, 0, 0, 0]

5. 成员检查（Membership） ​

使用 in / not in 判断元素是否存在于序列中。

python
# 列表成员检查
fruits = ['apple', 'banana', 'orange']
print('apple' in fruits)      # True
print('grape' not in fruits)  # True

# 字符串成员检查
text = "hello world"
print('hello' in text)        # True
print('hi' in text)           # False

# 元组成员检查
numbers = (1, 2, 3, 4, 5)
print(3 in numbers)           # True

7. 最大值与最小值 ​

使用 max() 和 min() 获取序列中的最大值和最小值。

python
# 数值序列
numbers = [3, 1, 4, 1, 5, 9, 2]
print(max(numbers))    # 9
print(min(numbers))    # 1

# 字符串序列（按字典序/ASCII码比较）
words = ['apple', 'banana', 'cherry']
print(max(words))      # 'cherry'
print(min(words))      # 'apple'

# 字符串（按字符ASCII码）
text = "hello"
print(max(text))       # 'o'（ASCII码最大）
print(min(text))       # 'e'（ASCII码最小）

三、序列操作速查表 ​

操作语法说明示例
索引seq[i]获取第 i 个元素[10,20,30][1] → 20
切片seq[i:j]获取子序列[10,20,30][:2] → [10,20]
相加seq1 + seq2连接两个序列[1,2]+[3,4] → [1,2,3,4]
乘法seq * n重复 n 次[1,2]*2 → [1,2,1,2]
成员检查x in seq判断是否存在3 in [1,2,3] → True
长度len(seq)元素个数len("hi") → 2
最大值max(seq)最大元素max([3,1,4]) → 4
最小值min(seq)最小元素min([3,1,4]) → 1

四、序列 vs 其他数据结构 ​

特性序列集合（Set）字典（Dict）
有序性✅ 有序❌ 无序Python 3.7+ 有序
索引访问✅ 支持❌ 不支持通过键访问
重复元素✅ 允许❌ 不允许键不允许重复
元素类型任意类型可哈希类型键需可哈希

五、易错点总结 ​

易错点正确理解
切片 [i:j] 包含 j❌ 不包含，[i:j] 取索引 i 到 j-1
负索引从 0 开始❌ 负索引从 -1 开始
不同序列类型可相加❌ 必须相同类型才能相加
seq * n 修改副本影响原序列❌ 乘法返回新序列，原序列不变
max()/min() 用于不同类型❌ 元素必须可比较

六、完整示例代码 ​

python
# 演示序列的各种操作

def demo_sequence_operations():
    # 创建一个列表作为示例序列
    seq = [10, 20, 30, 40, 50, 60, 70]
    
    print("=== 序列操作演示 ===")
    print(f"原始序列: {seq}")
    
    # 1. 索引
    print(f"\n1. 索引:")
    print(f"  seq[0] = {seq[0]}")
    print(f"  seq[-1] = {seq[-1]}")
    
    # 2. 切片
    print(f"\n2. 切片:")
    print(f"  seq[1:4] = {seq[1:4]}")
    print(f"  seq[:3] = {seq[:3]}")
    print(f"  seq[3:] = {seq[3:]}")
    print(f"  seq[::2] = {seq[::2]}")
    print(f"  seq[::-1] = {seq[::-1]}")
    
    # 3. 相加
    print(f"\n3. 相加:")
    print(f"  [1,2,3] + [4,5,6] = {[1,2,3] + [4,5,6]}")
    
    # 4. 乘法
    print(f"\n4. 乘法:")
    print(f"  [1,2] * 3 = {[1,2] * 3}")
    
    # 5. 成员检查
    print(f"\n5. 成员检查:")
    print(f"  30 in seq = {30 in seq}")
    print(f"  100 in seq = {100 in seq}")
    
    # 6. 长度
    print(f"\n6. 长度:")
    print(f"  len(seq) = {len(seq)}")
    
    # 7. 最大值/最小值
    print(f"\n7. 最大值/最小值:")
    print(f"  max(seq) = {max(seq)}")
    print(f"  min(seq) = {min(seq)}")

if __name__ == "__main__":
    demo_sequence_operations()
