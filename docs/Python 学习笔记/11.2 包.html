python

二、包（Package）概述 ​

包是一种管理 Python 模块命名空间的方式，通过 包名.模块名 来组织模块。

概念说明
包包含 __init__.py 文件的目录
子包包内的子目录，也包含 __init__.py
模块包内的 .py 文件

💡 包 vs 文件夹：包是一个特殊的文件夹，必须包含 __init__.py 文件（可以是空文件）。

包的目录结构示例 ​

plain

              wavwrite.py
              aiffread.py
              aiffwrite.py
              ...
      effects/                  # 用于音效的子包
              __init__.py
              echo.py
              surround.py
              reverse.py
              ...
      filters/                  # 用于过滤器的子包
              __init__.py
              equalizer.py
              vocoder.py

python
"""circle.py - 圆形计算模块"""

radius = 10
PI = 3.1415926

def area(radius):
    """计算圆面积"""
    return PI * radius * radius

def perimeter(radius):
    """计算圆周长"""
    return 2 * PI * radius

4.4 从包中导入模块 from 包名 import * ​

使用 from 包名 import * 时，Python 不会自动导入所有子模块。需要在 __init__.py 中定义 __all__ 指定要导入的模块。

在 graphic/__init__.py 中：

python
# graphic/__init__.py
__all__ = ["circle"]   # 只允许导入 circle 模块
1
2

在 main.py 中使用：

python
from graphic import *

print(circle.area(10))   # 314.15926（可以访问）
# print(rectangle.area(10))  # ❌ NameError: name 'rectangle' is not defined
1
2
3
4

⚠️ 注意：如果 __all__ 中不指定，from ... import * 无法导入任何模块。

4.5 包内模块间导入（__init__.py） ​

在包内的 __init__.py 中导入子模块时，建议使用相对路径：

python
# graphic/__init__.py
from . import circle   # ✅ 推荐：相对导入
# import circle        # ❌ 不推荐：绝对导入（会在 sys.path 中查找）

九、包的导入方式对比 ​

导入方式语法访问方式适用场景
全局导入import 包.模块包.模块.成员避免命名冲突
模块导入from 包 import 模块模块.成员简化路径
成员导入from 包.模块 import 成员成员只使用少量成员
* 导入from 包 import *模块.成员需要 __all__ 控制

十、易错点总结 ​

易错点正确理解
包只是普通文件夹❌ 包必须包含 __init__.py 文件
from 包 import * 自动导入所有子模块❌ 需在 __init__.py 中定义 __all__
__init__.py 只能为空✅ 可以执行初始化代码、定义 __all__
包内绝对导入比相对导入好✅ 包内推荐使用相对导入 from . import 模块
import 包.模块.函数 可以直接使用❌ import 最后一项不能是函数/类/变量

十一、快速参考卡片 ​

python
# === 创建包 ===
# 目录结构：
# my_package/
#     __init__.py
#     module1.py
#     module2.py

# === 导入方式 ===

# 1. 全局导入
import my_package.module1
my_package.module1.function()

# 2. 模块导入
from my_package import module1
module1.function()

# 3. 成员导入
from my_package.module1 import function
function()

# 4. 带别名
import my_package.module1 as m1
m1.function()

# === __init__.py 控制 ===
# 控制 from ... import *
__all__ = ["module1", "module2"]

# 便捷导入
from .module1 import function as f1
from .module2 import function as f2

# === 打包自己的库 ===
# setup.py
from distutils.core import setup
setup(
    name="package_name",
