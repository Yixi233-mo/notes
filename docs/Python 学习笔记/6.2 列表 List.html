一、列表概述 ​

列表（List） 是 Python 中最常用的序列类型，具有以下特点：

特性说明
可变性✅ 可修改（增、删、改）
有序性✅ 元素有固定顺序
索引支持✅ 支持正向和负向索引
元素类型✅ 可包含不同类型的元素
重复元素✅ 允许重复

定义方式：使用 [] 定义，元素之间用 , 分隔

python
list1 = [100, 200, 300, 400, 500]           # 纯数字列表
list2 = [1, "hello", 3.14, True]            # 混合类型列表
list3 = []                                   # 空列表

十、列表排序与反转 ​

方法说明是否修改原列表
sorted(list)返回排序后的新列表❌ 不修改
list.sort()就地排序✅ 修改
list.reverse()反转列表✅ 修改
list[::-1]返回反转的新列表❌ 不修改
python
list1 = [3, 1, 4, 1, 5, 9, 2]

# sorted() 返回新列表
new_list = sorted(list1)
print(new_list)      # [1, 1, 2, 3, 4, 5, 9]
print(list1)         # [3, 1, 4, 1, 5, 9, 2]（原列表不变）

# sort() 就地排序
list1.sort()
print(list1)         # [1, 1, 2, 3, 4, 5, 9]

# 降序排序
list1.sort(reverse=True)
print(list1)         # [9, 5, 4, 3, 2, 1, 1]

# 反转
list1.reverse()
print(list1)         # [1, 1, 2, 3, 4, 5, 9]

列表推导式是 Python 中创建列表的简洁优雅的方式。

for x in range(5):
    squares.append(x**2)

# 列表推导式

十三、zip() 函数 ​

zip() 将多个可迭代对象的对应元素打包成元组。

python
list1 = [1, 2, 3, 4, 5]
list2 = ["a", "b", "c", "d", "e"]

zipped = zip(list1, list2)
print(list(zipped))  # [(1,'a'), (2,'b'), (3,'c'), (4,'d'), (5,'e')]

# 遍历配对元素
for num, letter in zip(list1, list2):
    print(f"{num} -> {letter}")

# 长度不一致时，以最短的为准
list3 = [1, 2, 3]
list4 = ['a', 'b', 'c', 'd', 'e']
print(list(zip(list3, list4)))  # [(1,'a'), (2,'b'), (3,'c')]

十四、列表的复制（深拷贝 vs 浅拷贝） ​

python
# 浅拷贝方式
original = [1, 2, [3, 4]]

copy1 = original[:]           # 切片复制
copy2 = original.copy()       # copy()方法
copy3 = list(original)        # 构造函数

# 修改外层元素
copy1[0] = 100
print(original[0])   # 1（不受影响）

# 修改内层列表（重要！）
copy1[2][0] = 999
print(original[2])   # [999, 4]（受影响！）

# 深拷贝（创建完全独立的副本）
import copy
deep_copy = copy.deepcopy(original)
deep_copy[2][0] = 777
print(original[2])   # [999, 4]（不受影响）

十六、易错点总结 ​

易错点正确理解
append() 一次添加多个元素用 extend() 或 +
list = list + [x] 不会修改原列表返回新列表，需重新赋值
切片 [i:j] 修改时长度可以不同切片替换会调整列表长度
列表乘法 [[]]*n 创建的是引用内层列表指向同一对象
sort() 和 sorted() 混淆sort() 原地修改，sorted() 返回新列表
浅拷贝内层可变对象用 copy.deepcopy() 创建完全独立副本

十七、完整综合示例 ​

python
# 学生成绩管理系统示例

# 创建学生列表
students = ["张三", "李四", "王五", "赵六"]
scores = [85, 92, 78, 88]

# 1. 添加新学生
students.append("小明")
scores.append(95)

# 2. 修改李四的成绩
index = students.index("李四")
scores[index] = 96

# 3. 删除王五
index = students.index("王五")
del students[index]
del scores[index]

# 4. 计算统计信息
print(f"学生列表: {students}")
print(f"成绩列表: {scores}")
print(f"平均分: {sum(scores)/len(scores):.2f}")
print(f"最高分: {max(scores)}")
print(f"最低分: {min(scores)}")

# 5. 使用 enumerate 遍历
print("\n成绩单:")
for i, (name, score) in enumerate(zip(students, scores)):
    print(f"{i+1}. {name}: {score}分")

# 6. 列表推导式：筛选及格学生
passing = [(name, score) for name, score in zip(students, scores) if score >= 90]
print(f"\n优秀学生: {passing}")
