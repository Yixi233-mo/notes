Python 模块（Module）学习笔记 ​

一、模块概述 ​

模块（Module） 是一个以 .py 结尾的 Python 源文件，其中可以包含变量、函数、类等。

优点说明
提高可维护性将特定功能的代码集中在一个文件中
提高代码复用性编写一次，多处导入使用
避免命名冲突相同名字的变量/函数可以存在于不同模块中
便于组织代码按功能划分，结构清晰

💡 模块的本质：模块就是包含 Python 代码的文件，文件名即模块名。

二、创建模块 ​

命名规则：

  • 模块名区分大小写
  • 不能与 Python 自带的标准模块重名
  • 遵循标识符命名规范（通常使用小写字母和下划线）

创建 my_add.py 模块：

python
# my_add.py
"""
这是一个简单的加法模块
"""

num = 100

def add(a, b):
    """求两个数的和"""
    return a + b

三、导入模块 ​

3.1 全部导入 import ​

导入模块的所有成员，通过 模块名.成员名 的方式访问。

语法：

python
import 模块名 [as 别名]
1

案例：

python
# main.py
# 导入模块
import my_add

# 使用模块中的成员
print(my_add.add(1, 2))   # 3
print(my_add.num)         # 100

# 使用别名
import my_add as a1
print(a1.add(1, 2))       # 3
print(a1.num)             # 100

6.2 解决方案：使用 if __name__ == "__main__" ​

python

def add(a, b):
    return a + b

print(__name__)   # 直接运行时为 "__main__"，导入时为 "my_add"

if __name__ == "__main__":
    # 只有在直接运行此文件时才会执行
    print(add(10, 20))

七、模块的综合示例 ​

创建 calculator.py 模块：

python
# calculator.py
"""
计算器模块
提供基本的数学运算功能
"""

__all__ = ["add", "subtract", "multiply", "divide"]

# 常量
PI = 3.14159
VERSION = "1.0.0"
_DEBUG = False          # 私有常量，不会被 import * 导入

def add(a, b):
    """加法"""
    return a + b

def subtract(a, b):
    """减法"""
    return a - b

def multiply(a, b):
    """乘法"""
    return a * b

def divide(a, b):
    """除法"""
    if b == 0:
        raise ValueError("除数不能为0")
    return a / b

def _internal_validate(x):
    """私有方法：验证输入（不会被 import * 导入）"""
    return isinstance(x, (int, float))

# 测试代码
if __name__ == "__main__":
    print("计算器模块测试：")
    print(f"3 + 5 = {add(3, 5)}")
    print(f"10 - 4 = {subtract(10, 4)}")
    print(f"6 * 7 = {multiply(6, 7)}")
    print(f"15 / 3 = {divide(15, 3)}")
    print(f"PI = {PI}")

八、易错点总结 ​

易错点正确理解
模块名与变量/函数同名✅ 导入时使用别名可避免冲突
from ... import * 导入所有成员⚠️ 不导入以下划线开头的成员
模块导入后重新加载❌ 模块只会被导入一次，需用 importlib.reload()
__all__ 限制所有导入方式❌ 仅对 from ... import * 有效
忘记 __name__ == "__main__"❌ 测试代码会在模块被导入时执行
模块搜索路径问题使用 sys.path.append() 临时添加

九、快速参考卡片 ​

python
# === 导入方式 ===

# 1. 全部导入
import 模块名
import 模块名 as 别名
模块名.成员名

# 2. 局部导入
from 模块名 import 成员名
from 模块名 import 成员名 as 别名
from 模块名 import 成员名1, 成员名2

# 3. 全部导入（不推荐）
from 模块名 import *

# === 模块控制 ===

# 控制 from ... import * 的行为
__all__ = ["成员1", "成员2"]   # 只导入指定的公共成员

# 控制测试代码
if __name__ == "__main__":
    # 只在直接运行时执行

# === 搜索路径 ===
import sys
print(sys.path)                      # 查看搜索路径
sys.path.append("/自定义路径")        # 临时添加

# === 模块信息 ===
模块.__doc__          # 模块文档字符串
