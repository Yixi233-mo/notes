Python 集合（Set）学习笔记 ​

一、集合概述 ​

集合（Set） 是 Python 中一种重要的无序集合数据类型，具有以下特点：

特性说明
无序性❌ 元素没有固定顺序，不能通过索引访问
唯一性✅ 自动去重，元素不重复
可变性✅ 可以添加和删除元素
元素类型✅ 可包含不同类型的元素，但元素必须可哈希
集合运算✅ 支持数学上的并集、交集、差集等操作

核心特点：

  • 无序：元素存储顺序不确定，不能使用索引或切片
  • 不重复：自动去重，每个元素只出现一次
  • 可哈希：集合中的元素必须是可哈希的（不可变类型）

2. 删除元素 ​

方法说明元素不存在时
remove(x)移除指定元素❌ 抛出 KeyError
discard(x)移除指定元素✅ 不做任何事
pop()随机移除并返回一个元素❌ 集合为空时抛出 KeyError

python
set1 = {1, 2, 3, 4, 5}

print(set1)  # {1, 2, 4, 5}
# set1.remove(10)  # ❌ KeyError: 10

print(set1)  # {1, 2, 4, 5}

# pop()：随机移除并返回一个元素
removed = set1.pop()
print(f"移除的元素: {removed}")  # 随机（不确定）
print(set1)

# clear()：清空集合
set1.clear()
print(set1)  # set()

5. 集合关系判断 ​

方法说明示例
isdisjoint()判断是否不相交（无共同元素）{1,2}.isdisjoint({3,4}) → True
issubset()判断是否为子集{1,2}.issubset({1,2,3}) → True
issuperset()判断是否为超集{1,2,3}.issuperset({1,2}) → True
python
set1 = {1, 2}
set2 = {1, 2, 3, 4}
set3 = {5, 6}

# 子集判断
print(set1.issubset(set2))   # True
print(set1 <= set2)          # True（运算符形式）

# 真子集判断
print(set1 < set2)           # True（不等于时）

# 超集判断
print(set2.issuperset(set1)) # True
print(set2 >= set1)          # True（运算符形式）

# 不相交判断
print(set1.isdisjoint(set3)) # True（没有共同元素）
print(set1.isdisjoint(set2)) # False（有共同元素）

五、集合的更新操作 ​

方法说明等价运算符
update()并集更新`
intersection_update()交集更新&=

python
set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

# 差集更新
set1 = {1, 2, 3, 4}
set1 -= set2  # 等价于 set1.difference_update(set2)

七、frozenset（不可变集合） ​

frozenset 是集合的不可变版本，创建后不能修改。可哈希，可以作为字典的键或集合的元素。

python
# 创建 frozenset
fs1 = frozenset([1, 2, 3, 4, 5])
fs2 = frozenset({3, 4, 5, 6, 7})

print(fs1)  # frozenset({1, 2, 3, 4, 5})

# 支持集合运算（返回 frozenset）
print(fs1 | fs2)   # frozenset({1, 2, 3, 4, 5, 6, 7})
print(fs1 & fs2)   # frozenset({3, 4, 5})
print(fs1 - fs2)   # frozenset({1, 2})

# 不支持修改操作
# fs1.add(6)  # ❌ AttributeError: 'frozenset' object has no attribute 'add'

# 可以作为字典键
d = {frozenset([1, 2]): "value"}
print(d)  # {frozenset({1, 2}): 'value'}

# 可以作为集合的元素
s = {frozenset([1, 2]), frozenset([3, 4])}
print(s)  # {frozenset({1, 2}), frozenset({3, 4})}

十、完整综合示例 ​

python
# 电影推荐系统（基于集合运算）

class MovieRecommender:
    def __init__(self):
        self.user_history = {}  # 用户观影历史
        self.movie_categories = {}  # 电影分类
    
    def add_user_history(self, user, movies):
        """添加用户观影历史"""
        self.user_history[user] = set(movies)
    
    def add_movie_category(self, movie, category):
        """添加电影分类"""
        if category not in self.movie_categories:
            self.movie_categories[category] = set()
        self.movie_categories[category].add(movie)
    
    def get_recommendations(self, user, category=None):
        """获取推荐电影"""
        if user not in self.user_history:
            return set()
        
        watched = self.user_history[user]
        
        if category:
            # 推荐指定分类中未观看的电影
            candidates = self.movie_categories.get(category, set())
        else:
            # 推荐所有未观看的电影
            candidates = set()
            for movies in self.movie_categories.values():
                candidates.update(movies)
        
        return candidates - watched
    
    def get_similar_users(self, user):
        """找到相似用户"""
        if user not in self.user_history:
            return set()
        
        user_movies = self.user_history[user]
        similarity = {}
        
        for other_user, movies in self.user_history.items():
            if other_user == user:
                continue
            # 计算共同观看电影数量
            common = len(user_movies & movies)
            if common > 0:
                similarity[other_user] = common
        
        # 返回共同观影数最多的前3个用户
        return sorted(similarity.items(), key=lambda x: x[1], reverse=True)[:3]

# 使用示例
recommender = MovieRecommender()

# 添加用户观影历史
recommender.add_user_history("张三", ["泰坦尼克号", "阿凡达", "盗梦空间"])
recommender.add_user_history("李四", ["阿凡达", "盗梦空间", "星际穿越"])
recommender.add_user_history("王五", ["泰坦尼克号", "星际穿越", "教父"])

# 添加电影分类
recommender.add_movie_category("泰坦尼克号", "爱情")
recommender.add_movie_category("阿凡达", "科幻")
recommender.add_movie_category("盗梦空间", "科幻")
recommender.add_movie_category("星际穿越", "科幻")
recommender.add_movie_category("教父", "剧情")

# 获取推荐
print("=== 电影推荐 ===")
print(f"为张三推荐科幻电影: {recommender.get_recommendations('张三', '科幻')}")
print(f"为张三推荐所有未看电影: {recommender.get_recommendations('张三')}")

# 找到相似用户
print("\n=== 相似用户 ===")
similar = recommender.get_similar_users("张三")
for user, common_count in similar:
    print(f"与张三共同观影 {common_count} 部的用户: {user}")

十一、易错点总结 ​

易错点正确理解
{} 创建空集合❌ {} 是空字典，空集合用 set()
集合有序❌ 集合无序，不能通过索引访问
集合元素可重复❌ 自动去重，元素唯一
任何类型都能放入集合❌ 元素必须是可哈希的（不可变类型）
remove() 和 discard() 无区别remove() 不存在抛异常，discard() 不抛异常

十二、性能对比：集合 vs 列表 ​

python
import timeit
import random

# 创建测试数据
size = 10000
list_data = list(range(size))
set_data = set(range(size))

# 成员检查性能测试
def test_membership(data_structure, test_value):
    return test_value in data_structure

# 测试存在元素（最好情况）
list_time = timeit.timeit(lambda: test_membership(list_data, size-1), number=1000)
set_time = timeit.timeit(lambda: test_membership(set_data, size-1), number=1000)
print(f"列表成员检查（存在）: {list_time:.6f} 秒")
print(f"集合成员检查（存在）: {set_time:.6f} 秒")

# 测试不存在元素（最坏情况）
list_time = timeit.timeit(lambda: test_membership(list_data, -1), number=1000)
set_time = timeit.timeit(lambda: test_membership(set_data, -1), number=1000)
print(f"列表成员检查（不存在）: {list_time:.6f} 秒")
print(f"集合成员检查（不存在）: {set_time:.6f} 秒")

# 去重性能测试
def dedup_with_set(data):
    return list(set(data))

def dedup_with_loop(data):
    result = []
    for x in data:
        if x not in result:
            result.append(x)
    return result

data = [random.randint(0, 1000) for _ in range(10000)]
set_time = timeit.timeit(lambda: dedup_with_set(data), number=100)
loop_time = timeit.timeit(lambda: dedup_with_loop(data), number=100)
print(f"集合去重: {set_time:.6f} 秒")
print(f"循环去重: {loop_time:.6f} 秒")

📌 核心记忆：

  • 集合用 {} 定义，**空集合用 **set()
  • 集合无序、元素唯一
  • 不能通过索引访问，只能遍历
  • 元素必须是可哈希的（不可变类型）
  • 支持强大的数学集合运算：|（并）、&（交）、-（差）、^（对称差）
  • 成员检查速度极快（O(1)），适合快速查找和去重
  • frozenset 是不可变版本，可作为字典键或集合元素
