一、基本概念 ​

术语说明方向
编码 (encode)字符串 → 字节数据str → bytes
解码 (decode)字节数据 → 字符串bytes → str
python
str1 = '你好中国'        # 字符串类型 (str)
byte1 = str1.encode()    # 编码：str → bytes
str2 = byte1.decode()    # 解码：bytes → str

三、UTF-8 编码规则（重点） ​

字节数计算 ​

python
byte1 = str1.encode('utf-8')
print(len(byte1))   # 12个字节
1
2
字符UTF-8 字节数说明
英文字母/数字/常见符号1 字节ASCII 兼容
希腊字母、西里尔字母等2 字节扩展拉丁文
中文、日文、韩文（CJK）3 字节常用汉字
生僻字、Emoji4 字节辅助平面字符

示例分析：

  • '你好中国' 共 4 个中文字符
  • 每个中文 → 3 字节（UTF-8）
  • 总计：4 × 3 = 12 字节

四、常用编码格式对比 ​

编码特点中文占用适用场景
UTF-8变长编码，兼容 ASCII3 字节国际通用，首选
GBK/GB2312中文字符集，定长2 字节纯中文环境
ASCII仅英文、数字、符号不支持纯英文
UTF-16变长编码2 或 4 字节Windows 内部
ISO-8859-1西欧语言不支持欧洲语言
python
# 不同编码对比示例
text = '你好'

print(len(text.encode('utf-8')))   # 6 字节 (3×2)
print(len(text.encode('gbk')))     # 4 字节 (2×2)

五、编码/解码常见错误与处理 ​

1. 编码错误（UnicodeEncodeError） ​

python
# 字符串包含当前编码无法表示的字符
text = '你好'
text.encode('ascii')   # ❌ UnicodeEncodeError: 'ascii' codec can't encode...
1
2
3

解决方法：

python
text.encode('ascii', errors='ignore')   # 忽略无法编码的字符
text.encode('ascii', errors='replace')  # 用 ? 替换
text.encode('ascii', errors='xmlcharrefreplace')  # 转为 XML 实体

七、实际应用场景 ​

场景说明
网络传输socket 发送数据需先 encode 为 bytes
文件读写二进制模式（图片、音视频）用 bytes
加密/解密加密函数通常输入 bytes
JSON 交互API 传输需指定编码
数据库存储确认数据库编码设置
python
# 文件写入示例
with open('test.txt', 'wb') as f:    # 二进制写入
    f.write('你好'.encode('utf-8'))

with open('test.txt', 'rb') as f:    # 二进制读取
    data = f.read()
    print(data.decode('utf-8'))      # 你好

八、易错点总结 ​

易错点正确做法
编码和解码使用不同编码必须使用相同编码
忘记指定 encoding 参数建议显式指定：.encode('utf-8')
混淆 str 和 bytestype() 检查类型
网络传输直接发 str需先 .encode() 为 bytes
打印 bytes 直接看内容bytes 以十六进制显示，需 decode 才能看原文
