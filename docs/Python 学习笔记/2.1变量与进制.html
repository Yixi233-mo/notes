📌 五、进制转换 ​

5.1 进制的表示方法 ​

进制前缀示例十进制值
二进制0b0b102
八进制0o0o108
十进制无1010
十六进制0x0xA10

5.2 进制转换函数 ​

函数功能示例
bin()转二进制bin(506) → '0b111111010'
oct()转八进制oct(1356) → '0o2514'
hex()转十六进制hex(1356) → '0x54c'
int()转十进制int('1010',2) → 10

5.3 实战示例 ​

python
# 二进制 → 十进制
print(0b11011)    # 输出: 27

# 十进制 → 二进制
print(bin(506))   # 输出: 0b111111010

# 十六进制 → 十进制
print(0x3E4)      # 输出: 996

# 十进制 → 十六进制
print(hex(1356))  # 输出: 0x54c

# 十进制 → 八进制
print(oct(11111111))  # 输出: 0o52330607

# 十六进制 → 二进制
print(bin(0xBBB))     # 输出: 0b101110111011

# 八进制 → 二进制
print(bin(0o777))     # 输出: 0b111111111
