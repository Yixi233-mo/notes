一、输入函数 input() ​

基本语法 ​

python
变量 = input("提示信息")
1

核心特性 ​

特性说明
返回值类型永远是字符串 (str)
执行行为程序暂停，等待用户输入，按回车结束
提示信息可选参数，会在输入前显示

示例代码 ​

python
input_str = input("请输入：")
print("input_str数据类型为:", type(input_str))  # <class 'str'>
print(input_str)  # 输出用户输入的内容

⚠️ 重要注意事项 ​

python
age = input("请输入年龄：")  # 用户输入 18
print(age + 1)  # ❌ TypeError! age 是字符串，不能与整数相加

# 正确做法：类型转换
age = int(input("请输入年龄："))  # 转换为整数
print(age + 1)  # ✅ 输出 19

方式四：四种方式对比总结 ​

方式语法示例优点缺点推荐度
% 格式化"%d" % num简洁，C语言习惯类型不匹配易出错⭐⭐
format()"{}".format(num)功能强大，灵活性高写法稍长⭐⭐⭐
f-stringf"{num}"简洁直观，性能好Python 3.6+ 可用⭐⭐⭐⭐⭐
python
# 四种方式对比
name = "张三"
age = 18

# 1. % 格式化
print("姓名：%s，年龄：%d" % (name, age))

# 2. format()
print("姓名：{}，年龄：{}".format(name, age))

# 3. f-string（推荐）
print(f"姓名：{name}，年龄：{age}")

# 4. 普通拼接（不推荐）
print("姓名：" + name + "，年龄：" + str(age))

四、完整案例演示 ​

python
# 综合示例：用户信息录入与格式化输出

# 输入部分
name = input("请输入姓名：")
age = int(input("请输入年龄："))  # 注意类型转换
score = float(input("请输入成绩："))

# 输出部分 - 使用 f-string（推荐）
print("\n" + "="*30)
print(f"用户信息")
print("="*30)
print(f"姓名：{name}")
print(f"年龄：{age}")
print(f"成绩：{score:.1f}")

# 调试输出（使用 = 符号）
print(f"\n调试信息：{name = }, {age = }, {score = }")

# 格式化表格输出
print("\n" + "-"*30)
print(f"{'姓名':<10}{'年龄':<10}{'成绩':<10}")
print("-"*30)
print(f"{name:<10}{age:<10}{score:<10.1f}")

五、易错点总结 ​

易错点正确做法
忘记 input 返回的是字符串使用 int(input()) 转换类型
print(a, b) 误以为无空格默认有空格，用 sep="" 去除
f-string 忘记加 f 前缀f"{var}" 不能写成 "{var}"
格式化时花括号数量不匹配{​{ 转义为 {，}​} 转义为 }
% 格式化时变量用元组包裹"%d %s" % (num, str)

六、速查卡片 ​

python
# 输入
var = input("提示")        # 返回 str
num = int(input("提示"))   # 转换为整数

# 输出基础
print(value)               # 输出并换行
print(a, b, c)             # 多个值，空格分隔
print(a, b, sep="")        # 自定义分隔符
print(a, end="")           # 不换行

# 格式化（推荐 f-string）
print(f"{var}")            # 变量
print(f"{var = }")         # 变量名=值（调试）
print(f"{num:.2f}")        # 2位小数
print(f"{num:,}")          # 千位分隔符
print(f"{num:>10}")        # 右对齐，宽度10
print(f"{{花括号}}")       # 输出字面量 {}
