Python 元组（Tuple）学习笔记 ​

一、元组概述 ​

元组（Tuple） 是 Python 中另一种重要的序列类型，与列表非常相似，但有一个关键区别：

特性元组（Tuple）列表（List）
可变性❌ 不可变✅ 可变
定义方式() 或 tuple()[] 或 list()
性能更快（不可变带来的优化）相对较慢
内存占用更小相对较大
使用场景数据保护、字典键频繁增删改的数据

核心特点：

  • 不可变：创建后不能修改、添加或删除元素
  • 有序：元素有固定顺序
  • 索引支持：支持正向和负向索引
  • 元素类型：可包含不同类型的元素
  • 可哈希：元素都是不可变类型时可作为字典的键

二、创建元组 ​

1. 基本创建方式 ​

python
# 使用小括号创建

print(tuple1)  # (100, 200, 300, 400, 500)

# 不加括号也可以（但建议加括号）
tuple2 = 1, 2, 3
print(tuple2)  # (1, 2, 3)

# 空元组
empty_tuple = ()
print(empty_tuple)  # ()

# 使用 tuple() 构造函数
tuple3 = tuple([1, 2, 3])      # 从列表转换
tuple4 = tuple("hello")        # 从字符串转换 → ('h', 'e', 'l', 'l', 'o')
tuple5 = tuple(range(5))       # 从 range 转换 → (0, 1, 2, 3, 4)

三、访问元组元素 ​

与列表的访问方式完全相同。

python
tuple1 = (100, 200, 300, 400, 500)

# 索引访问
print(tuple1[0])     # 100（第一个）
print(tuple1[2])     # 300（第三个）
print(tuple1[-1])    # 500（最后一个）
print(tuple1[-2])    # 400（倒数第二个）

# 切片访问
print(tuple1[2:4])   # (300, 400)
print(tuple1[:3])    # (100, 200, 300)
print(tuple1[2:])    # (300, 400, 500)
print(tuple1[::-1])  # (500, 400, 300, 200, 100)

七、遍历元组 ​

与列表的遍历方式完全相同。

python
tuple1 = (100, 200, 300, 400, 500)

# 方式1：直接遍历元素
print("方式1:")
for val in tuple1:
    print(val)

# 方式2：通过下标遍历
print("方式2:")
for i in range(len(tuple1)):
    print(i, tuple1[i])

# 方式3：使用 enumerate()（推荐）
print("方式3:")
for i, val in enumerate(tuple1):
    print(i, val)

八、元组的"不可变"深入理解 ​

1. 什么是不可变？ ​

元组的不可变指的是：元组对象本身在内存中的内容不可变。

python
tuple1 = (100, 200, 300)
print(id(tuple1))  # 例如: 140234567890

# ⚠️ 不能修改元素
# tuple1[0] = 999  # ❌ TypeError: 'tuple' object does not support item assignment

# ⚠️ 不能添加元素
# tuple1.append(400)  # ❌ AttributeError: 'tuple' object has no attribute 'append'

# ⚠️ 不能删除元素
# del tuple1[0]  # ❌ TypeError: 'tuple' object doesn't support item deletion

2. 重新赋值（改变引用） ​

虽然不能修改元组内容，但可以让变量指向新的元组：

python
tuple1 = (100, 200, 300)
print(id(tuple1), tuple1)  # 地址A: (100, 200, 300)

# 重新赋值：变量指向新元组，原元组可能被垃圾回收
tuple1 = tuple1 + (1, 2, 3)
print(id(tuple1), tuple1)  # 地址B（不同的地址）: (100, 200, 300, 1, 2, 3)

3. 元组中的可变元素（重要！） ​

如果元组中包含可变类型的元素（如列表），虽然不能替换该元素，但可以修改该元素内部的内容。

python
# 元组包含列表（可变元素）
tuple1 = (100, 200, 300, [1, 2, 3])
print(tuple1)  # (100, 200, 300, [1, 2, 3])

# ✅ 可以修改列表的内容（列表是可变的）
tuple1[3].append(4)
print(tuple1)  # (100, 200, 300, [1, 2, 3, 4])

# ✅ 可以修改列表中的元素
tuple1[3][0] = 999
print(tuple1)  # (100, 200, 300, [999, 2, 3, 4])

# ❌ 不能替换整个列表元素
# tuple1[3] = [4, 5, 6]  # ❌ TypeError: 'tuple' object does not support item assignment

九、元组 vs 列表使用场景 ​

使用场景推荐类型原因
数据不需要修改元组不可变性提供数据保护
作为字典的键元组元组可哈希（元素都是不可变类型时）
函数返回多个值元组Python 默认返回元组
数据需要频繁增删改列表可变性更灵活
性能要求高元组创建和访问更快，内存占用更小
python
# 元组作为字典键的示例
# 列表不能作为字典键（不可哈希）
# my_dict = {[1, 2]: "value"}  # ❌ TypeError

# 元组可以作为字典键
location = (40.7128, -74.0060)  # 坐标
city_dict = {location: "New York"}
print(city_dict)  # {(40.7128, -74.006): 'New York'}

# 函数返回多个值（自动打包为元组）
def get_min_max(numbers):
    return min(numbers), max(numbers)

result = get_min_max([1, 2, 3, 4, 5])
print(result)        # (1, 5)
print(type(result))  # <class 'tuple'>

# 元组解包
min_val, max_val = get_min_max([1, 2, 3, 4, 5])
print(min_val, max_val)  # 1 5

十一、元组解包（Tuple Unpacking） ​

元组解包是 Python 的一个强大特性：

python
# 基本解包
point = (10, 20)
x, y = point
print(x, y)  # 10 20

# 交换变量（利用元组解包）
a, b = 5, 10
a, b = b, a
print(a, b)  # 10 5

# 使用 * 收集剩余元素
numbers = (1, 2, 3, 4, 5)
first, *middle, last = numbers
print(first)   # 1
print(middle)  # [2, 3, 4]（注意：变成列表）
print(last)    # 5

# 忽略不需要的元素（使用 _）
person = ("Alice", 25, "Engineer")
name, _, job = person
print(name, job)  # Alice Engineer

十二、元组与列表的相互转换 ​

my_tuple = tuple(my_list)
print(my_tuple)  # (1, 2, 3, 4, 5)

# 元组 → 列表
my_tuple = (1, 2, 3, 4, 5)
my_list = list(my_tuple)

# 转换后可以修改
my_list.append(6)
print(my_list)   # [1, 2, 3, 4, 5, 6]

十三、完整综合示例 ​

python
# 学生成绩管理系统（使用元组存储不可变数据）

# 定义学生信息（使用元组确保数据不被意外修改）
students = [
    ("张三", 2024001, [85, 90, 88]),   # 元组包含可变列表
    ("李四", 2024002, [92, 87, 94]),
    ("王五", 2024003, [78, 85, 80]),
]

def calculate_average(scores):
    """计算平均分"""
    return sum(scores) / len(scores)

def display_student_info(students):
    """显示学生信息"""
    print("=" * 50)
    print(f"{'姓名':<10}{'学号':<12}{'成绩':<15}{'平均分':<8}")
    print("=" * 50)
    
    for name, stu_id, scores in students:
        avg = calculate_average(scores)
        scores_str = ", ".join(map(str, scores))
        print(f"{name:<10}{stu_id:<12}{scores_str:<15}{avg:<8.1f}")

def add_score(students, name, new_score):
    """为学生添加新成绩"""
    for i, (n, stu_id, scores) in enumerate(students):
        if n == name:
            students[i][2].append(new_score)  # 修改元组中的列表
            print(f"已为 {name} 添加成绩 {new_score}")
            return
    print(f"未找到学生: {name}")

# 显示初始信息
print("=== 初始学生信息 ===")
display_student_info(students)

# 添加新成绩
print("\n=== 添加成绩 ===")
add_score(students, "张三", 95)

# 显示更新后的信息
print("\n=== 更新后的学生信息 ===")
display_student_info(students)

# 演示元组的不可变性
print("\n=== 元组不可变性演示 ===")
student = ("赵六", 2024004, [70, 75, 80])
print(f"原始元组: {student}")

# 不能修改元组内的元素
try:
    student[0] = "赵六改"  # ❌ 会报错
except TypeError as e:
    print(f"错误: {e}")

# 但可以修改元组内的列表元素
student[2].append(90)
print(f"修改列表后: {student}")

# 元组解包使用
print("\n=== 元组解包示例 ===")
name, stu_id, scores = student
print(f"学生: {name}, 学号: {stu_id}, 成绩: {scores}")

十四、易错点总结 ​

易错点正确理解
单元素元组忘记加逗号(1,) 是元组，(1) 是整数
以为元组完全不可变元组中的可变元素（如列表）可以修改
混淆元组和列表的语法元组用 ()，列表用 []
尝试修改元组元素t[0] = x 会报 TypeError
以为元组推导式直接返回元组元组推导式返回生成器，需用 tuple() 转换
元组作为字典键时包含可变元素包含可变元素的元组不可哈希，不能作为字典键
python
# 错误示例合集
# 1. 单元素元组
t = (5)      # 这是整数，不是元组
t = (5,)     # 这才是元组

# 2. 修改元组元素
t = (1, 2, 3)
# t[0] = 10  # TypeError

# 3. 元组推导式
t = (x for x in range(5))  # 生成器，不是元组
t = tuple(x for x in range(5))  # 正确

# 4. 元组作为字典键
d = {}
# d[[1, 2]] = "value"   # 列表不可哈希
d[(1, 2)] = "value"      # 元组可哈希
# d[([1], 2)] = "value"  # 元组内包含列表，不可哈希

十五、性能对比：元组 vs 列表 ​

python
import timeit

# 创建性能对比
tuple_time = timeit.timeit('(1, 2, 3, 4, 5)', number=1000000)
list_time = timeit.timeit('[1, 2, 3, 4, 5]', number=1000000)
print(f"创建元组耗时: {tuple_time:.6f} 秒")
print(f"创建列表耗时: {list_time:.6f} 秒")

# 访问性能对比
t = (1, 2, 3, 4, 5)
l = [1, 2, 3, 4, 5]
tuple_access = timeit.timeit('t[2]', globals=globals(), number=1000000)
list_access = timeit.timeit('l[2]', globals=globals(), number=1000000)
print(f"访问元组耗时: {tuple_access:.6f} 秒")
print(f"访问列表耗时: {list_access:.6f} 秒")

# 内存占用对比
import sys
t = (1, 2, 3, 4, 5)
l = [1, 2, 3, 4, 5]
print(f"元组内存占用: {sys.getsizeof(t)} 字节")
print(f"列表内存占用: {sys.getsizeof(l)} 字节")

📌 核心记忆：

  • 元组用 () 定义，不可变但有序
  • 单元素元组必须加逗号：(1,)
  • 元组的不可变是指不能修改元素引用，但可变元素的内容可以改
  • 元组可哈希，可以作为字典的键（前提是所有元素可哈希）
  • 元组更适合保护数据、函数返回多值、字典键
  • 元组性能优于列表，内存占用更小
  • 列表与元组可互相转换：list() 和 tuple()
