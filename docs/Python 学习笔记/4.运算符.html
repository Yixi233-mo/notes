一、算术运算符 ​

运算符说明示例结果
+加20 + 1030
-减/取负20 - 1010
*乘20 * 10200
/除（返回浮点数）20 / 102.0
//整除（向下取整）10 // 33
%取模（求余数）20 % 100
**幂运算2 ** 38

代码示例 ​

python
a, b = 20, 10
print(f"{a} + {b} = {a + b}")   # 20 + 10 = 30
print(f"{a} - {b} = {a - b}")   # 20 - 10 = 10
print(f"{a} * {b} = {a * b}")   # 20 * 10 = 200
print(f"{a} / {b} = {a / b}")   # 20 / 10 = 2.0（注意结果是浮点数）
print(f"{a} % {b} = {a % b}")   # 20 % 10 = 0
print(f"2 ** 3 = {2 ** 3}")     # 8
print(f"10 // 3 = {10 // 3}")   # 3（向下取整）

# 字符串复用技巧
print("-" * 30)  # 输出 30 个 "-"

二、赋值运算符 ​

运算符说明等价形式
=赋值a = 1
+=加法赋值a += 2 → a = a + 2
-=减法赋值a -= 2 → a = a - 2
*=乘法赋值a *= 2 → a = a * 2
/=除法赋值a /= 2 → a = a / 2
//=整除赋值a //= 2 → a = a // 2
%=模赋值a %= 2 → a = a % 2
**=幂赋值a **= 2 → a = a ** 2
:=海象运算符（Python 3.8+）表达式中同时赋值并返回值

代码示例 ​

python
# 普通赋值
num3, num4 = 30, 40
num5 = num3 + num4
print(num5)  # 70

# 复合赋值
num3 += 50   # num3 = num3 + 50
print(num3)  # 80

# 海象运算符（Python 3.8+）
num1 = 20
print((num2 := 3 ** 2) > num1)  # 先赋值 num2=9，再判断 9 > 20
print(num2)  # 9

四、逻辑运算符 ​

运算符说明短路规则
and与：两边都为 True 才返回 True左为 False 则返回左值，不计算右值
or或：一边为 True 就返回 True左为 True 则返回左值，不计算右值
not非：取反返回布尔值

代码示例 ​

python
b1, b2 = False, True
print(b1 and b2)   # False
print(b1 or b2)    # True
print(not b1)      # True

# 短路逻辑与返回值（重要！）
print(5 and 8)     # 8（5 为 True，返回右值 8）
print(0 and 8)     # 0（0 为 False，返回左值 0）
print(5 or 8)      # 5（5 为 True，返回左值 5）
print(0 or 8)      # 8（0 为 False，返回右值 8）
print(not 5)       # False（5 为 True，取反得 False）

💡 核心理解：逻辑运算符的返回值 ​

python
# and: 左为假返回左，否则返回右
x and y  →  if x is False: return x else: return y

# or: 左为真返回左，否则返回右
x or y   →  if x is True: return x else: return y

# not: 总是返回布尔值
not x    →  return False if x else True

1. 原码、反码、补码（预备知识） ​

概念正数（如 +7）负数（如 -7）
原码0000011110000111
反码0000011111111000
补码0000011111111001

💡 位运算规则：计算机中位运算以补码形式进行

2. 正数位运算示例 ​

python
num1, num2 = 17, 13
# 17 = 00010001
# 13 = 00001101

print(f"与运算: {num1} & {num2} = {num1 & num2:08b}")   # 00000001
print(f"或运算: {num1} | {num2} = {num1 | num2:08b}")   # 00011101
print(f"异或: {num1} ^ {num2} = {num1 ^ num2:08b}")     # 00011100
print(f"取反: ~{num1} = {~num1:08b}")                   # 补码表示

七、身份运算符 ​

运算符说明
is判断两个变量是否引用同一个对象（内存地址相同）
is not判断两个变量是否引用不同对象

is vs == 的区别（重点！） ​

运算符比较内容说明
is内存地址是否为同一个对象
==值内容是否相等
python
# 场景1：变量指向同一个对象
a = [1, 2, 3]
b = a                    # b 和 a 指向同一个列表
print(b is a)            # True（同一对象）
print(b == a)            # True（值相等）

# 场景2：切片创建新对象
b = a[:]                 # 创建副本
print(b is a)            # False（不同对象）
print(b == a)            # True（值相等）

# 小整数池影响
m, n = 20, 20
print(m is n)            # True（小整数池复用）

result = 2 + 3 * 4     # 14（先乘后加）

九、易错点总结 ​

易错点正确理解
/ 结果类型总是浮点数，不是整数
// 整除方向向下取整，负数时注意
% 符号规则结果符号与除数相同
and/or 返回值返回操作数的值，不一定是布尔值
is 与 == 混淆is 比较地址，== 比较值
位运算符号& 不是 and，`
负数右移向负无穷取整，不是简单截断
